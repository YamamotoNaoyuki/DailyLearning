# 基底核と手続き記憶——"体で覚える" を脳が作り出すメカニズム

**日付**: 2026-04-25
**分野**: 解剖学

## 学んだこと

### 基底核とは ——大脳深部の 5 つの構造

**基底核 (basal ganglia)** は大脳の深部に位置する、皮質下核群の総称。古典的に以下 5 つを含む（異同はあるが実用的な集合）:

1. **線条体 (striatum)**: 尾状核 (caudate) + 被殻 (putamen) — **入力側**
2. **淡蒼球 (globus pallidus)**: 外節 (GPe) + 内節 (GPi) — **出力側**
3. **視床下核 (subthalamic nucleus, STN)**: **ハイパー直接路の入力**
4. **黒質 (substantia nigra)**: 緻密部 (SNc, ドーパミン産生) + 網様部 (SNr, 出力側)
5. **側坐核 (nucleus accumbens)**: 報酬系の一部、古典的には "腹側線条体"

機能的には **大脳皮質 → 線条体 → 淡蒼球/黒質 → 視床 → 大脳皮質** という **ループ**。中心的役割は **「行動の選択とゲーティング」**——大脳皮質が発する多数の "候補行動" から 1 つを通過させ、他を抑制する。

### 古典的モデル: 直接路 vs 間接路

Albin & Young (1989)、DeLong (1990) の古典モデル:

**直接路 (direct pathway)**: Cortex → Striatum (D1 受容体) → GPi/SNr → Thalamus → Cortex
- 線条体 D1 ニューロンが発火すると、GPi/SNr の **抑制性出力を抑制**（脱抑制）
- 視床が活性化 → 皮質が活性化 → **行動が選択・実行される**
- "行きたい" 経路

**間接路 (indirect pathway)**: Cortex → Striatum (D2) → GPe → STN → GPi/SNr → Thalamus
- 経路が長く、**追加の抑制** を挟む
- 視床の抑制が強化 → 皮質が抑制 → **行動が抑制される**
- "止めたい" 経路

**ドーパミン (SNc → Striatum)** はこの 2 経路を調整:
- D1 受容体（直接路）: ドーパミンで **興奮**
- D2 受容体（間接路）: ドーパミンで **抑制**
- 結果: ドーパミン放出 → 直接路促進 + 間接路抑制 → **行動選択が促進**

**パーキンソン病**: SNc ドーパミン細胞の死 → 直接路弱化 + 間接路強化 → **無動 (akinesia), 寡動 (bradykinesia)**
**ハンチントン病**: 線条体 D2 細胞（間接路）の変性 → 間接路弱化 → **舞踏運動（不随意運動が抑制できない）**

### ハイパー直接路 (hyperdirect pathway)

Nambu et al. (2002) が強調した第三の経路:

**Hyperdirect**: Cortex → STN (直接) → GPi/SNr
- 線条体を飛ばして STN に直接入力
- **最速の抑制**——皮質から即座に "全ての行動を停止せよ" の信号
- 実装上、予期せぬ情報で現在の行動を中断する "emergency brake"

時間的には: **Hyperdirect (10-15ms) → Direct (15-20ms) → Indirect (20-30ms)**
- 先に止めて (hyperdirect)
- 選択したい行動を解除 (direct)
- それ以外を継続的に抑制 (indirect)

この 3 段階構造が **行動選択の精密制御** を実現する。

### 現代的理解: "対立" ではなく "協働"

古典モデルでは直接路と間接路は **反対の機能** とされたが、2000 年代以降の光遺伝学・in vivo 記録は **両経路が同時に発火する** ことを示した (Cui et al., 2013 Nature)。**選択された行動では両方活性、抑制された行動では両方活性低**。

現代モデル: 直接路は **「選ばれた行動を押し進める」**、間接路は **「選ばれなかった行動を抑える」**——両方の行動を同時に管理する **centre-surround** 型制御。

さらに最近 (2024 bioRxiv) の研究は、**運動記憶そのものが D1/D2 の 2 経路に独立に保存される** と示唆。"成功した動作" は D1 路に、"失敗した/避けるべき動作" は D2 路に——これは **"正例と負例を別々に保持する" 二値分類** のような構造。

### 手続き記憶 (procedural memory)

**手続き記憶** = "体で覚える" 非宣言的記憶。

- **宣言的記憶** (declarative): 海馬が司る "言葉で説明できる" 記憶（昨日の昼食、首都の名前など）
- **手続き記憶** (procedural): 基底核 + 小脳 + 運動野が司る "体で実行する" 記憶（自転車の乗り方、タイピング、楽器演奏、ゴルフスイング）

健忘症 (H.M. 患者) の古典的発見: 海馬を切除されると新しい宣言記憶を作れないが、**新しい運動技能（鏡像描画など）は正常に習得できる** ——**手続き記憶は海馬に依存しない**。

手続き記憶の **構築フェーズ**:
1. **Cognitive phase** (認知段階): 前頭前野 + 海馬が主導、"どうやるか" を言語的に理解
2. **Associative phase** (連合段階): 基底核が参加、運動と結果の連合が形成
3. **Autonomous phase** (自律段階): **基底核と小脳が主、意識は離脱**——"自動化" された

**chunking（チャンク化）**: 基底核は複数の個別動作を **1 つの行動単位に束ねる**。ピアノで「ドミソ」を 1 つのコードパターンとして、ゴルフで「アドレス → テークバック → トップ」を 1 つのルーティンとして。**意識は個別でなく chunk を選ぶだけでよい**。

### 小脳との分業

基底核と小脳はどちらも運動学習に関わるが、分業している:

| | 小脳 | 基底核 |
|--|------|--------|
| 主機能 | **タイミング、予測、誤差修正** | **行動選択、chunk 実行** |
| 学習信号 | 登上線維の誤差 | **ドーパミン (報酬予測誤差)** |
| 時間スケール | ミリ秒 | 秒〜分 |
| 病変症状 | 運動失調 (ataxia) | 動作開始困難 / 不随意運動 |

**小脳 = 運動の "how"、基底核 = 運動の "what/when"**。ピアノで言えば、小脳は打鍵のタイミング・力の微調整、基底核はどの指パターンを選ぶかと、どの和音 chunk を発火するか。

両方が協働して初めて流暢な動作が成立する。どちらか片方が損なわれると、運動はぎこちなくなる。

### ドーパミンと報酬予測誤差

Wolfram Schultz (1997) の歴史的発見: **SNc のドーパミン細胞は "報酬予測誤差" をコード** している。
- 予想外の報酬 → ドーパミン一過性上昇
- 予想通りの報酬 → ドーパミン変化なし
- 予想された報酬が来ない → ドーパミン低下

これは **強化学習の TD (temporal difference) エラー** と数学的に同型。基底核は **報酬予測誤差を使って "良い行動" の選択確率を調整** する——つまり **強化学習の生物学的実装**。

現代の深層強化学習（AlphaGo、ChatGPT の RLHF）は、この基底核の学習則を人工ニューラルネットで模倣したもの。**脳の基底核 = 生物版の RLHF**。

### 習慣形成 (habit formation) と柔軟性喪失

基底核は **習慣形成** の中核。同じ文脈で同じ行動を繰り返すと、基底核は **"文脈 → 行動" のマッピングを直接学習** する——目標指向性（"何のためにやるか"）は徐々に喪失し、**刺激駆動型の自動行動** になる。

これは効率的だが **柔軟性を失う**: 環境が変わっても古い習慣が出てしまう。依存症はこのメカニズムの病的亢進——基底核が "薬物刺激 → 摂取行動" を強固に学習し、前頭前野の抑制が効かなくなる。

逆に **運動学習の長期定着** は、まさに習慣化の恩恵。ピアノスケールを 1 万回繰り返すと、前頭前野を通さず基底核だけで実行可能になる（意識はメロディや感情表現に自由になる）。

## 気づき・洞察

### "意識は遅い、基底核は速い"

全ての身体技能の熟達は **「意識から基底核へのタスク転送」**。初学者は全ての動作を前頭前野で考え、応答時間 300-500ms。熟達者は基底核で chunk 実行し、応答時間 100-150ms。

これは **ピアノ・ゴルフ・タイピング・会話・運転**、あらゆる技能に共通する。熟達とは「前頭前野を空ける」こと。前頭前野が空くと **メタ認知・戦略選択・感情表現** に使える。

### 習慣の両義性

基底核の chunking は **効率の源泉** だが、**硬直化の源泉** でもある。良い習慣（歯磨き、ストレッチ、練習ルーティン）も悪い習慣（喫煙、不健康な食事、悪いスイングフォーム）も、同じメカニズムで形成される。

"悪い癖を直すのが難しい" のは、**基底核は一度書き込まれた chunk を消さない**——新しい chunk を上書きで優位にするしかない。これは数ヶ月かかる。だから **最初に正しく習う** のが長期的に最重要——"初心者の時に良いコーチ" の価値はここ。

### ランダム練習 = 基底核の再選択を強制する

今日の golf で学んだ **Contextual Interference 効果** の神経基盤がまさにこれ。ブロック練習は同じ chunk を連続発火するだけで、基底核は **新しい選択** をしない。ランダム練習は毎試行で基底核が **"どの chunk を呼び出すか" を再選択** ——この再選択プロセスが chunk 間の連合を強化し、長期定着を生む。

つまり **"基底核を選択作業に晒し続ける"** のが深い学習の鍵。反復は **同じ chunk の実行** だけで、新しい chunk の形成ではない。

### パーキンソン病は "選択" の病気

パーキンソン病の無動は単に "動けない" のではなく **"行動を開始できない"**——基底核が行動を選択・解除できない。L-DOPA がドーパミンを補うと、選択機能が部分的に回復する。

これは **基底核 = 選択装置** という理解を支持する臨床的証拠。パーキンソン患者は考えることも感じることもでき、"歩きたい" 意思もあるが、**意思を行動に変換する段階** で詰まる。意識から基底核への架け橋が壊れている。

### "自動性" が表現を可能にする

熟達したピアニスト・ゴルファー・スピーカーは、技術に意識を割かない。**技術が自動化されているからこそ、芸術的表現・戦略・相手の読み** に意識を使える。

これは **"技術は目的ではなく、目的への通過点"** という深い真実。毎週末ゴルフレッスンで internal focus をする人は、永遠に技術レベルで止まる。**基底核に転送して初めて、ゴルフが楽しくなる**。同じことがピアノ・ドイツ語会話・プログラミングに言える。

### 基底核は "進化的に古く、機能的に新しい"

基底核は爬虫類にもある進化的に古い構造。しかし人間では **認知的機能**（作業記憶、意思決定、言語、感情）にも関わる。これは進化が "古いアーキテクチャを再利用" する典型——新しい機能のために新しい脳を作らず、古い基底核に **皮質からの投射** を追加して運動以外にも拡張した。

だから依存症、強迫性障害、トゥレット症候群、統合失調症——多数の精神疾患が **基底核機能異常** と関連する。感情・思考・行動は、神経回路の深いレベルで絡み合っている。

## 他分野との接続

### 今日の全分野に通底する "基底核"

今日の 10 分野の多くが、結果的に **基底核の機能** に収束する:

- **piano**: ボイシングの自動化 = chunk を基底核に書き込む
- **golf**: ランダム練習 = 基底核に多様な chunk を形成
- **tech (Confidential Computing)**: TEE の行動選択（"enclave 内で実行するか、外で実行するか"）= 基底核の直接路/間接路と類比
- **cs (Shannon エントロピー)**: 情報理論が学習の定量化 = ドーパミン報酬予測誤差の数理基盤
- **music (クセナキス)**: ランダム音響雲 = 基底核のランダム chunk 選択の美学的対応
- **deutsch (Smalltalk)**: 会話の流暢さ = 定型フレーズが基底核の chunk として自動化

### 小脳（昨日）との接続 —— "運動の 2 大皇帝"

昨日学んだ小脳（forward model）と今日の基底核（行動選択）は、運動の "いつ・どう・何を" の分業:
- **何を** 選ぶか: 基底核
- **いつ** 実行するか: 基底核（開始） + 小脳（タイミング）
- **どう** 実行するか: 小脳（forward model で微調整）

どちらかが欠けても技能は成立しない。**運動は "意図 → 選択 → タイミング → 実行 → フィードバック" の連鎖**。

### cs分野（強化学習）との直接接続

基底核のドーパミン・TD エラーは **強化学習の生物学的実装**。これは **"脳はアルゴリズムを実行する"** という計算論神経科学の強い主張の裏付け。AlphaGo, ChatGPT の RLHF, ロボット制御——すべて基底核のモデルを工学的に模倣している。

### tech分野（Passkeys, Confidential Computing）との類比

TEE の "attestation で承認された行動だけ実行、他を拒否" は **直接路（承認）+ 間接路（拒否）+ ハイパー直接路（緊急停止）** と構造的に同型。**行動のゲーティング** という機能は、生物・電子システムで独立に発見されたアーキテクチャ。

### golf/piano 両分野（今日の練習法）との直接接続

今日の golf のランダム練習、piano のボイシング練習——両方とも **基底核を酷使する練習設計**。反復でも暗記でもなく、**"異なる chunk を文脈に応じて選択し続ける" 練習** が基底核の能力を鍛える。練習哲学の根底にある神経科学。
