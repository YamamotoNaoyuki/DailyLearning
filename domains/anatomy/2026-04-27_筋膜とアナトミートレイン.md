# 筋膜とアナトミー・トレイン — 全身を貫くテンセグリティ
**日付**: 2026-04-27
**分野**: anatomy
**タグ**: #筋膜 #fascia #Anatomy_Trains #Tom_Myers #Wilke #SBL #テンセグリティ #connective_tissue

## 学んだこと

### 筋膜 (fascia) とは何か
**筋膜 (fascia)** は身体全体を覆い、貫き、繋ぐ**結合組織 (connective tissue) の網**である。20世紀後半まで「**ただの包装紙**」として軽視され、解剖学教科書では筋肉の絵を描くときに「**剥がして捨てる**」対象だった。21世紀に入り、**Schleip, Findley, Stecco, Myers** らの研究により、筋膜は**独立したセンサー器官・力学伝達系・全身的連続体**として再発見されつつある。

### Fascia Research Congress と新パラダイム
2007年の **First Fascia Research Congress** (Harvard Medical School) は、筋膜を**独立した研究分野**として確立した転換点。それまで断片的だった筋膜研究 (Rolfing 学派、osteopathy、生物力学、組織学) が**統一された科学領域**として認知された。

**主要発見**:
- 筋膜は**コラーゲン繊維 + ヒアルロン酸 + エラスチン + 細胞外マトリクス**で構成される連続体
- **筋線維芽細胞 (fasciacyte)** が機械刺激に応答して組織を再構築する
- **自由神経終末密度が皮下筋膜で骨格筋の6倍** (Stecco, 2018)、**深筋膜は強力な感覚器官**
- **ヒアルロン酸の粘性変化**で筋膜層間の滑走性が変動——「**癒着**」の物理学的基盤

### Tom Myers と Anatomy Trains
**Thomas W. Myers** (1949-) は **Ida Rolf** (Rolfing 創始)、**Moshe Feldenkrais**、**Buckminster Fuller** (テンセグリティ理論) に学んだボディワーカー。30年超の解剖実習体験 (主に新鮮屍体での dissection) から、「**個別の筋肉**」ではなく「**筋・筋膜の連続体としてのライン**」が身体運動の機能単位だと提唱した。

主著『**Anatomy Trains: Myofascial Meridians for Manual and Movement Therapists**』(初版2001、第4版 2020) で、**12のミオファシャル・メリディアン (myofascial meridians)** を体系化:

| メリディアン | 通る筋・筋膜 |
|---|---|
| **Superficial Back Line (SBL)** | 足底筋膜 → アキレス腱 → 腓腹筋 → ハムストリング → 仙結節靭帯 → 仙腰筋膜 → 脊柱起立筋 → 後頭筋 → 帽状腱膜 |
| **Superficial Front Line (SFL)** | 前脛骨筋 → 大腿四頭筋 → 腹直筋 → 胸骨筋 → 胸鎖乳突筋 |
| **Lateral Line (LL)** | 腓骨筋 → 腸脛靭帯 → 中・小殿筋 → 外腹斜筋 → 肋間筋 → 胸鎖乳突筋・頭板状筋 |
| **Spiral Line (SPL)** | 頭板状筋 → 反対側の菱形筋 → 前鋸筋 → 外腹斜筋 → 内腹斜筋 → 大腿筋膜張筋 → 前脛骨筋 → 長腓骨筋 → 腰部脊柱起立筋（らせん状） |
| **Deep Front Line (DFL)** | 後脛骨筋 → 内転筋群 → 腸腰筋 → 横隔膜 → 心膜 → 舌骨下筋群 → 翼突咬筋 |
| **Arm Lines (4本)** | Deep Front / Superficial Front / Deep Back / Superficial Back Arm Line |
| **Functional Lines (3本)** | Front / Back / Ipsilateral Functional Line（対角線的・力伝達用） |

これらは「**東洋医学の経絡の解剖学的対応物**」と位置付けられたため、初期には「ニューエイジ的疑似科学」と批判されたが、**2010年代の系統的レビュー**により**部分的科学的妥当性**が確認されつつある。

### Wilke et al. (2016) — 系統的レビューによる検証
ドイツの Goethe University の **Jan Wilke** らによる **2016 年の系統的レビュー** *"What Is Evidence-Based About Myofascial Chains: A Systematic Review"* (The Archives of Physical Medicine and Rehabilitation) は、Myers の主張を解剖学文献で検証した最初の本格研究。MEDLINE / ScienceDirect / Google Scholar の 1900-2014 の文献を網羅:

**結果（メリディアン別）**:
- **Superficial Back Line**: **強い証拠** (14 研究)。3つの主要連結（足底筋膜→アキレス腱、腓腹筋→ハムストリング、ハムストリング→脊柱起立筋）すべて検証済み
- **Back Functional Line / Front Functional Line**: 強い証拠
- **Spiral Line**: 中程度の証拠 (9連結中5連結検証、21研究)
- **Lateral Line**: 中程度の証拠 (5連結中2連結検証、10研究)
- **Superficial Front Line**: **証拠なし** (7研究で連結確認できず、特に大腿直筋と腹直筋の構造的連結は未確認)

つまり「**SBL は科学的に立証された機能単位**」だが、「**SFL は Myers の理論的構築物に留まる**」と結論された。Myers の理論は**全体としては経験的・実用的に有効だが、すべての meridian が等しく解剖学的に裏付けられているわけではない**。

### Krause et al. (2016) — 力学的伝達の証拠
別の系統的レビュー *"Intermuscular force transmission along myofascial chains: a systematic review"* (Journal of Anatomy) は**力伝達**の機構を検証。生体内・摘出組織・ロボット計測で、**隣接する筋肉が筋膜を介して力を伝達する**ことが確認された。例:

- **腓腹筋の張力がアキレス腱を超えて足底筋膜にかかる**: dissection と ultrasound elastography で確認
- **ハムストリングの張力が仙結節靭帯を超えて腰仙筋膜に伝達**: cadaver 実験で測定
- **腰仙筋膜の張力が広背筋・胸腰筋膜・反対側の大殿筋に伝達**: 「**運動の co-activation pattern**」の解剖学的基盤

これは Vleeming, Schuenke らの研究と整合し、**「力は筋単位で完結せず、筋膜ネットワーク全体に分散する」**という新パラダイムを支持する。

### Tensegrity (テンセグリティ) — 力学的全体論
**Buckminster Fuller** が建築理論で提唱した **tensegrity (tensile integrity)** は、「**圧縮要素 (骨)** と **張力要素 (筋膜)**」が**相補的に支え合う構造**。Donald Ingber は1990年代にこれを**細胞生物学**に適用し、**個別の細胞も tensegrity 構造を持つ**ことを示した (*Scientific American*, 1998)。

身体全体への適用 (Levin の **biotensegrity**, 2002):
- **骨は浮島** (compression elements) で、**筋膜の張力ネットワーク**が骨を空中に保持する
- **荷重は局所的でなく全身的に分散**——足首の捻挫が首のこわばりを生みうる
- **個別関節の運動は単独では決まらず、全身の張力配分が決める**

これは古典的な「**梁-アーチ**モデル (建築学的な圧縮優位)」とは対照的。**人体の力学は緊張-圧縮バランスとして読むべき**であり、Anatomy Trains の meridian はこのテンセグリティ・ネットワークの解剖学的可視化。

### Stecco の Fascial Manipulation
イタリアの **Luigi Stecco** とその子供 (Carla, Antonio Stecco) が体系化した **Fascial Manipulation** (2004-) は、Anatomy Trains と独立に発達した臨床技法。**Centers of Coordination (CC)** と **Centers of Fusion (CF)** を解剖学的に正確に定位し、ヒアルロン酸の凝集による**筋膜層間滑走障害**を**摩擦的マッサージ**で解消する。**EBM ベースの筋膜療法**として、ヨーロッパの理学療法で標準化が進む。

Carla Stecco の『**Functional Atlas of the Human Fascial System**』(Elsevier 2015) は、**世界初の筋膜系の解剖学アトラス**で、新鮮屍体の精密 dissection 写真集として臨床医・研究者の必携書。

### 横隔膜と DFL — 「呼吸が全身を動かす」
**Deep Front Line (DFL)** は Myers の最も特異な発見の一つ——**横隔膜が直接、後脛骨筋・内転筋・腸腰筋・心膜・舌骨下筋とつながる**ライン。臨床的には:

- **足のアーチ低下** (後脛骨筋弱化) が**横隔膜の機能不全**と関連
- **腸腰筋の慢性収縮**が**横隔膜下垂と腰痛**を同時に生む
- **顎関節障害** (TMJ) と**骨盤底筋群**の関連は DFL を介する

これは Janda の "**lower crossed syndrome**" (1983) や Hodges の体幹安定化研究と整合する。

### 動物の筋膜メリディアン
Anatomy Trains 第4版 (2020) では、**馬と犬の筋膜メリディアン**も解剖学的に確認されたと報告。**Ribechini, Wilke** らがウマ屍体で SBL に相当する連結を確認 (2018)。哺乳類全般に**進化的に保存された筋膜ネットワーク**が存在する可能性が示唆される。これは「**Anatomy Trains が人類学的偶然ではなく動物界の構造原理**」という強い主張に繋がる。

## 気づき・洞察

### 「個別の筋肉」概念は解剖学の便宜的虚構
古典解剖学は身体を「**900個の個別の筋肉**」に分解する。これは Vesalius (1543) 以来の**dissectionの利便性**から生まれた**便宜的な概念枠**であり、**機能的実体ではない**。Myers の「**個別の筋肉は織物の繊維のひとつでしかない**」という主張は、ベッドカバーの**1本の糸**を取り出して「これが布だ」と言うのと同じ誤りを指摘する。**機能単位は筋膜連続体**であり、個別筋肉の名前は**学習の補助具**に過ぎない。

### "Strong evidence for SBL, no evidence for SFL" の意味
Wilke レビューが示すのは「**理論の全肯定でも全否定でもない、部分的妥当性**」。これは科学のあるべき姿——理論が**経験的事実によって部分的に修正される**プロセス。Myers は SFL の証拠不足を認め、**第4版で慎重な記述に変更**した。**理論を捨てずに改訂する誠実さ**は、Schleip, Findley, Stecco らの fascia research community 全体の特徴。

### 「東洋経絡」と「西洋筋膜」の構造的近接
Anatomy Trains の SBL は東洋医学の **太陽膀胱経** とほぼ重なる。SFL は **太陰肺経・陽明胃経** に対応する部分。これは「**東洋医学が筋膜系を直感的に発見していた**」のか、「**Myers が経絡を解剖学的に再構築した**」のかは確定しない。しかし**両者の地理的一致**は偶然以上であり、**身体感覚の地理学**には文化を超えた基盤がある可能性を示唆する。**経絡を疑似科学として一蹴する態度は、構造的事実を見落とす**。

### Tensegrity = 力学の全体論
古典力学は**局所原理**で動く——力は加えられた場所で計算される。Tensegrity は**全体論的**——一箇所への力が**ネットワーク全体に分散**する。これは Newton 力学から Field theory への遷移と類比的——**点の力学から場の力学へ**。**身体の故障は「故障した場所」では治らない**——テンセグリティ的に分散した張力配分の再調整が必要となる。これが**整体・Rolfing・Feldenkrais の理論的根拠**。

### 「ヒアルロン酸の粘性」というミクロな鍵
Carla Stecco が発見した「**筋膜層間のヒアルロン酸が炎症で凝集→粘性増大→滑走障害**」というメカニズムは、**マクロな機能不全がナノスケールの分子変化に還元される**例。これは**慢性疼痛の生化学的基盤**を提供する重要な発見であり、**マッサージが「気持ちいいだけ」でなく粘性を物理的に低下させる**という科学的根拠を与える。

## 他分野との接続

- **piano (前腕回旋・暗譜の手の感覚)**: ピアニストの「腕の重さの伝達」はまさに上肢 Arm Line の力学。Matthay の前腕回旋は **Deep Front Arm Line** (橈骨→上腕二頭筋→烏口腕筋→烏口突起) を通じて指先まで伝達される。**個別筋を意識する練習より、筋膜ライン全体を感じる練習**が効率的——Taubman 学派の "**continuous arm motion**" は biotensegrity 的
- **golf (スイングと回旋)**: ゴルフスイングは典型的な **Spiral Line** 動作——逆側の頭板状筋から始まり、菱形筋・前鋸筋・外腹斜筋を通じて反対側の下肢へ螺旋的に伝達。スイングの「**コイリング**」は Spiral Line の張力蓄積であり、Anatomy Trains 視点で動作分析が可能
- **anatomy (基底核・小脳・固有受容覚・足底アーチ)**: 既習の知識との統合点。固有受容覚は筋紡錘単独ではなく、**筋膜の自由神経終末**が密度6倍で寄与する。足底アーチは SBL の根元として全身姿勢に直結
- **bread (グルテンネットワーク)**: グルテンは小麦タンパク質の**3次元連続体**として、パン生地の構造全体を支える。**ヒアルロン酸-コラーゲン**ネットワークと**グリアジン-グルテニン**ネットワークは共に**機械的全体論**の例
- **cs (グラフ・ネットワーク構造)**: Anatomy Trains は身体を**重み付き有向グラフ**として再記述する——筋を頂点、筋膜連結を辺、張力伝達を重みとして。**身体の機能解析はネットワーク科学**に翻訳可能
- **other (深海熱水噴出孔の生態系)**: 個別生物を切り出さず**生態系ネットワーク全体**として理解する視点が、筋膜連続体の理解と同型——**還元主義から関係論的視点へ**
