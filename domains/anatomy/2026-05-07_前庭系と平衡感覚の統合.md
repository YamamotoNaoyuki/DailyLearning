# 前庭系と平衡感覚の統合 — 半規管・耳石器と多感覚統合

**日付**: 2026-05-07
**分野**: anatomy
**タグ**: #前庭系 #半規管 #耳石器 #平衡感覚 #多感覚統合

## 学んだこと

### 前庭系の解剖 — 内耳の隠れた器官

前庭系 (vestibular system) は内耳の側頭骨錐体部に位置し、聴覚を担う蝸牛と隣接して存在する。3 つの **半規管 (semicircular canals)** と 2 つの **耳石器 (otolith organs)** = 卵形嚢 (utricle)・球形嚢 (saccule) から構成される。両者は「動きの種類」を分業して感知する:

- **半規管**: 角加速度 (頭の回転) を感知
- **耳石器**: 線形加速度 (前後・左右・上下の並進、および重力) を感知

半規管は3つあり、互いに**ほぼ直交**(各々が90度ずつ離れた3次元軸)に配置されている: 水平 (lateral)、前方 (anterior/superior)、後方 (posterior/inferior)。これは数学的に「3次元回転を直交基底で分解する」ことを生物学的に実装している ─ どんな頭の回転も、この3軸成分に分解して計測できる。これは IMU センサ (3 軸ジャイロ) と全く同じ設計原理。

### 半規管の物理 — クプラと内リンパの慣性

各半規管はリング状の管で、**内リンパ液** (endolymph) で満たされている。リングの一部に **膨大部 (ampulla)** という膨らみがあり、その中に **クプラ (cupula)** というゼラチン状の隔壁がある。クプラには **有毛細胞 (hair cells)** の繊毛が埋め込まれている。

頭が回転すると:
1. 半規管自体は頭と一緒に回る
2. 内リンパ液は **慣性で取り残される** (相対的に逆回転する)
3. この相対流がクプラを傾ける
4. クプラに埋まった有毛細胞の繊毛がしなる
5. しなる方向によって有毛細胞の脱分極/過分極が起き、神経信号が生成される

この機構は「**慣性質量を使った加速度計**」そのもので、定速回転 (例えば回転椅子に等速で乗っている状態) では内リンパも一緒に回り始めるため信号が出なくなる。これが「平衡感覚は『回転している』ではなく『回転を始めた/終えた』を感知する」という性質の由来。回転を急停止すると、内リンパだけ慣性で回り続けるため、「逆方向に回っている」錯覚 (回転後性めまい) が起きる。

### 耳石器の物理 — カルシウム結晶の重み

卵形嚢 (utricle) は水平面、球形嚢 (saccule) は垂直面に有毛細胞層を持つ。その上に **耳石膜 (otolithic membrane)** が乗り、その上に **耳石 (otoconia, otoliths)** という炭酸カルシウムの微結晶が並んでいる。

頭が傾くと:
1. 耳石は重力の影響で「下」へ滑る
2. 耳石膜が引きずられ、有毛細胞の繊毛がしなる
3. 卵形嚢は水平加速度 (前後左右、頭を傾ける) を、球形嚢は垂直加速度 (頭の上下、エレベーターでの加速感覚) を感知

加齢 (50 歳以降) や頭部打撲で耳石が剥がれて半規管に迷い込むと **良性発作性頭位めまい症 (BPPV)** が起こる。Epley 法という頭位変換手技で耳石を半規管から引き戻すことで治療する。

### 多感覚統合 — 「平衡」は3 つの入力の重み付け融合

平衡感覚は前庭系単独では成立しない。3 つの感覚情報を **小脳・前庭核 (vestibular nuclei)** で統合して初めて「自分の姿勢と動き」が知覚される:

1. **前庭入力**: 頭の動きと重力方向
2. **視覚入力**: 環境の動き (オプティカルフロー)、地平線の傾き
3. **体性感覚 (固有受容覚) 入力**: 関節角度、筋の張力、足底からの圧力分布

通常はこれらが整合するが、不整合が起きると **乗り物酔い**・**めまい**・**眩暈** が生じる。船酔いは「視覚は静止 (船室の壁)・前庭系は揺れている」という不整合、宇宙酔いは「重力ゼロで耳石器が機能しない」状況。VR 酔いは「視覚は動いている・前庭系は静止」の逆向きの不整合。

Bayes 推論的に脳は「最も信頼できる入力に重み付けを大きくする」: 暗闇では視覚の重みが下がり前庭系・体性感覚の重みが上がる、酔ったときは前庭系の信号が劣化するので視覚に頼る、など。これが「閉眼で立てない」「暗い部屋でめまいが悪化する」現象の説明。

### 前庭脊髄反射と前庭眼反射 (VOR)

前庭系の出力は 2 つの主要反射ループに繋がる:

1. **前庭眼反射 (VOR)**: 頭が右に回ると眼球が左に回り、視野を安定させる。レイテンシは 7-15 ms と人体最速の反射の一つ。歩行中に視野がブレないのはこの反射のおかげ。VOR が壊れると「頭を動かしながら本を読む」ことが不可能になる。
2. **前庭脊髄反射 (VSR)**: 体の傾きを感知して、抗重力筋 (脊柱起立筋、下肢伸筋) を瞬時に活性化し転倒を防ぐ。これはプロプリオセプティブな調整より速く、無意識的。

### 加齢と前庭機能低下

60 歳以上で前庭機能は約 30% 低下し、80 歳で 50% 以上低下する。これが「高齢者が転倒しやすい」原因の生理学的基盤。**前庭リハビリテーション** (vestibular rehabilitation) は、視覚追跡と頭部運動を組み合わせた訓練で、中枢補償 (脳が劣化した前庭信号を視覚・体性感覚で補う学習) を促進する。

## 気づき・洞察

前庭系は「**意識に上らない感覚**」の代表である。視覚・聴覚・触覚は意識化されるが、自分が今どう傾いているかを「感じる」ことは普通ない。ところが視覚を遮ると (目を閉じて片足立ちする)、たちまち平衡感覚の難しさが意識化される。これは「平衡」が単独感覚ではなく**多感覚統合の創発的結果**であり、しかも統合が裏でスムーズに動いているうちは意識化されない、ということを示唆する。

そして前庭系は **3 次元空間における自己位置の参照系**を提供している。これは脳の他のシステム (空間記憶を担う海馬の場所細胞、グリッド細胞、頭方向細胞、2026-05-05 のエントリ参照) と密接に連携する。前庭系の損傷者は空間記憶の障害も呈する: ナビゲーションは「頭の動きと移動の累積」によって成立しており、前庭系がなければマップが構築できない。これは慣性航法 (INS, Inertial Navigation System) と同じ原理。

最も興味深いのは、前庭系が **時間の知覚** にも関与している証拠。回転刺激中・直後は時間知覚が歪む (短く感じる)、めまい中は時間が引き延ばされる、という研究がある。これは「時間 = 自己の動きの積分」という、Henri Bergson の哲学的洞察に通じる。動かない自己は時間も持たない。

## 他分野との接続

ピアノ・ゴルフ両分野の **姿勢制御** と直接関係する。ピアノを弾きながら体を揺らす演奏者 (Lang Lang など) は前庭系を積極的に動員し、音楽のリズム感と身体運動を連動させている。一方ゴルフのスイングでは「頭を動かさない」ことが基本だが、これは前庭系を**安定参照系**として保つためでもある。頭が動くと VOR が起動し、視覚が一瞬不安定になり、ボールへの空間認識が乱れる。

CS 分野の **慣性航法系・センサーフュージョン** と直接対応。スマートフォンの IMU (3 軸加速度計 + 3 軸ジャイロ) は前庭系と同じセンサ構成を持ち、Kalman filter による多センサ統合 (磁気・GPS・視覚オドメトリ) は脳の多感覚統合と同じ Bayes 推論。生物学が 5 億年かけて進化させた解と、人類が 80 年で工学的に再発明した解が同じ構造に到達している。

音楽の **アルヴォ・ペルト** や **ドビュッシー** の「揺らぎ」「曖昧さ」を聴くとき、前庭系も音楽体験に関与している可能性がある。ティンティナブリの揺らぐ和音、ドビュッシーの不確定な調性は、聴覚 + 前庭系の協調的な「方向喪失感」を生み出す。これが「漂うような音楽」の生理学的基盤かもしれない。
