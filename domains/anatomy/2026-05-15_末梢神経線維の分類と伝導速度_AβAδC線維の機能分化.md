# 末梢神経線維の分類と伝導速度：Aβ・Aδ・C線維の機能分化
**日付**: 2026-05-15  
**分野**: anatomy  
**タグ**: #末梢神経 #神経生理学 #伝導速度 #ミエリン #痛覚 #触覚

## 学んだこと

### Erlanger-Gasser 分類：神経線維の3つの軸

末梢神経線維の分類は、ノーベル賞受賞者 Herbert Gasser と Joseph Erlanger（1944年生理学・医学賞）によって体系化された。彼らは神経幹を電気刺激し、複合活動電位の波形を観察することで、伝導速度の異なる線維群がはっきり分離して現れることを発見した。分類の3軸：

1. **軸索直径**（大→小）
2. **ミエリン鞘の有無と厚さ**（厚いミエリン→薄いミエリン→無髄）
3. **伝導速度**（速い→遅い）

これら3要素は強く相関する。太い軸索ほどミエリンが厚く、伝導速度が速い。

### A線維（有髄）の4亜型

#### Aα線維 — 最速の運動・固有感覚軸索
- **直径**: 13-20 μm
- **伝導速度**: 80-120 m/s（最速、新幹線並み）
- **機能**:
  - 骨格筋への α運動ニューロン（運動指令）
  - 筋紡錘 Ia 求心線維（筋長変化）
  - ゴルジ腱器官 Ib 求心線維（張力変化）
- **臨床的意味**: 反射弓の速度を決定。膝蓋腱反射の潜時（30-40ms）は Aα 線維の伝導速度で決まる

#### Aβ線維 — 触覚・振動覚の高速路
- **直径**: 6-12 μm
- **伝導速度**: 33-75 m/s
- **機能**:
  - 皮膚機械受容器（メルケル盤、マイスナー小体、パチニ小体、ルフィニ終末）から
  - 触覚、圧覚、振動覚を伝達
  - 筋紡錘 II 求心線維としても機能
- **臨床的意味**: ピアニストやゴルファーの繊細な触感は Aβ 線維による（[[指先の皮膚機械受容器と能動的触知覚]]）

#### Aγ線維 — 筋紡錘の遠心制御
- **直径**: 5-8 μm
- **伝導速度**: 4-24 m/s
- **機能**: γ運動ニューロンとして錘内筋線維（intrafusal）を制御し、筋紡錘の感度を調整
- **臨床的意味**: γ運動系の活性化は筋緊張・姿勢制御の基盤（[[筋紡錘とゴルジ腱器官の固有受容感覚]]）

#### Aδ線維 — 「速い痛み」と冷覚
- **直径**: 1-6 μm
- **伝導速度**: 5-25 m/s（陸上選手並み）
- **機能**:
  - 鋭い、局在化された痛み（「Fast pain」）
  - 冷覚
  - 一部の触覚（毛のなびき検出など）
- **臨床的意味**: 怪我した瞬間の「鋭い痛み」は Aδ 線維が0.1秒以内に脊髄に到達するため

### B線維 — 自律神経の節前線維
- **直径**: <3 μm（軽くミエリン化）
- **伝導速度**: 3-15 m/s
- **機能**: 交感神経・副交感神経の節前ニューロン（preganglionic）
- **臨床的意味**: [[迷走神経と自律神経バランス]] における自律神経出力の高速経路

### C線維 — 無髄、遅い痛覚と温覚
- **直径**: 0.4-1.2 μm
- **伝導速度**: 0.5-2.0 m/s（歩くより遅い）
- **機能**:
  - 鈍い、持続的な痛み（「Slow pain」）
  - 温覚（warm > 30°C, cold < 30°C）
  - かゆみ（histaminergic と non-histaminergic）
  - 内臓求心
  - 自律神経節後線維（postganglionic sympathetic）
- **臨床的意味**: 慢性痛、内臓痛、灼熱感はすべて C 線維。神経ブロック剤（局所麻酔）は細い線維ほど効きやすいので、C → Aδ → Aβ → Aα の順に遮断される

### 伝導速度の物理学：ミエリンと跳躍伝導

無髄線維では活動電位が連続的に膜上を伝わる（continuous conduction）。これは脱分極が膜全体に順次広がる過程で、速度は軸索直径の **平方根** に比例する（v ∝ √d）。

有髄線維ではミエリン鞘が絶縁体として働き、活動電位は Ranvier の絞輪（node of Ranvier）でのみ生成される。電気的に1mm 程度の距離を「飛び越して」次の絞輪に到達する（saltatory conduction = 跳躍伝導）。この場合、速度は **軸索直径に線形** に比例する（v ∝ d）。

→ 太い軸索でミエリンが厚いほど、加速度的に速くなる。Aα の 120 m/s と C の 1 m/s は実に **100倍の差**。

### Fast Pain と Slow Pain の二重痛覚

指を切ったときの経験を分解すると：
1. **0-0.5秒**: 鋭く局在化した「痛い！」 → Aδ 線維による Fast pain
2. **数秒後**: 鈍く広がる、灼けるような痛み → C 線維による Slow pain

Fast pain は脊髄視床路（新脊髄視床路 / neospinothalamic tract）を経由し、体性感覚野に局在化情報として届く。Slow pain は古脊髄視床路（paleospinothalamic tract）を経由し、辺縁系（島皮質、前帯状皮質）に情動的情報として届く。**痛みには「位置」と「不快感」の2軸がある**。

### 脱髄疾患と神経伝導検査（NCS）

ミエリンが破壊される病態：
- **多発性硬化症（MS）**: 中枢神経のミエリン障害
- **ギラン・バレー症候群（GBS）**: 末梢神経の急性脱髄
- **シャルコー・マリー・トゥース病（CMT）**: 遺伝性末梢神経障害

これらでは伝導速度が劇的に低下する（健常値 40-60 m/s → 病態 10-20 m/s）。神経伝導検査（NCS）は神経幹を表面電気刺激し、遠位筋の活動電位（CMAP）を記録して伝導速度を計測する。臨床診断の柱。

### 絞扼性ニューロパチーと「太い線維から先に障害される」

[[絞扼性ニューロパチーとピアニスト・ゴルファー]] で見たように、神経が機械的に圧迫されると、太く速い線維（Aα, Aβ）から先に機能が落ちる。これは：
- 太い線維ほど機械的圧迫に弱い（脆弱な髄鞘の表面積比）
- 太い線維ほど高い代謝要求（虚血に弱い）

ため。臨床的には「最初に運動機能（Aα）と触覚（Aβ）が落ち、痛覚（Aδ, C）は最後まで残る」というパターン。

## 気づき・洞察

**「速さ」と「機能」の相関**は、生物学的に必然性がある。運動・反射には100ms 以内の応答が必要 → 太いミエリン化軸索が要る。慢性痛・内臓感覚は秒単位で十分 → 細い無髄線維で OK。**生物は速度のコストを払うべき場所と払わない場所を選別している**。これは技術的トレードオフのアナロジー：Service Mesh で「L4 は ztunnel で高速処理、L7 は waypoint で柔軟処理」と分けたのと同じ発想。

**「痛覚の二重構造」が運動学習に与える示唆**。Fast pain は瞬時の警告信号として手を引かせる（脊髄反射）。Slow pain は学習信号として「この動きは危険」を皮質に刻む（記憶形成）。**痛みは即時の防衛と長期の学習という2つの役割を担うシステム**。ピアノやゴルフの過練習で生じる慢性痛は C 線維シグナルであり、これを無視すると皮質可塑性が誤学習する。

**Erlanger-Gasser 分類はゴルファー・ピアニストの感覚を分析する道具**。クラブのインパクト感触は Aβ（パチニ小体、振動覚）と Aα（筋紡錘、筋緊張感）の統合。バンカーショットでの「砂を捉えた感じ」は Aβ 振動覚。テクニックを「主観的感覚」だけでなく「どの線維がどの情報を伝えているか」で分析できる。

## 他分野との接続

- **piano / 触覚と演奏**: 鍵盤の重さ・反発の感知は Aβ 線維。ppp の繊細さ（[[弱音pppのコントロールとエスケープメント]]）は Aβ 高感度の世界
- **golf / 感覚**: クラブグリップ感、インパクト振動の知覚は Aβ。S&T でコンタクト感を一貫させるのは Aβ シグナルの再現性を高める訓練
- **music / 演奏家の慢性痛**: 演奏家の局所性ジストニア、慢性痛は Aδ/C 線維と皮質可塑性の関係。フォーカルジストニアでは Aβ 求心とα運動の対応が壊れる
- **cs / 通信プロトコル**: Aα = 高速優先（TCP高速パス）、C = 信頼性重視・低帯域（背景タスク）。生体神経系は **QoS 階層化** された通信網
- **tech / 階層プロトコル**: Service Mesh が L4/L7 を分離するのと、神経系が「反射弓は高速、認知は遅い」と分離するのは構造的同型
