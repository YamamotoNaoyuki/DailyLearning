# 喉頭・声帯の解剖と発声メカニズム
**日付**: 2026-05-04
**分野**: anatomy
**タグ**: #喉頭 #声帯 #発声 #body-cover理論 #Hirano #Sundberg #appoggio #反回神経 #フォルマント

## 学んだこと

### 1. 喉頭の骨格軟骨構造
喉頭は **9 個の軟骨** から構成される。**不対 3 個**（甲状軟骨・輪状軟骨・喉頭蓋）と **対 3 対**（披裂軟骨・小角軟骨・楔状軟骨）。
- **甲状軟骨 thyroid cartilage**: 最大、硝子軟骨。前面正中で左右板が癒合し **laryngeal prominence（喉仏）** をなす。男性で角度約 90°、女性で約 120°（思春期テストステロンによる前突で男声低音化）。
- **輪状軟骨 cricoid**: 硝子軟骨、**唯一気管周囲を完全に取り囲む環状軟骨**（signet ring）。気道の構造的基盤。
- **披裂軟骨 arytenoid**: ピラミッド状の対軟骨、輪状軟骨の上後縁にのる。**声帯突起 vocal process**（前方・声帯付着）と **筋突起 muscular process**（外側・後輪状披裂筋付着）の二突起構造が、内転・外転・伸長の幾何学を決める。
- **喉頭蓋 epiglottis**: 弾性軟骨。嚥下時に喉頭口を閉鎖。歌唱では epilaryngeal narrowing で singer's formant 形成に寄与。

### 2. 声帯の組織学的 5 層構造（Hirano 1974）
表層から：
1. **上皮 epithelium**（重層扁平上皮、極薄）
2. **表層粘膜固有層 superficial lamina propria（SLP）= Reinke's space**：疎な elastin と HA に富むゼリー状層、**最大の振動振幅**を生む
3. **中間層 intermediate lamina propria**：弾性線維豊富
4. **深層 deep lamina propria**：膠原線維豊富（中間層 + 深層 = **声帯靭帯 vocal ligament**）
5. **声帯筋 vocalis（甲状披裂筋内側部）**

### 3. Body-Cover 理論と粘膜波形
Hirano は 5 層を機械的に **3 ユニット** に再編：
- **Cover** = 上皮 + SLP（柔らかく、自由振動）
- **Transition** = 声帯靭帯
- **Body** = 声帯筋

カバーは body から弱結合し、**mucosal wave**（粘膜波）として下方→上方へ伝播（高速ストロボ撮影で観察）。**Myoelastic-aerodynamic theory（van den Berg 1958, JSHR）**：声門下圧 P_sub が閉じた声帯を押し開け、弾性復元 + 気流による負圧（Bernoulli 効果）で再閉鎖、自己持続発振。Titze (1980) は **収束-発散グロッタル形状の非対称性** こそが主要エネルギー伝達機構と定式化（Bernoulli 単独では不十分）。発振閾値圧 ~0.3 kPa、通常会話 0.5-1 kPa、フォルテ歌唱 2-3 kPa。

### 4. 内喉頭筋の機能区別
| 筋 | 起始-停止 | 作用 | 神経 |
|---|---|---|---|
| **甲状披裂筋 TA**（vocalis） | 甲状-披裂 | 声帯短縮・厚化、緊張増（body緊張） | 反回神経 RLN |
| **輪状甲状筋 CT** | 輪状-甲状 | 甲状軟骨を前下方回転 → 声帯**伸展・薄化** → F0 上昇 | **上喉頭神経外枝 SLN external** |
| **後輪状披裂筋 PCA** | 輪状後面-筋突起 | **唯一の外転筋**。筋突起後方牽引 → 声帯突起外側回転 | RLN |
| **外側輪状披裂筋 LCA** | 輪状側面-筋突起 | 内転（主作用）。声門前 2/3 閉鎖 | RLN |
| **披裂間筋 IA**（横/斜） | 披裂間 | 声門後 1/3 閉鎖（**唯一の不対筋**、両側 RLN 支配） | RLN |

PCA 麻痺は両側で **緊急気道閉塞**（甲状腺手術合併症）。

### 5. 声区 register の生理学
- **胸声 chest voice**: TA 優位。声帯短く厚い、振動部 body 巻き込み、第二倍音優勢、talk range（男 100-300 Hz）
- **頭声 head / falsetto**: CT 優位。声帯伸展し薄い、cover のみ振動、純音的
- **ミドル / mix**: TA-CT 拮抗。Kochis-Jennings 2014 (J Voice): **300 Hz 超では register に関わらず CT/TA 比 ≒ 1 または CT 優位**。古典的 "TA dominant chest" 像は低音域限定。
- 声区移行（passaggio）は **筋活動比とフォルマント整合の同時切替**。

### 6. 神経支配
両側とも **迷走神経 X** 由来。
- **上喉頭神経 SLN**: 内枝（声門上の感覚＋上皮反射）、外枝（**CT のみ運動支配**）。外枝麻痺で高音域消失（プロ歌手の最重要神経）。
- **反回神経 RLN**: **CT 以外の全内喉頭筋を運動支配**＋声門下感覚。**左 RLN は大動脈弓を回り** 全長 ~12cm（右は鎖骨下動脈で短い）。胸郭縦隔病変（甲状腺手術・大動脈瘤・肺尖癌 = Pancoast）で麻痺、嗄声の古典所見。

### 7. 声門下圧と気流
- 声門下圧 P_sub と F0：おおむね **3-5 Hz/cmH₂O**（Titze）。声量は P_sub の 2-3 乗に比例（dB SPL ∝ 約 10 log P²）。
- ベルヌーイ効果は声帯閉鎖の補助、主役はグロッタル形状の時間変動による自己発振。
- Open quotient（OQ）= 開放期/周期。falsetto OQ 高、pressed phonation OQ 低。

### 8. 共鳴系とフォルマント
声帯音源（鋸歯波スペクトル、-12dB/oct ロールオフ）が **vocal tract（咽頭+口腔+鼻腔）** で frequency-domain フィルタされる（**source-filter theory, Fant 1960**）。共鳴ピーク = フォルマント。
- **F1 ≈ 250-900 Hz**（vowel height、開口度・顎開きと正比例）
- **F2 ≈ 600-2500 Hz**（tongue backness、舌前後位置）
- **F3 ≈ 1500-3500 Hz**（lip rounding、咽頭形状）
- **歌唱フォルマント singer's formant**（Sundberg 1974）: **F3-F4-F5 が 2.5-3.5 kHz でクラスタ**化、オーケストラエネルギーが減衰する周波数帯にスペクトルピークを生み増幅なしで突き抜ける。中心周波数：bass 2420 / baritone 2550 / tenor 2840 / contralto 3000 Hz。**喉頭管の狭め（喉頭管:咽頭 ≈ 1:6）** が必要条件。ソプラノは高音域で別戦略（formant tuning：F1 を F0 に合わせる）。

### 9. ブレスコントロールと appoggio
**appoggio**（伊「lean against」）は横隔膜の上昇を遅延させ呼気相を延長させる技法。
- **吸気保持筋**: 横隔膜・外肋間筋（吸気位胸郭維持）
- **呼気制御筋**: 内肋間筋・腹横筋・内外腹斜筋・腹直筋・骨盤底筋
- **腹横筋 transversus abdominis** が「コルセット」として腹腔内圧を均等に上げ、横隔膜上昇を**抑制**しながら制御する。これは骨盤底筋とのピストン同期と相同（既習：横隔膜と骨盤底筋のピストン機構）。
- ピアニストは「support」と意識せずとも**長句フレーズで自然に appoggio 様呼気**を行う（呼吸と打鍵タイミングの同期）。

### 10. 演奏者の職業病
- **声帯結節 vocal nodules**: 振動振幅最大点（前 1/3 と中 1/3 の境）に**両側性硬結**。"singer's nodules" / "screamer's nodules"。重度 voice misuse の集積。
- **声帯ポリープ**: 一側性、急性 vocal abuse（叫び・咳）。
- **Reinke 浮腫 polypoid corditis**: SLP の浮腫性肥大、**喫煙との強い因果**（GERD・甲状腺機能低下も）。低音化・嗄声、女性で目立つ（社会的影響大）。
- 治療階層：voice therapy → 行動修正 → microphonosurgery（cover 損傷を最小化する Reinke 空間温存術）。

### 11. 解剖学的「支え appoggio」
**横隔膜の遠心性収縮**（呼気時にもゆっくり下降を維持）+ 腹横筋による腹圧 + 骨盤底筋の同期収縮 + 外肋間筋による胸郭吸気位保持の **4 軸テンセグリティ**。Deep Front Line（DFL: 横隔膜-腸腰筋-骨盤底-内転筋）の機能単位そのもの。

## 気づき・洞察

- **「Body-Cover」は ピアノにも応用可能なメンタルモデル**: vocalis（質量・剛性）と SLP（自由振動）の役割分担は、指（実体）と関節 cover（潤滑・微振動）の関係に類比できる。
- **左 RLN の異常な迂回路**: 大動脈弓を回る ~12 cm の経路は進化的痕跡（魚類の 6 番鰓弓動脈に伴走）。「冗長な迂回が機能を残す」設計。
- **Singer's formant は 1:6 の幾何学的比** で生まれる純粋な物理共鳴 — 訓練とはこの幾何を再現可能にする筋協調学習。
- **CT/TA の拮抗** は弦楽器の張力（CT）と質量（TA）の独立制御に厳密に対応 — F0 = (1/2L)√(T/μ) の生体実装。
- **300 Hz 超では TA dominant chest という古典像は崩れる**（Kochis-Jennings 2014）。教則本の register 概念は近代生理学で書き換え進行中。

## 他分野との接続

- **piano**: appoggio = ピアノの「支え」（横隔膜・腹圧維持で打鍵安定）。フレージングは呼気制御と神経学的に同期。
- **anatomy（既習）**: 横隔膜・骨盤底筋・DFL・腸腰筋・前庭系（嚥下と発声の neural overlap）。
- **music**: フォルマント → 母音・歌詞知覚 → 楽器の音色設計（ヴァイオリンの "singer's formant" と同じ 3 kHz 帯ピーク = projection の物理）。
- **cs**: source-filter theory はデジタル信号処理 LPC（Linear Predictive Coding）の生理学的起源。声合成・音声認識の数理基盤。
- **golf**: 「臍下に集める」体幹圧 = appoggio 様の腹圧コントロール、インパクト時の Valsalva 様緊張と発声における支えは同じ筋連動。
