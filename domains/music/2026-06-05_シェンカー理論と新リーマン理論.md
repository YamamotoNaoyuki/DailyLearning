# シェンカー理論と新リーマン理論——「還元主義」対「変換主義」：音楽分析を支える二つの数理的世界観

> **この記録の新規性**: これまでの music エントリは一貫して「作曲家・作品の鑑賞分析」（バッハのフーガ、ワーグナーのトリスタン和音、R.シュトラウスの交響詩…）だった。本エントリは初めて**分析手法そのもの＝メタレベルの理論**を正面から扱う。曲を読むのではなく「曲を読むための装置」を読む。ラモーの和声理論（[[ラモーと和声理論の誕生]]）は18世紀の生成的和声論だが、本稿は20世紀の二大分析パラダイム（階層的還元 vs 群論的変換）の対比が主題。／ **関連既習**: [[ラモーと和声理論の誕生]]（basse fondamentale）／[[ワーグナー《トリスタンとイゾルデ》]]（半音階和声＝新リーマン理論が最も得意とする領域）

## 学んだこと

20世紀後半の調性音楽分析は、互いにほぼ正反対の数学的世界観を持つ二つのパラダイムに支配された。**ハインリヒ・シェンカー（Heinrich Schenker, 1868–1935）**の階層的還元理論と、**デイヴィッド・ルーウィン（David Lewin）／リチャード・コーン（Richard Cohn）**に始まる**新リーマン理論（Neo-Riemannian theory）**である。前者は「時間軸に沿った木構造への還元」、後者は「状態空間上の群作用」。同じ三和音の連結を、片方は深層構造への投影として、もう片方は変換の合成として記述する。

### 1. シェンカー理論：調性を1つの主和音の「引き伸ばし」に還元する

シェンカーの中心思想は逆説的に単純である——**一つの調性作品全体は、ただ1個の主和音（トニック三和音）を時間方向に引き伸ばしたものにすぎない**。彼の遺作 *Der freie Satz*（『自由作曲』, 1935）はこれを徹底する。

**3つの構造層（Schichten）。** 楽曲は三層に分節される：

- **前景（Vordergrund / foreground）**：楽譜の表面そのもの。すべての音符・装飾・リズム。
- **中景（Mittelgrund / middleground）**：装飾を剥がした中間的な声部進行の骨格（複数層ありうる）。
- **背景（Hintergrund / background）**：最深層。ここに**根本構造 Ursatz**が現れる。

**Ursatz（根本構造）と Urlinie（根本線）。** 背景は驚くほど貧弱だ。上声部は**Urlinie（根本線）**＝主和音の構成音から下行する全音階的進行のみを許す：`3̂–2̂–1̂`、`5̂–4̂–3̂–2̂–1̂`、または `8̂–…–1̂`。低声部は**Bassbrechung（バスのアルペジオ）**＝ `I–V–I`。両者を重ねると、Ursatz とは結局「`I–V–I` の上で旋律が主音へ全音階下行する」という、ほぼ一意な対位法的骨格になる。すべての真正な調性作品はこの一つの型に還元される、というのがシェンカーの主張だ。Ursatz は定義上**転調を含まない**（主和音の全音階的展開だから）——これが理論の限界と力の両方を規定する。

**鍵概念は「prolongation（引き伸ばし）」と「Auskomponierung（作曲的展開／composing-out）」**。ある和声（Stufe）が、実際には鳴っていない区間にもわたって支配力を及ぼし続ける——これが prolongation。`Auskomponierung` はその逆操作の生成版で、安定した和音に diminution（装飾＝刺繍音・経過音・アルペジオ・線的進行 Zug）を付加して表面を生成していく。

**Stufe（音度 / scale-step）の革命性**。ここがラモーとの決定的な分岐点だ。ラモーは各和音に basse fondamentale（根音）を与え、和音を**ローカルに**分類した（[[ラモーと和声理論の誕生]]）。シェンカーの **Stufe** は和音ではなく**ある音度に統べられた一区間全体**を指す。表面に現れる多数の和音は、より上位の単一の Stufe を composing-out したものにすぎない。つまりシェンカーは和声分析を**「ラベル付け」から「階層的支配の入れ子構造」へ**転換した。`ii–V–I` の連鎖も、上位から見れば一つの I の prolongation でありうる。**和声は局所記号ではなく、再帰的な入れ子（recursion）になった。**

**生成 vs 還元の方向性（重要な微妙さ）**。シェンカー自身は *Der freie Satz* で**生成的方向**——Ursatz から出発し層を降りて楽譜に至る——で記述した。しかし実務としての「シェンカー分析（Schenkerian analysis）」はほぼ常に**逆＝還元的**に行われる：楽譜から出発し、装飾を剥ぎ取って Ursatz へ収束させる。コーンの指摘では、*Harmonielehre* 以降のシェンカーは理論を「対位法による絶え間ない拡張」として展開し、*Der freie Satz* に至って**時間的連関が後退し、Ursatz からの導出という空間的・無時間的モデルが全てを覆う**。つまり後期シェンカーの世界像はすでに「時間」より「空間的派生」に傾いていた——ここが次の新リーマン理論との接続点になる。

### 2. 新リーマン理論：三和音を状態とし、変換を群作用とする

新リーマン理論は、シェンカーが苦手とする音楽——**機能和声が崩れた半音階的な後期ロマン派（ワーグナー、リスト、ブルックナー、後の映画音楽）**——を扱うために生まれた。「リーマン」は19世紀の音楽学者フーゴー・リーマン（Hugo Riemann）の二元論（major/minor の対称的扱い）に由来するが、現代版は**ルーウィンの変換理論（transformational theory, *GMIT*, 1987）**と**コーンの1996–98年の論文**で確立した。

**思想の転換：オブジェクトから関係へ**。ルーウィンの綱領は明快だ——「C major chord」「G major chord」という**対象**ではなく、対象を結ぶ**関係（変換）**に焦点を移す。変換を群の元として形式化し、和声進行を群の元の列として読む。

**PLR 変換**。24個の長短三和音（12 major + 12 minor）に作用する3つの基本変換。いずれも**2音を共有し1声部だけが動く（parsimonious voice leading）**：

| 変換 | 名前 | 共通2音 | 動く音 | 例 |
|------|------|---------|--------|-----|
| **P** | Parallel | 根音と第5音 | 第3音が半音 | C dur → c moll（E→E♭） |
| **L** | Leittonwechsel（導音交換） | 第3音と第5音 | 根音が半音上行 | C dur → e moll（C→H） |
| **R** | Relative | 根音と第3音 | 第5音が全音上行 | C dur → a moll（G→A） |

3つとも**自己逆元（involution）**：2回適用すると元へ戻る。P・L は1声部が半音、R だけ全音動く——この差が後で効いてくる。これらは「文脈的反転（contextual inversion）」：作用は major と minor で鏡像的に逆向きになる（リーマン二元論の継承）。

**群構造（ここが工学部向けの核心）**。P, L, R が生成する群を**PLR 群**と呼ぶ。これは**位数24の二面体群 D₁₂（＝ Dih₂₄）に同型**であり、24個の三和音の集合に**単純推移的（simply transitive）**に作用する——任意の三和音から任意の三和音へ、ちょうど一つの群元で到達できる。三和音集合は群そのものと一対一対応し（torsor／主等質空間）、「和声空間 = 群」となる。

**双対性（Lewinian duality）——最も美しい定理**。三和音にはもう一つ自然な群が作用する：**T/I 群**（12個の移調 T と12個の反転 I、これも位数24の二面体群）。驚くべきは、**PLR 群と T/I 群が対称群 S₂₄ の中で互いの中心化群（mutual centralizers / 互いの可換子）になっている**こと。両者は元ごとに可換で、各々が相手の完全な中心化群をなす。**同じ24元集合に、互いに双対な二つの単純推移的 D₂₄ 作用が乗っている。** これは群論の「自由作用＝正則表現の左作用と右作用」の音楽的実現にほかならない（左乗法と右乗法が互いの中心化群をなす、という古典的事実の具現）。

**Tonnetz（音度網／harmonic network）**。これらの変換を幾何化したのが Tonnetz だ。三軸——**完全5度・長3度・短3度**——を持つ三角格子で、各頂点が音名、各三角形が三和音（上向き三角形＝長三和音、下向き＝短三和音）。隣接する三角形は辺を共有＝2共通音で結ばれ、PLR はその「辺をまたぐ反転」に対応する。重要なのは**12平均律ではこの格子が周期的に閉じ、位相的にトーラス（torus）になる**こと——5度方向も3度方向も12（あるいは適切な周期）で巡回するため、無限格子が捻れて閉曲面に巻き込まれる。和声進行は**トーラス上の経路**として可視化される。

**極大平滑な周期軌道**。PLR を交互に合成すると周期軌道が現れる：

- **PL を交互**に適用 → **六声循環（hexatonic cycle）**：6個の三和音を巡る閉路（例：C dur–c moll–A♭ dur–a♭ moll–E dur–e moll–C dur）。24三和音はこうした六声系4つに分割される。各サイクルは**増三和音を共通核**に持つ。
- **RL を交互** → 八声循環（octatonic cycle）。

コーンはこれを **maximally smooth cycles（極大平滑循環）** と呼んだ。声部移動の総量を最小化する経路が、自然に閉じた幾何的サイクルをなす。**機能（dominant→tonic）ではなく、声部進行の倹約（parsimony）だけで和声論理を駆動できる**——これが新リーマン理論の革命だ。

### 3. 哲学的対比：還元主義 vs 変換主義

| | シェンカー理論 | 新リーマン理論 |
|---|---|---|
| 単位 | 階層・層（Schichten） | 変換（群の元） |
| 操作 | prolongation（引き伸ばし／還元） | PLR 等の変換合成 |
| 構造 | 木（再帰的入れ子）／生成文法 | 群作用／状態遷移グラフ |
| 駆動原理 | トニックの支配・対位法的展開 | 共通音保持・声部の倹約 |
| 時間 | 本質的（階層は時間の入れ子） | 非本質的（空間内の遷移） |
| 得意領域 | 機能的・全音階的調性（バッハ〜ブラームス） | 半音階的・非機能的進行（ワーグナー後） |
| 中心 | 一つの主和音（中心化された宇宙） | 中心なき対称空間（どの三和音も等価） |

シェンカーは**中心（トニック）を持つ階層的・神学的な宇宙**を描く。すべては一者へ還元される。新リーマンは**中心を持たない対称的な平坦空間**を描く（群作用は推移的＝すべての三和音が等価）。前者はモノテオロジー、後者は群論的民主主義だ。

## 気づき・洞察

- **シェンカー = コンパイラのAST／再帰下降、新リーマン = 状態機械上の群作用**、という工学的双対が見事に成立する。シェンカーの三層（前景→中景→背景）はソースコード→構文木→正規化された核への**還元（parsing + 定数畳み込み）**そのものだ。prolongation は「この区間は実質この一つの式に等しい」という**簡約規則**であり、Ursatz は全プログラムが還元される**正規形（normal form）**。実務が「還元的」なのは、コンパイラが具象構文から抽象構文へ昇るのと同じ方向。一方シェンカー自身の生成的記述は、文法からプログラムを**生成**する方向で、これは形式文法の生成 vs 構文解析の双対に正確に対応する。

- 対して新リーマン理論は**純粋に状態遷移**だ。三和音＝状態、PLR＝遷移ラベル。和声進行は**ラベル付き経路（trace）**。シェンカーには「正解の木」があるが（だから分析者間で論争になる）、新リーマンには木がなく、ただ経路があるだけ——**どの分解が正しいかを問わない**。これは「構文木は一意でない（曖昧文法）」問題を、そもそも木を作らないことで回避している。

- **Lewinian duality は群論の左右正則表現そのもの**。群 G が自分自身に左乗法・右乗法で作用すると、両作用は互いの中心化群をなす（これは Cayley 定理の精緻版）。T/I と PLR の双対は、24三和音という torsor 上でこの抽象的事実が**聴こえる形**で実現したもの。T が「絶対的な移調」（外から見た座標変換）なら、PLR は「和音内部から見た文脈的操作」（内部座標）——**受動変換と能動変換、あるいは座標基底の取り換えと点の移動**の双対に対応する。音楽理論がここまで素直に群論の核心に触れているのは驚異的だ。

- **「P・L は半音、R は全音」という非対称が幾何を歪める**。声部移動の倹約度が違うため、PL サイクル（六声）と RL サイクル（八声）で周期が変わり、Tonnetz トーラスの「測地線」の長さが変わる。コーンの maximally smooth cycle は、要するに**トーラス上の最短閉測地線**を探す問題に近い。後期ワーグナーが「機能を捨てても進行が滑らかに聴こえる」のは、彼が無意識にこのトーラス上の短い経路を辿っていたから、という説明が成り立つ——[[ワーグナー《トリスタンとイゾルデ》]]の半音階和声は、新リーマン空間で初めて「論理」を与えられた。

- 両理論は**互いの盲点を埋める相補物**だ。シェンカーは転調を Ursatz に含められない（全音階の檻）ので半音階音楽に弱い。新リーマンは中心と時間（どこが終わりか、なぜ I で閉じるか）を語れない。**機能的調性＝シェンカー、機能崩壊後＝新リーマン**という棲み分けは、調性史の臨界点（ワーグナー）でちょうどバトンが渡される。

## 他分野との接続

- **CS / コンパイラ理論**：前述の通り、シェンカーの還元は AST への parsing と簡約、Ursatz は正規形。生成的記述は文脈自由文法による生成。新リーマンはラベル付き遷移系（LTS）／オートマトン。**「曲を一意な構文木に還元できるか」という未解決の緊張**は、自然言語の構文的曖昧性と同型。

- **数学（群論・トポロジー）**：PLR 群 ≅ D₂₄、T/I との双対 = 正則表現の中心化群、Tonnetz = トーラス（商位相空間 ℝ²/格子）。**音楽が群作用と被覆空間の最も直感的な教材**になりうる。単純推移作用＝torsor の概念（「原点を選ぶまでベクトル空間にならないアフィン空間」）が、音楽では「絶対的に特権的な三和音はない」という美的主張に化ける。

- **piano 演奏**：シェンカー的階層は**演奏の「方向性」と暗譜の構造**に直結する。優れた演奏家は前景の音符を弾いているのではなく、中景の声部進行（どの音が構造的でどれが装飾か）を聴かせている——[[暗譜の科学]]で触れた「構造的記憶」は本質的にシェンカー的還元の内在化だ。一方、新リーマンの parsimony（共通音保持・最小声部移動）は、ピアノの**運指と手の移動量の最小化**と幾何的に対応する：共通音を保持する声部連結は、物理的にも指を残せる＝省エネな運動になる。

- **物理学**：T/I（外的座標変換）と PLR（内的文脈変換）の双対は、**受動変換と能動変換**、あるいはゲージ理論における「大域対称性 vs 局所（文脈依存）対称性」の構図に近い。同じ遷移を二つの異なる対称性群で記述できるという事態そのものが物理的に示唆的。

## 次に深掘りしたいこと

- **Tymoczko の幾何学的音楽理論**（*A Geometry of Music*, 2011）：新リーマン理論を一般化し、n音和音を**オービフォールド（orbifold）**上の点として扱う枠組み。PLR の torus を、より一般の声部進行空間（商空間で生じる特異点・鏡映境界）へ拡張する数理は何か。Callender–Quinn–Tymoczko の "Generalized Voice-Leading Spaces"（*Science*, 2008）まで射程に。

- **シェンカー分析の「非一意性」と計算機実装**：同一曲の Ursatz 還元が分析者間で割れる問題。これは曖昧文法の構文解析と同型のはずで、機械学習・確率文法による自動シェンカー分析（probabilistic grammar of tonal music, e.g. Rohrmeier の Generative Syntax Model）はどこまで成功しているか。

- **リーマンの原・二元論（Funktionstheorie）と現代版の乖離**：歴史上のフーゴー・リーマンの機能理論（T/S/D の三機能、和声二元論）と、ルーウィン以降の「新」リーマン理論が実はどれだけ違うのか。Riemann の Schritt/Wechsel と PLR の関係を一次資料で。

- **シェンカー理論のイデオロギー批判**：シェンカーの「有機的統一」「ドイツ天才の優越」という思想的背景（Schenker の政治的著述）と、それが理論の構造（中心化された一者への還元）にどう刻印されているか。Cohn / Suzannah Clark / 批判的音楽学（critical musicology）の議論。
