# パン生地の温度管理
**日付**: 2026-03-06
**分野**: パン
**タグ**: #DDT #生地温度 #発酵 #水温 #温度管理

## 学んだこと

### なぜ温度がパン作りの最重要変数なのか

パン生地の温度は発酵速度を直接支配する。**2°Cの温度上昇で発酵速度が約25%増加**する。つまり、同じレシピでも生地温度が24°Cと28°Cでは、発酵時間が大幅に異なり、結果として風味、食感、気泡構造が変わる。

プロのベーカリーが一定品質のパンを毎日焼ける理由は、スキルではなく「温度管理」にある。

### DDT（Desired Dough Temperature）——目標生地温度

パン作りの世界で最も重要な数値の一つ。

- **イースト発酵パン**: 24°C（75°F）
- **サワードウパン**: 25-26°C（78°F）

サワードウの方が高いのは、乳酸菌と酵母の最適温度帯が異なるため。25-26°Cは両者のバランスが最も良い「スイートスポット」。

### DDT計算式

バーカーが唯一コントロールできる変数は**水温**だ。他の要素（室温、粉温、摩擦熱）は環境依存で変更が難しい。

**ストレート法（直捏ね法）の場合：**
```
水温 = (DDT × 3) - (室温 + 粉温 + 摩擦係数)
```

**中種法（プリファーメント使用）の場合：**
```
水温 = (DDT × 4) - (室温 + 粉温 + 中種温度 + 摩擦係数)
```

### 摩擦係数（Friction Factor）

混練方法によって生地に加わる機械的エネルギーが異なる：

| 混練方法 | 摩擦係数 |
|----------|----------|
| 手ごね | 約3°C（5°F） |
| スタンドミキサー | 約11°C（20°F） |
| フードプロセッサー | 約14°C（25°F） |

摩擦係数はミキサーの速度、混練時間、生地量で変動する。正確な値は実測（混練前後の温度差）で求める。

### 実践例

夏（室温30°C、粉温28°C）、DDT 25°Cのサワードウを手ごねする場合：
```
水温 = (25 × 3) - (30 + 28 + 3) = 75 - 61 = 14°C
```
→ 冷水（冷蔵庫から出した直後の水）を使う必要がある。

冬（室温18°C、粉温16°C）、DDT 25°Cのサワードウを手ごねする場合：
```
水温 = (25 × 3) - (18 + 16 + 3) = 75 - 37 = 38°C
```
→ ぬるま湯を使う。

### 温度帯と発酵の特性

| 生地温度 | 発酵特性 |
|----------|----------|
| 20°C以下 | 遅い。酢酸（シャープな酸味）が支配的。冷蔵リタードの領域 |
| 22-24°C | ゆっくり。バランスの良い風味発達 |
| 25-26°C | 最適ゾーン。乳酸菌と酵母のバランス。サワードウの理想 |
| 27-28°C | やや速い。乳酸（まろやかな酸味）が増加 |
| 30°C以上 | 速すぎる。べたつき、扱いにくさ、過発酵リスク。酵母の活性は高いが制御困難 |

### 発酵中の温度維持

DDTは「出発点」に過ぎない。発酵中も温度を維持する工夫が必要：

- **オーブンのライト点灯**: 庫内が約27-28°Cに。安定した発酵環境
- **電子レンジ+熱湯カップ**: 密閉空間に湿度と温度を提供
- **冬の工夫**: 生地を毛布で包む、暖房器具の近くに置く（直接の熱風は避ける）
- **夏の工夫**: 冷蔵庫のバルクファーメンテーション。あるいは涼しい部屋へ移動

## 気づき・洞察

DDT計算式は、制御理論の基本そのものだ。「変えられない変数（室温、粉温、摩擦熱）」を測定し、「変えられる変数（水温）」を計算で求める。出力（生地温度）を一定に保つために、入力（水温）をフィードフォワード制御する。

これまで学んだサワードウの全プロセスが温度で繋がる：
- **温度→酵母/乳酸菌のバランス**（サワードウスターターの微生物学）
- **高温→乳酸、低温→酢酸**（冷蔵発酵の科学）
- **DDTで出発温度を制御→発酵時間と風味の予測可能性**（今回）

パン作りの全てが「温度管理」という一つの原理に収斂する。

## 他分野との接続

- **テクノロジー**: DDT計算式＝フィードフォワード制御。既知の外乱（室温、粉温）を測定し、制御入力（水温）を事前計算。制御工学の基本パターン
- **SDV**: 車載ECUの温度管理（サーマルマネジメント）とDDTは同じ課題。バッテリー温度を最適範囲に保つことが航続距離とバッテリー寿命を決定する
- **ゴルフ**: コースマネジメントの「環境を読む」と同じ。風速、気温、標高——変えられない環境変数を計測し、変えられる変数（クラブ選択、狙い目）を調整する
- **哲学**: DDT計算は「条件を正確に把握すれば、正しい行動が導かれる」合理主義。一方、道元の「只管打坐」は「条件を超えた実践」。両者の統合が「計算の上に直感を載せる」パン職人の技
