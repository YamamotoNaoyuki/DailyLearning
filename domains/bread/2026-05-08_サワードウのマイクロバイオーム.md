# サワードウのマイクロバイオーム — LABと酵母の共生生態系
**日付**: 2026-05-08
**分野**: bread
**タグ**: #sourdough #microbiome #lactobacillus #yeast #fermentation #ecology

## 学んだこと

### サワードウは「微生物の生態系」である

サワードウスターターは **小麦粉と水だけで安定した微生物生態系を維持する**、人類最古の biotechnology の一つだ。商業酵母（Saccharomyces cerevisiae）が単一菌株での発酵であるのに対し、サワードウは**乳酸菌（LAB, Lactic Acid Bacteria）と複数の野生酵母**の共生コミュニティで発酵する。

成熟したサワードウの微生物比率：
- **LAB:酵母 = 10:1 〜 100:1**
- 細胞数で換算すると、1g のサワードウに **LAB が 10⁹ cells/g**、**酵母が 10⁷ cells/g** 程度

LABが酵母の100倍多いのに、なぜ酵母（パンを膨らませる）が機能するのか。答えは「**LABは酸を作り、酵母はガスを作る**」という**機能分業**にある。

### 主要LAB種と代謝戦略

サワードウの代表的LAB：
- **Fructilactobacillus sanfranciscensis**（旧 Lactobacillus sanfranciscensis）: サンフランシスコ・サワードウブレッドの命名由来。最も研究された種
- **Lactiplantibacillus plantarum**（旧 L. plantarum）: 多様な基質で代謝可能
- **Lactobacillus paralimentarius**, **Levilactobacillus brevis** 等

LABの代謝タイプ：
- **Homofermentative（同質発酵）**: グルコース → 乳酸のみ。L. plantarum など
- **Heterofermentative（異質発酵）**: グルコース → 乳酸 + 酢酸 + CO₂ + エタノール。F. sanfranciscensis など

異質発酵LABが**サワードウの香り**を作る。乳酸が「**まろやかな酸味**」、酢酸が「**ツンとした酸味と香り**」を担当し、両者の比率（**Fermentation Quotient FQ = 乳酸/酢酸**）でパンの個性が決まる。

### F. sanfranciscensis の代謝詳細 — 共生の経済学

F. sanfranciscensis の特徴：
- **マルトース優先代謝**: 小麦粉の主要糖はマルトース（α-アミラーゼがデンプンを分解して生成）
- **マルトース取り込み**: 構成的に発現する**マルトース・ホスホリラーゼ**でマルトースをグルコースとグルコース-1-リン酸に分解
- **グルコースを排出**: F. sanfranciscensis は**グルコースを使わずに排出**する（**メイラード反応の強い香気成分の前駆体になる**）
- **フルクトース → マンニトール**: ヘテロ発酵経路でフルクトースを電子受容体として使い、マンニトールに還元する。これが甘み・口溶け感に寄与

つまり、F. sanfranciscensis は「**マルトースを食べてグルコースを残し、酵母にプレゼントする**」。野生酵母（Kazachstania humilis等）は**マルトースを使えない**ことが多く、グルコースを必要とする。**LABがマルトースを処理し副産物として酵母にグルコースを供給する**という**メタボリック・クロスフィーディング（代謝相互供与）**が生態系の安定の核心だ。

### 主要酵母種

サワードウの代表酵母：
- **Kazachstania humilis**（旧 Candida humilis）: 最も広く分布。マルトース negative
- **Kazachstania exigua**: 同様にマルトース不可
- **Saccharomyces cerevisiae**: 商業酵母と同種だが野生株。マルトース可能

興味深いのは **Saccharomyces cerevisiae は「商業パン用」、Kazachstania humilis は「サワードウ用」と棲み分け**ていること。商業酵母を加えても、長期維持されるサワードウでは野生酵母が優勢になる。理由：
- 野生酵母は**酸性環境（pH 3.5-4.5）に強い**
- LABの作る酸が S. cerevisiae の競争力を弱める
- 低温（25°C前後）で野生酵母が S. cerevisiae より速く増殖

### サワードウの「テロワール」— 起源は本当に違うのか

長年、「**サンフランシスコ・サワードウは現地の微生物コミュニティが特殊**」と信じられてきた。eLife (2021) の論文 *The diversity and function of sourdough starter microbiomes* では世界各地の500近いスターターをメタゲノム解析した結果：
- **地理的差異は限定的**。同じLAB種は世界中に分布
- **より大きな差は「メンテナンス方法」**：温度、リフレッシュ頻度、粉の種類、ハイドレーション
- **パン職人の手の微生物叢**がスターターと相関（!）

つまり「サンフランシスコの空気が違う」のではなく「**パン職人ごとの管理慣習**」がコミュニティを決める。これは生物学的なテロワールというより、**文化的なテロワール**だ。

### 環境因子と生態系シフト

スターターの微生物比率を決める要因：
| 因子 | 影響 |
|------|------|
| **温度** | 高温（30°C+）→ L. plantarum 優位（より乳酸多い）、低温（20°C）→ F. sanfranciscensis 優位 |
| **pH** | 低pH選択圧でLAB優位、酵母にとっては抑制 |
| **リフレッシュ頻度** | 頻繁（4-12h）→ 増殖期菌優位、低頻度→ 耐性菌優位 |
| **ハイドレーション** | 高水分（>100%）→ LAB優位、低水分（50-60%、stiff starter）→ 酵母優位 |
| **粉の種類** | 全粒粉→栄養豊富で多様性高、白粉→限られた種が優勢 |

これは**「同じスターターでも管理を変えると別の微生物コミュニティになる」**ことを意味する。Tartine Bakery（San Francisco）の Chad Robertson が**冷蔵保存からルームテンプ管理に切り替えた**時、スターターのキャラクターが変わったというエピソードはこの理論で説明できる。

### Postbiotic 効果と健康

サワードウ発酵は単なる「風味付け」を超えて、**ポストバイオティクス（postbiotic-like components）**を生み出す：
- **エキソポリサッカライド（EPS）**: LABが分泌する多糖。プレバイオティクス的に働く
- **フィチン酸の分解**: 全粒粉のフィチン酸（ミネラルキレート）をLABのフィターゼが分解 → 鉄・亜鉛吸収率向上
- **GIスケール低下**: 乳酸が **デンプンの結晶化（retrogradation）を促進**し、消化速度を遅らせる → サワードウは血糖上昇が緩やか
- **グルテン部分分解**: 長時間発酵で**プロリン残基がLABプロテアーゼに切断**される。完全なグルテンフリーではないが、軽度のglutene sensitivityには有効との報告

## 気づき・洞察

サワードウは**「単一性能の最大化ではなく、生態系の安定性が目的」**という珍しい発酵プロセスだ。商業酵母は「最大限のCO₂を最短時間で」と単一目的最適化されているが、サワードウは「**安定的に維持できる多様性**」を優先する。これは：

- **モノカルチャー vs エコシステム**: 単一菌株の脆弱性 vs 多様性の頑健性
- **短期効率 vs 長期持続**: バッチ発酵 vs 永続的スターター
- **画一性 vs 個性**: 工場パン vs 職人パン

の対比そのもの。**マイクロバイオーム研究の知見が「バゲット工房」というローテクな現場と直結**しているのが面白い。

「**LABがマルトースを処理し、酵母にグルコースを供給する**」というクロスフィーディングは、**生態系における専門化と相互依存**の教科書的事例。これは経済学のRicardian比較優位（同じ資源でも分業すると総生産量が増える）と同じ原理が、**何百万年もかけて微生物進化で実装されている**ことを示す。

「テロワールは文化だ」という発見は重要だ。ワインや日本酒で「土地の微生物が独特」と言われるが、サワードウ研究はそれが**ロマンチックな誤解で、実は管理慣習の差**であることを示唆する。**「自然」と思われているものが実は「文化」**——これは音楽の「自然なフレージング」が実は時代様式の習得結果であるのと同じ構造。

## 他分野との接続

- **anatomy (横隔膜の二重機能)**: 横隔膜が「呼吸 + 姿勢」の二機能を担うように、F. sanfranciscensis は「酸生成 + 糖供給」の二機能を担う。**多機能要素が生態系を安定化させる**
- **CS (分散システム、CAP定理)**: サワードウのLAB:酵母 10:1〜100:1比率は「**ローカル最適でない、生態系最適**」な平衡。分散システムにおける**最終的整合性（eventual consistency）**と類似
- **golf (Strategic course design = 多様性)**: MacKenzieの「**コースは多様性を持て**」原則は、サワードウが多様な微生物を保つことで安定するのと同型
- **tech (eBPF program types)**: tracepoint/kprobe/uprobe等の**異なるフックタイプの組み合わせ**で観測の多様性を確保するのは、LABと酵母の役割分担と類似
