# グリアジンとグルテニン — 小麦タンパク質の分子構造と粉の使い分け
**日付**: 2026-05-02
**分野**: bread
**タグ**: #小麦タンパク質 #グルテン #グリアジン #グルテニン #ジスルフィド結合

## 学んだこと

### 小麦の貯蔵タンパク質と Osborne 分類
小麦粒中のタンパク質はおよそ 8-15%。1907 年に Thomas Osborne が溶解性で 4 種類に分類した：
1. **アルブミン** (水溶性) — 約 15%、酵素が中心
2. **グロブリン** (希塩水溶性) — 約 5%
3. **グリアジン** (60-70% エタノール可溶) — 約 30-40%
4. **グルテニン** (希酸/希アルカリ可溶) — 約 30-40%

このうち「グリアジン + グルテニン = グルテン (湿) ≒ 80%」。これがパン・麺の物性を決める主役。アルブミン・グロブリンは焼成過程で大半が変性するため、製パン的にはマイナーな扱い。

### グリアジン — 単量体タンパク質
グリアジンは **モノマー** (単量体) で、分子内ジスルフィド結合は持つが分子間ジスルフィド結合を作らない。分子量 28-55 kDa。α/β/γ/ω の 4 サブクラス。

機能的特性: **粘性 (viscosity) を担う**。生地に「流動性・伸展性」を与える。グリアジン分率が高い粉は「だれやすく、伸びやすい」生地になる。

### グルテニン — 巨大重合体
グルテニンは **ポリマー** で、分子間ジスルフィド結合 (-S-S-) で多量体を形成する。サブユニットは 2 種類：
- **HMW-GS (High Molecular Weight Glutenin Subunit)** — 65-90 kDa。少数だが製パン適性に大きく寄与。
- **LMW-GS (Low Molecular Weight Glutenin Subunit)** — 30-45 kDa。数は多い。

これらが S-S 結合で巨大網目を形成し、最終的な分子量は数百万 Da まで達する。これは天然の生体高分子としては最大級のサイズ。

機能的特性: **弾性 (elasticity) を担う**。生地に「腰・形状保持力」を与える。グルテニン分率が高い粉は「ガス保持力が高く、立ち上がりの良い」パンになる。

### Gluten = グリアジン + グルテニン の協調
グルテンは「粘弾性体 (viscoelastic)」と呼ばれる。これは「粘性 (グリアジン) + 弾性 (グルテニン)」の両方を持つから。

- グリアジンだけ → 流動性が高すぎて、ガスを保持できない (ケーキ生地的)
- グルテニンだけ → 硬く伸びず、グルテン膜が破れる
- 両者の比率が適正 → 二次発酵で炭酸ガスを抱え込み、焼成で steam expansion (オーブンスプリング) が起きる

この比率 **グリアジン/グルテニン = ω-G/G ratio** は粉の用途を決める核心指標。

### 粉の用途とタンパク質量・質の対応
- **強力粉 (パン用)**: タンパク質 11-13%、HMW-GS 比率高、グルテニン強い → 食パン・バゲット
- **準強力粉**: タンパク質 10-12% → 菓子パン・うどん
- **中力粉**: タンパク質 9-10% → うどん、平麺、お好み焼き
- **薄力粉 (ケーキ用)**: タンパク質 7-9%、グリアジン優位、軟質小麦 → ケーキ・天ぷら

タンパク質量だけでなく **質 (HMW-GS のサブユニット組成)** が決定的。Glu-D1 遺伝子座の HMW-GS サブユニット 5+10 を持つ品種は製パン適性が高く、2+12 を持つ品種は低い。日本のパン用小麦「ゆめちから」「春よ恋」は 5+10 系統を持つ高品質パン用品種。

### ジスルフィド結合と還元剤・酸化剤
グルテン網目の強度は -S-S- 結合数に比例する。
- **酸化剤 (アスコルビン酸 = ビタミン C)**: -SH → -S-S- を促進し、生地を強くする。製パンで広く使用。
- **還元剤 (L-システイン, グルタチオン)**: -S-S- → -SH に切る。生地を弱くする (ピザ生地のリラクゼーション、ベーグル等で用)。

イースト (酵母) の自己融解物には還元型グルタチオンが含まれ、過剰な発酵 (overproof) で生地がだれるのはこの還元作用も一因。

### グルテン形成のステップ
1. **水和** — 粉に水を加えるとタンパク質粒子が水を吸って膨潤
2. **混捏 (mixing)** — 機械的力で分子鎖が伸ばされ、近接した -SH 基が出会う
3. **酸化** — 大気酸素 (および添加酸化剤) で -SH → -S-S- 形成
4. **熟成 (autolyse, bulk fermentation)** — 静置中にタンパク質分解酵素 (プロテアーゼ) と還元・酸化のバランスで網目が再編成。延性が増す。

オートリーズ (粉と水だけを 20-60 分静置) の科学的根拠はここにある: 機械的混捏なしに水和とゆっくりした酵素作用でグルテン形成が進む。混捏時間を短縮でき、生地温度の上昇を防ぐ。

### グルテンと健康問題
- **セリアック病**: グリアジンの 33-mer (ペプチド断片) が腸管の HLA-DQ2/8 に結合し、自己免疫反応を起こす。完全グルテンフリー食が唯一の治療。
- **NCGS (非セリアックグルテン過敏症)**: 機構不明、診断は除外診断のみ。
- **FODMAP 仮説**: 小麦の不耐は実はフルクタン (オリゴ糖) が原因で、グルテンではない例も多いとする説。

ライ麦のセカリン、大麦のホルデインも同じプロラミン族で、セリアック患者には不耐。一方オーツのアベニンは大半の患者で安全 (ただし加工汚染に注意)。

## 気づき・洞察

「強力粉」「薄力粉」という日本語の分類は実用的だが、**「タンパク質量 × タンパク質の質 (HMW-GS 組成) × グリアジン/グルテニン比」** という三軸の二次元射影に過ぎない。同じタンパク質量 12% でも、HMW-GS 5+10 の北米硬質春小麦と、HMW-GS 2+12 のオーストラリア小麦では製パン性が全く異なる。この事実は実家ベーキングでも「カナダ産マニトバ vs ゆめちから vs 春よ恋」で焼き比べると一目瞭然になる。

グルテン形成が **「機械的力で近接配置 → 化学反応で固定」** という二段階構造であることは興味深い。同じ原理がコラーゲン繊維の架橋形成 (リシルオキシダーゼによる -OH/-NH 架橋) でも見られ、生体の構造タンパク質の普遍的設計原理になっている。

セリアック病の発症メカニズムが「33-mer ペプチドの特定 HLA への結合」という分子レベルで解明されている事実は、栄養疫学の進化を象徴する。「悪い」「良い」ではなく、特定の分子 × 特定の遺伝子型の組み合わせが病態を作る。

## 他分野との接続

- **anatomy (腱コラーゲン階層構造 2026-04-29)**: コラーゲンの階層構造 (Tropocollagen → Microfibril → Fiber) と、グルテンの階層構造 (グリアジン/グルテニンモノマー → S-S 結合多量体 → 網目) は、両者ともジスルフィドや共有結合架橋で「機械強度を出す多階層タンパク質ネットワーク」という共通設計原理。
- **cs (確率的データ構造)**: グルテン網目は「決定論的設計」ではなく「確率的に -SH 基が出会って結合する」確率過程。Bloom Filter 的な「ある程度のばらつきを許容して大域的性質を達成する」発想と同型。
- **bread (湯種法 2026-04-27, ストーンミル製粉 2026-05-01)**: 湯種法は α アミラーゼで澱粉を糊化させる手法だがグルテンには (温度高すぎて) 損傷を与える。ストーン挽きは胚芽の脂質酸化を抑え、グルテン酸化反応にも影響する。粉の前処理ごとにグルテン形成戦略が変わる。
- **other (調香 2026-04-26)**: 香水のトップ・ミドル・ベースの三層構造と、グルテン網目の階層構造は、いずれも「分子量と揮発性/反応性の階層」で機能を実現する設計。
