# ピザ生地のナポリ式vsニューヨーク式の科学
**日付**: 2026-04-24
**分野**: パン・発酵
**タグ**: #ピザ生地 #ナポリピザ #NYピザ #AVPN #加水率 #熱伝達

## 学んだこと

### 2つの伝統の物理的基礎
ナポリ式ピザとニューヨーク式ピザは、同じ「小麦粉・水・塩・イースト」から出発するが、**焼成条件（温度・時間）の違い**が生地設計全体を規定する。これは別々の伝統というより、**オーブンの物理特性への最適化解**の違い。

| 項目 | ナポリ式 | NY式 |
|------|---------|------|
| 焼成温度 | 430-485℃ | 260-315℃ |
| 焼成時間 | 60-90秒 | 5-15分 |
| 加水率 | 55.5-62.5% | 60-67% |
| 脂質 | なし（伝統） | オリーブオイル少量 |
| 糖 | なし | 少量（砂糖・モルト） |
| 発酵時間 | 8-24時間、室温 | 24-72時間、冷蔵 |
| 強力粉 | W280-320（中程度） | W300-350（強い） |
| サイズ | 直径約30cm（個別） | 直径約45cm（スライス） |

### AVPN規定の正統ナポリピザ
Associazione Verace Pizza Napoletana (AVPN)——1984年設立の真正ナポリピザ協会——は国際的に規格を定めている。

**規格の核**:
- **小麦粉**: "00"挽き、タンパク質11-12.5%、W 220-380
- **水**: 1リットル、**塩 40-50g**（3%前後、比較的高め）
- **イースト**: 生イースト1-3g、または自然発酵
- **加水率 55.5-62.5%**
- **手ごね、または低速ミキサー**
- **薪窯、炉床温度430-485℃、ドーム485℃超**
- **焼成60-90秒**

手で1点から広げ（麺棒禁止）、周縁の"**コルニチョーネ（cornicione）**"——焼き膨れた縁——が3-4cmになること。中央は薄く焼き、端は気泡で膨らむ。

### NYピザの起源と技術
NYピザは**1905年Gennaro Lombardi**がマンハッタンに開業した最初のピザ店から始まる。ナポリ移民の技法だが、アメリカでは薪窯が使えずガス・石炭オーブンで焼いた。**低い温度を時間で補う**設計が生まれた。

- 石炭オーブンで315℃前後、5-7分
- 生地に**オリーブオイルと砂糖を少量**添加——低温焼成での褐変（メイラード反応）と柔軟性のため
- **コールドファーメンテーション（冷蔵長時間発酵）**——24-72時間、味の複雑化
- 45cm超の大サイズ、切り売り（**"slice"**）文化
- 「**フォールド・テクニック**」——ピザを縦に折って食べる
- セモリナ粉やコーンミールで底面を仕上げる

### 加水率と焼成時間の逆説的関係
直感では「高温で焼くほど水分が早く蒸発するから、高加水が良い」と思えるが、実は**逆**。

- **ナポリ式（高温・短時間）**: 低加水（55-62%）。**水分蒸発の時間がないため**、元から水を少なくしないと生焼けになる
- **NY式（低温・長時間）**: 高加水（60-67%）。**長い焼成時間で水分が逃げる余裕がある**、高加水で気泡を育てつつクラムを軽くできる

つまり**水は焼成時間の関数**として設計する。これは昨日のクロワッサンで学んだラミネーション（バター層の挙動）とも通じる——温度プロファイルが生地設計を決める。

### グルテン発達とタンパク質強度
- **ナポリ式**: 60-90秒の焼成で構造形成が終わるため、**中程度のタンパク質**（W220-320）で十分。強すぎると噛み切れない
- **NY式**: 生地が大きく、持ち上げて食べるため**機械的強度が必要**。強いグルテン（W300-350、high-gluten flour、タンパク質13-14%）で引張強度を確保

**"New York style"専用の高グルテン小麦粉**が商業化されている（King Arthur Sir Lancelot等）。

### 発酵時間と風味
ナポリ式は**室温でのバルク発酵**（8-24時間）で、比較的単純だが生のトマト・モッツァレラ・バジルの鮮烈さを邪魔しない味。

NY式は**冷蔵72時間発酵**が伝統的。低温では酵母活動は緩慢だが、**乳酸菌による有機酸生成**、**酵素によるタンパク質・デンプン分解**が進み、複雑なフレーバーが生まれる。これは3月に学んだ長時間冷蔵発酵の科学そのもの。

### 熱伝達の違い
- **ナポリ式**: 薪火の**輻射熱**がドームから反射し、炉床の**伝導熱**と合わせて上下から攻める。ピザは空気がほぼ動かない環境で急速加熱（**~60秒で炉床から1000Wの熱流束**）
- **NY式**: ガス/電気オーブン、**対流熱**主体。低温で時間をかけ、**デッキオーブン**（石床）で底から焼き、天井のバーナーで上部を焼く

この熱伝達モードの違いが、焦げ方（ナポリ=レオパード・スポッティング／NY=均一な茶色）、チーズの溶け方（ナポリ=とろり／NY=完全に溶けて油が分離）、クラムの気泡（ナポリ=大きく不均一／NY=小さく均一）の違いを生む。

## 気づき・洞察

### 同じ材料でも"温度プロファイル"が全てを決める
小麦粉・水・塩・イーストの組成は似ているのに、**窯の温度/時間の違いが、加水・タンパク質・発酵・成形・食べ方まで規定する**。材料ではなく**プロセス条件が料理を決める**という、発酵科学全般に通じる洞察。

### "制約が美学を生む"
ナポリの薪窯 → 短時間焼成 → 低加水 → 大きな気泡 → 手で折りたたむ食べ方。
NYの石炭オーブン → 低温長時間 → 高加水 → 柔軟性 → フォールドして食べる。

**物理的制約が技法を規定し、技法が料理の形と食文化を生む**。ナポリピザが直径30cmなのもNYピザが45cmなのも、起源のオーブンサイズに縛られている。

### 加水率は"湿度"ではなく"熱輸送の遅延バッファ"
パンづくりで加水は「しっとり感」と捉えがちだが、ピザ生地では**水は熱輸送の遅延バッファ**として働く——蒸発に大量の熱を奪うため、生地表面が焦げる前に内部に熱を伝える時間を稼ぐ。加水率は「ジューシーさ」ではなく「**焦げずに火を通すための工学変数**」。

### 冷蔵発酵はNY式の"低速な秘密兵器"
ナポリの味の鮮烈さは素材の新鮮さから。NYの深みは**時間をかけた発酵の複雑な代謝産物**から。低温で乳酸菌が優勢になり、pHが徐々に下がって酸味の層が加わる。オーブンで失った温度を発酵庫で取り戻している。

### 食べ方までが料理の一部
「折って食べる」NYスライスの背景には、**通勤中に立ち食いする文化**がある。大きく薄く、冷めても硬くならない、片手で食べられる。ピザの物理設計は**食べる人の生活様式**にまで反映されている。一方ナポリピザは座って食事する前提の、ナイフとフォークで切る（または手でも）料理。

## 他分野との接続

### golf分野（今日のリンクスゴルフ）との類比
リンクスゴルフ（グラウンドゲーム）はエアリアル（空中）を諦めて地面にボールを預ける戦略。NYピザは高温（空中熱）を諦めて時間（プロセス熱）に預ける戦略。**直接的な高出力解法が使えないとき、プロセスの時間で代替する**という構造的共通点。

### tech分野（今日のDurable Objects）との類比
DOは「グローバルに分散した状態」を諦めて「エンティティごとに局所化」する設計。ナポリピザは「オーブン能力の均一化」を諦めて「薪窯の輻射熱を活用」する設計。**制約をそのまま受け入れて最適化する**という共通の知恵。

### music分野（今日のスクリャービン）との類比
スクリャービンの神秘和音は「3度積み重ね」という西欧の構文を捨て「4度積み重ね」で書き直した。NYピザはナポリの「00粉・薪窯」という前提を捨て「高グルテン粉・石炭オーブン」で書き直した。**伝統の再設計**はどちらも新しい美学を生む。

### cs分野（今日のHindley-Milner）との接続
HM型推論は「関数が受け付ける型を推論する」——入力範囲の最適化。ピザ生地設計は「オーブン条件が受け入れる加水を推論する」——入力条件の最適化。**制約条件から最適解を逆算する**という構造が共通。

### anatomy分野（今日の小脳）との接続
熟練ピザ職人の手作業——低加水の硬い生地を広げ、コルニチョーネを潰さない——は、明らかに小脳の運動学習の産物。「薄すぎる」「厚すぎる」のフィードバックでforward modelが精密化され、**見ずに広げても正しく伸びる**ようになる。
