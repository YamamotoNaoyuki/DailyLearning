# パンのクラスト形成 ― 表面温度勾配と糊化・Maillard・カラメル化の連続物語
**日付**: 2026-05-14
**分野**: bread
**タグ**: #Crust #Maillard #Caramelization #HeatTransfer #Gelatinization

## 学んだこと

### クラスト ― パンの「皮膚」のような境界相
**クラスト (crust)** はパンの表面で形成される **乾燥・着色・硬化した層**で、内部のクラム (crumb) とは全く異なる物理化学状態を持つ。厚さは典型的に **2-5 mm**、含水率は **5-15%**（クラムは 40-45%）、密度はクラムの 1.5-2 倍。**「同じ生地が、表面と内部で全く異なる構造に分岐する」** ことがパン焼成の本質的な不思議で、その分岐を駆動するのが **熱と水分の表面勾配**。

### 焼成中の温度勾配 ― 4 つの相
オーブン投入直後から焼成終了まで、パンの表面と中心では全く異なる時間スケールの変化が並行する：

**相 1: オーブンスプリング (0–10 分)**
- 表面温度: 100°C 未満（蒸気と生地表面の蒸発冷却で）
- 中心温度: 30–60°C
- 重要現象: 酵母の最終発酵（35-50°C で最大活性、60°C で死滅）、**生地表面が湿潤を保つことで弾性的に膨張可能**
- **スチームの役割**：オーブン内蒸気が表面に凝結し、表面温度を 100°C 以下に抑える → クラスト形成を遅らせて生地膨張を最大化

**相 2: 糊化と凝固 (5–20 分)**
- 表面温度: 100-140°C
- 中心温度: 60-80°C
- **デンプン糊化 (gelatinization)**：55-80°C で開始、デンプン粒が水を吸って膨潤・破裂 → ゲル網目を形成
- **グルテン凝固 (coagulation)**：60-70°C で開始、70-80°C で完了 → 弾性ネットワークが硬化
- **生地構造の「ロック」**：この段階で気泡構造が固定、これ以後膨張しない

**相 3: 表面乾燥と着色開始 (15–30 分)**
- 表面温度: 140-160°C（水分が局所的に < 5%、表面の水分活度 a_w が急落）
- 中心温度: 90-95°C（沸点に達するが、水蒸気は内部圧上昇のみで蒸発できず）
- **メイラード反応 (Maillard reaction) 開始**：140-165°C でアミノ酸＋還元糖の縮合が加速、メラノイジン（茶色色素）と数百種の香気成分が生成
- 水分蒸発潜熱が表面を冷却するため、水分が枯渇するまで温度は上がりにくい

**相 4: カラメル化と完成 (25–45 分)**
- 表面温度: 170-200°C
- 中心温度: 95-98°C
- **カラメル化 (caramelization)**：糖単独の熱分解（160-180°C 以上）、ショ糖は約 180°C、グルコースは約 150°C で開始
- メイラードは依然継続、メラノイジンが深色化
- **完了の合図**：中心温度 95-98°C 到達、表面の鳴き音（クラックリング）、底を叩いた中空音

### Maillard 反応の化学詳細
**Maillard 反応**（1912 年に Louis-Camille Maillard が発見）は **還元糖**（グルコース、フルクトース、マルトース）と **遊離アミノ酸または遊離アミノ基** の縮合反応：

1. **第 1 段階: Amadori 転位**
   - 糖のカルボニル基がアミノ基と縮合 → Schiff base → Amadori 化合物
   - 50-100°C で可逆的に進行、まだ着色なし

2. **第 2 段階: Strecker 分解とエノール化**
   - 100-150°C で Amadori 化合物が分解 → α-ジカルボニル化合物 → 多様な香気分子
   - **Strecker aldehyde**: アミノ酸特有のアロマ（フェニルアラニン → フェニルアセトアルデヒド「バラ・蜂蜜」、メチオニン → メチオナール「焼きジャガイモ」、ロイシン → 3-メチルブタナール「モルティ」）

3. **第 3 段階: 重合とメラノイジン形成**
   - 高分子量褐色色素 **melanoidin** の生成
   - 抗酸化性・微弱な抗菌性を持つ

### カラメル化の化学
**カラメル化**は糖単独の熱分解（アミノ酸不要）：
- ショ糖: 約 160-180°C で分解開始
- フルクトース: 約 110°C（最も低温）
- グルコース: 約 150°C

経路：糖 → エノール化 → 脱水 → **HMF (5-ヒドロキシメチルフルフラール)** → 重合 → カラメラン・カラメレン・カラメリン（褐色色素）+ **ジアセチル、フラン、マルトール、ピラジン**（カラメル香気）。

メイラードとカラメル化は **連続的に起こる**――表面では同時並行し、互いの分解産物が反応に参与する。香味は両者の合作。

### 水分活度 (Water Activity, a_w) の決定的役割
クラスト形成を支配するのは温度だけでなく **水分活度 a_w**（純水基準の蒸気圧比）。クラム部は a_w ≈ 0.94-0.96、クラスト部は a_w 0.3-0.5 まで低下。**Maillard 反応は a_w 0.6-0.7 で最速**となる――水が少なすぎても多すぎても進まない。だからパンの **クラスト表面（中央部）で最も Maillard が進む**：表面の最外層は乾燥しすぎ、内部は水分過剰、その中間層で反応が爆発的に進行する。

### スチームの精密な役割
家庭でも商業でもダッチオーブン・スチームインジェクションを使うのは：
1. **表面温度を 100°C 以下に保ち**、生地膨張時間を最大化（オーブンスプリング延長）
2. **表面のデンプン糊化を促進**、ガラス化した糊化デンプンがツヤ（gelatinized gloss）を生む
3. **表面の水分が一時的に増えることで Maillard 開始 a_w に到達**（あまりに乾いた表面はメイラードを起こしにくい）

20 分以降に **スチームを止めて乾燥環境に切り替える**ことで、表面が乾燥・硬化・着色する。**「最初は湿らせ、後で乾かす」二相戦略**がプロのオーブン操作の核心。

### クラストの香気プロファイル
GC-MS 分析（Schieberle et al., 1998）が示すパン皮の主要香気成分：
- **2-Acetyl-1-pyrroline**（ジャスミン米様の焼きたて香、Maillard 由来、閾値 0.1 ng/L という超低閾値）
- **3-Methylbutanal**（モルティ、ロイシン由来）
- **Methional**（焼きジャガイモ様、メチオニン由来）
- **Phenylacetaldehyde**（蜂蜜・バラ様、フェニルアラニン由来）
- **2,3-Butanedione (diacetyl)**（バター様、カラメル化と発酵両方から）
- **Furaneol**（カラメル・苺様、Maillard 副生成物）

これらが集合して **「焼きたてパンの匂い」** という単一の香気イメージを生む。

### クラスト形成の制御因子
- **オーブン温度**: 高温（230-260°C）= 急速で薄いクラスト + 内部水分保持。低温（200-220°C）= 厚いクラスト + 内部乾燥
- **オーブン時間**: 長いほど厚く乾燥
- **生地表面の処理**: 卵塗布（卵黄たんぱくの Maillard 加速）、水塗布（一時的に a_w 上昇）、粉塗布（粉が表面の Maillard を抑え白いまま残す）
- **生地配合**: 砂糖（カラメル化加速＋色濃く）、ミルク・乳製品（ラクトース＋たんぱく質で Maillard 強化）、油脂（油でコーティング → 水蒸気透過低下 → 内部水分保持）
- **発酵時間**: 長時間冷蔵発酵 [[長時間冷蔵発酵リターディングの科学と風味形成]] で遊離アミノ酸増加 → Maillard 強化 → 香り豊かなクラスト

## 気づき・洞察

- **「クラストとクラムは同じ生地の物理状態の二相」**——クラストは別の食材ではなく、**温度と水分の境界条件が違うだけの同じ生地**。これは [[筋線維タイプとトレーニング適応]] で同じ筋細胞が訓練刺激で Type I/II に分化する現象と同型。**境界条件が分岐を生む**
- **「水分活度はクラスト形成の隠れたマスタースイッチ」**——多くのレシピは温度と時間を強調するが、本質は a_w の時間プロファイル。スチーム導入は a_w を一時的に上げ、後半はそれを乾燥環境に晒す。**目に見えない変数を読めるとレシピが言語化される**
- **「Maillard を最大化する中間層」の普遍性**——表面最外層でも内部でもなく、**両者の境界中間層で反応が最大化**するのは、酵素反応（基質と酵素の中間濃度）、扇形気象（前線の境界）、市場経済（消費者と生産者の中間に流通）など、多くの自然現象で見られるパターン。**反応が起こる場所は「両極の中間」が多い**
- **「香気は数百分子の合奏」**——「焼きたてパンの匂い」は単一分子ではなく **数百の香気分子の比率パターン**。香水の調香と同じ。AI による香気再現（電子鼻 e-nose）がまだ単純な臭気の判別レベルにとどまるのは、パン皮のような **多次元比率パターンの認知**に人間の嗅覚系が極端に最適化されているから
- **「クラストの厚さは保存性の指標」**——薄いクラストは流通向きでない（皮が割れて水分が逃げる）が、保存性も低い。**バゲットは 8 時間で硬くなる**のはクラストが内部水分を急速に失うから。**機能と寿命のトレードオフ**

## 他分野との接続

- **anatomy (脊柱起立筋と多裂筋)**: 表層筋（脊柱起立筋）と深層筋（多裂筋）の機能分化は、クラストとクラムの物理状態分化と同型。**同じ生地（同じ骨格筋系）が境界条件で異なる機能に分化する**
- **tech (Apache Arrow)**: クラストとクラムが「同じ生地の異なる物理状態」であるように、Arrow のバッファも「同じバイト列の異なる解釈」。**物理的不変、論理的多様**
- **music (モーツァルト K.491)**: K.491 の「形式は古典的、内容は浪漫的」二重性は、パンの「外は硬く、中は柔らか」二重性に対応。**外側の制約が内側の表現を保護する**
- **piano (内的聴感)**: パン職人が完成パンの音（叩いた中空音）と色を内的に想像してオーブンを操作するのは、ピアニストが内的聴感で音色を想像してから鍵盤を打つのと同じ。**プロは結果イメージから逆算する**
- **golf (傾斜地ライ)**: クラスト形成のスチーム→乾燥の二相戦略は、傾斜地での「アドレス調整→スイング実行」の二相戦略と同型。**条件付きで段階的に実行する**
