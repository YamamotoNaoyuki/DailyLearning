# ペダリング技法とダンパー共鳴の物理学
**日付**: 2026-05-08
**分野**: piano
**タグ**: #pedaling #damper #una-corda #sostenuto #sympathetic-resonance

## 学んだこと

### ピアノの3本ペダルとそれぞれの物理機構

**右ペダル（ダンパー/サスティン）— 共鳴を解き放つ**:
すべての弦からダンパー（フェルトの消音器）を一斉に持ち上げる。これにより打鍵された弦の振動が空気を介して、または駒（ブリッジ）を介して**他のすべての弦を共鳴させる**。ピアノの音色の豊かさは、ハンマーが叩く3本の弦の振動だけでなく、この**交感共鳴（sympathetic resonance）**による倍音の総和で決まる。

**左ペダル（ウナ・コルダ/ソフト）— 音色を変える**:
グランドピアノでは**鍵盤・ハンマーアクション全体が右に約4mmシフト**する。これにより：
- 3弦区間（中音域以上）では、ハンマーが**3本のうち2本だけ**を叩く（残り1本は同じ音高だが鳴らない＝鳴っているのは2弦）
- ハンマーは**普段使われていない部分**で弦を叩く。グルーヴ（ハンマーのフェルトに長年の打弦で刻まれた溝）の外側部分で打弦するため、フェルトが柔らかく音が「**こもる・覆われた**」音色になる
- 音量低下（約20-30%）と**音色の質的変化（音響スペクトルが低周波寄り）**が同時に起きる

「Una corda」（一本の弦）はモーツァルト時代の名残で、当時のフォルテピアノでは本当に1本だけ叩いた。現代グランドは2本＝due corde が正確だが名称は残った。

**中央ペダル（ソステヌート）— 選択的サスティン**:
押した瞬間に**すでに上がっているダンパーだけ**を保持する。低音をペダルで持続させながら、上声を staccato で弾くといった、**右ペダルでは不可能な選択的持続**を実現する。Steinwayが1874年に特許化した比較的新しい機構。アップライトでは「マフラー」になっていることが多い。

### ハーフ・ペダリング（半踏み）— 連続パラメータとしてのペダル
ダンパーは「上がっているか降りているか」の二値ではなく、**連続的な高さ**を取りうる。
- **ハーフペダル**: ダンパーが弦に**軽く触れる**位置。低音は持続するが減衰し、高音は短く減衰する（高音弦のほうが短く軽いため接触で速く止まる）
- **クォーターペダル**: 触れるか触れないかの境界。倍音の長い余韻だけを残し、基音は早く減衰させる
- **三段階以上の段階的踏み替え**（Verger pedaling, gradient pedaling）

これは**ハーモニーの濁りを最小化しつつ低音の支えを保つ**ための核心技法。Debussy、Ravelの印象派演奏ではほぼ必須。Cziffraはハーフペダルを微細に変化させて「ペダルだけで dynamics を作る」と評された。

### ペダル交換のタイミング — Legato Pedaling

問題: 和声が変わる瞬間にペダルを上げ下げすると音が途切れる
解決: **新しい音を打鍵した直後に**ペダルを上げて踏み直す（**syncopated pedaling**, **legato pedaling**, **rhythmic pedaling**）

物理的には：
1. 古い和声の弦が振動中
2. 新しい音を打鍵 → 新しい弦も振動
3. **打鍵の数十ミリ秒後**にペダルを上げる → 古い弦のダンパーが降り、急速に減衰
4. 即座にペダルを踏み直す → 新しい弦のダンパーが再び上がる

このタイミングは**新しい音のアタック後・古い音の知覚的減衰前**に行う必要があり、人間の知覚閾値（約30-50ms）に頼っている。Schnabel やHorowitzはこのタイミングを「秒以下」の精度で制御していた。

### 交感共鳴とハーモニックの物理学
A4 (440Hz) を弾くと、E5 (660Hz) の弦が共鳴する。なぜなら：
- A4の3次倍音 = 1320Hz
- E5の2次倍音 = 1320Hz
- 共有倍音1320Hzで結合する

**完全5度・完全4度・オクターヴ・3度** の関係にある弦が強く共鳴し、**全音・半音**の関係では弱い（共有倍音が高次で振幅が小さい）。

Steinwayの「**Duplex Scale**」（駒の手前にある短い「補助弦」を打鍵された弦の倍音と同調させる）はこの物理を積極的に利用した発明（1872年特許）。

### ペダルの「ノイズ」と表現
**「ペダル・カチャ」**: ダンパーが弦に降りる時の音。これも音楽の一部であり、Liszt や Debussy はこの音を計算に入れたとされる。**Una cordaのきしみ音**もまた、**過渡的な音色変化**として聴覚体験を作る。

## 気づき・洞察

ペダリングは「音を伸ばす」道具と教えられがちだが、本質は「**ピアノの音響モードを切り替えるパラメトリック制御**」である。
- 右ペダル: 共鳴モード ON/OFF（連続変数化可能）
- 左ペダル: 音色モード（全弦/部分弦、フェルト位置）
- 中央ペダル: 選択的持続モード

これら3つの**直交する制御次元**を同時に操ることで、ピアノは単なる打鍵楽器を超えて**3次元音色制御楽器**になる。

「**ハーフペダル**」の概念は、ペダルが「**離散的なフット・スイッチ**」ではなく「**連続コントローラー**」であるという認識転換を要求する。これは**初心者にとって最大の壁**で、視覚フィードバックがないため習得には数年かかる。

「Possente spirto」が装飾を二版で記譜したように、ベートーヴェン以降の作曲家はペダル指示を楽譜に書き込むようになった。だがそれは**作曲家の意図の固定化**であって、現代のピアノ（弦の長さ、フェルトの硬さ、音域バランスが当時と違う）で**そのまま再現すべきか**は議論が続く。Rosen は「ベートーヴェンのペダル指示は当時の楽器を前提にしているので、現代では原則として無視か再解釈すべき」と主張した。

## 他分野との接続

- **music (Monteverdi の Possente spirto 装飾記譜)**: 楽譜に細部まで指示を書く流れの起点。Monteverdiの装飾→Beethovenのペダル→Debussyの細密ニュアンス。**「演奏者の自由」と「作曲家の意図」のせめぎ合い**
- **anatomy (横隔膜の連続制御)**: ハーフペダルが「離散→連続」の認識転換を要求するのと同様、**呼吸も**「吸う/吐くの二値」ではなく「胸郭・腹部・骨盤底の連続的協働」として捉え直すと演奏が変わる
- **golf (シャフトフィッティングの連続パラメータ)**: ペダルの踏み込み深さは、シャフトのフレックスやスイングウェイトのように**連続パラメータ**で、最適値は奏者・楽器・音楽の組合せ依存
- **tech (eBPFの hook point 選択)**: tracepoint/kprobe/uprobeのトレードオフは、damper/sostenuto/una corda の使い分けと類似。**「目的に応じた最小コストの介入点」を選ぶ**思考
