# タッチと音色の物理学 — ピアノは「タッチで音色を変えられる」のか
**日付**: 2026-05-03
**分野**: piano
**タグ**: #タッチ #音色 #ハンマー速度 #ハンマーフェルト #Askenfelt #Goebl #Ortmann #Neuhaus

## 学んだこと

### 物理的事実: ハンマー離脱後、終端速度のみが音を決める
ハンマーは let-off (escapement) で jack から外れた後、**完全に慣性のみで弦に到達する**。Hart, Fuller, Lusby, Bryan-Boyle ら 20 世紀の研究者は、ハンマー終端速度 v_h が同一なら、それ以前の加速プロファイル (指の慣性、Hammerschank の回転履歴) は弦振動の初期条件に影響しないことを示した。Cristofori が 1720 年に escapement を発明した時点で、この「機械的因果切断」は構造的に組み込まれた。

物理パラメータ: ハンマー質量 8–13 g、ストローク 47 mm、ff で 4–6 m/s。**ハンマー離脱後の人間の力 (after-touch) は弦に伝わらない**。これが「タッチで音色を変えられない」物理学派の論拠。

### しかし演奏家は変えていると感じる — Askenfelt & Jansson の解明
KTH (ストックホルム王立工科大学) の Anders Askenfelt と Erik Jansson "From Touch to String Vibrations" 三部作 (JASA 1990, 1991, 1993) は、光学位置測定で鍵盤・ハンマー・弦の運動を mm 精度で計測。結論: **終端速度が同じなら弦振動はほぼ同じ**だが、演奏家のタッチは次のものを「同時に」変えている:

- chord balance / voicing (同時音の比率)
- legato/staccato の時間構造 (agogics)
- ペダルによる sympathetic resonance
- そして決定的に **key noise** — finger-key 衝突音、key-bottoming 音、Hammerschank 振動

これらすべてが「音色の差」として知覚される。**物理は正しく、演奏家の感覚も正しい。両者は別の事象を語っている**。

### Werner Goebl の knockout 実験
Vienna mdw の Goebl は Bösendorfer 290 SE Reproducing Piano (MIDI 入出力可能な計測グランド) で、**同一終端速度** での "pressed touch" (指が鍵に乗った状態から押し下げる) と "struck touch" (上から叩く) を比較。Goebl, Bresin, Fujinaga (JASA 2014) の knockout 実験では、被験ピアニストの約半数が両者を聴き分けたが、**finger-key 衝突音 (打鍵 20–200 ms 前) をデジタル除去すると識別率はチャンスレベルに落ちた**。Key-bottom 音の有無は 82% の識別率。

つまり「タッチの音色差」の正体の大半は、**楽音そのものではなく noise precursor** だった。Struck touch では finger-key 加速度のピークが 150 m/s² (15G) 超に達し、これが鍵盤・木部に高周波振動を伝える。

### ハンマーフェルトの非線形ばね特性
フェルトは hardening spring: F = K·δ^p、指数 p ≈ 2 (低音) から ~4 (高音)。Hall & Askenfelt の動的測定では p ≈ 2.5–3.5。**演奏家が好む範囲は 2 < p < 3** (Russell)。p = 1 なら ff は単に大きい pp、p > 3 なら ff は耳障りに尖り pp は退屈になる。

**強打で高次倍音が増える** (brilliant) のは、弦の線形性ではなく**フェルトの瞬間硬化と接触時間短縮による高調波励起**。Voicing (ニードリング) で表面繊維を緩めれば dolce、アイロンや薬剤で硬めれば brilliant。Steinway は明るめ、Bösendorfer は暖色、Yamaha CFX は均整 — フェルト密度勾配の哲学差。

### Ortmann 1925 — 科学的タッチ研究の出発点
Otto Ortmann (Peabody Conservatory) "The Physical Basis of Piano Touch and Tone" (1925) と "The Physiological Mechanics of Piano Technique" (1929) は、Tobias Matthay の主観的「重量奏法」議論への科学的反証。「**同じハンマー速度なら同じ音**」を最初に実験的に示し、現代研究の系譜の祖となった。

### Penetration vs surface — ロシア派「鍵盤の底まで」の意味
Heinrich Neuhaus "The Art of Piano Playing" (1958, モスクワ音楽院) の「鍵盤を掴む」奏法は、物理的には after-touch は弦に作用しない (ハンマーは既に離脱)。しかし機能は実在する:

- **次音への準備姿勢** (forward model の精度)
- **レガートの体性感覚的フィードバック**
- **key-bed への力伝達による触覚連続性**
- **聴覚イメージとの内部モデル整合**

Gilels, Richter, Lupu を育てた言説は、物理ではなく **sensorimotor control の言語**として読むべきだ。

### Half-pedaling と sympathetic resonance
ダンパーペダルは on/off ではなく**連続変数** (full / 3/4 / half / 1/4)。Half では高音側ダンパーが弦に軽く触れた状態で、高次倍音優先で減衰、低音域は鳴り続ける。これにより全 88 鍵 200+ 本の弦が partial に共鳴し、コンサートホール残響と相互作用する複雑な音色パレットが生まれる。**Ravel・Debussy のテクスチャは half-pedal なしには成立しない**。

### デジタルピアノと Pianoteq
サンプル方式は MIDI velocity → 録音音への一対一マッピングのみ。タッチ差 (finger-key noise, half-pedal の連続性, una corda の弦シフト) は表現不能。

**Pianoteq (Modartt)** は物理モデリング (~50 MB、サンプルなし) で、フェルト非線形性 p、弦剛性、ダンパー位置、duplex scale 共鳴を計算。それでも実機の key-bed 体性感覚は再現不可 — **タッチ感覚の半分は耳ではなく指にある**。

## 気づき・洞察

**1. 「音色を変える」言語は嘘ではなく、複合変数のショートカット** — 演奏家は voicing/agogics/pedal/noise を統合的に「タッチ」と呼ぶ。物理学者の還元は技術的に正しく、実用的に貧しい。**両者の対話の難しさは、語彙の抽象化レベルが違うこと**にある。

**2. noise は信号 — finger-key 音は音色の構成要素** — 録音エンジニアが嫌う打鍵ノイズを、ホールの聴衆は「**ピアニスティックな質感**」として無意識処理している。ノイズと音楽の境界は、文脈で動く。

**3. Pressed touch の練習意義** — 物理的に楽音に効かなくても、finger-key noise を消し、鍵盤への力伝達経路を最短化する。**Cantabile の物理的基盤**は noise minimization にある。

**4. 強打で brilliant なのはフェルトのおかげ** — pianissimo を dolce に、fortissimo を brilliant にする全ピアノ表現は、p ≈ 2.5–3.5 の felt nonlinearity に依存。**線形なら表現の幅は半減**する。楽器の表現力は非線形性に住んでいる。

**5. after-touch は次音への bridge** — Neuhaus の「底まで」は物理ではなく**時間の連続化**。レガートは音と音の隙間を体性感覚で繋ぐ。

**6. Half-pedal はデジタルピアノの最大の弱点** — 二値変数では表せない芸術的次元。

**7. Voicing は最強のタッチ調整** — フェルト硬度の調律師による調整は、奏者の任意の練習量より大きく音色を変える。**機械の状態が奏者の音色の天井を決める**。

**8. 練習の本質は内部モデルの校正** — 終端速度を脳が予測制御するために、key-bed の体性感覚と聴覚を統合する forward model を磨く作業。これが M1 と S1 の neuroplasticity (前回学習) に対応。

## 他分野との接続

- **解剖学/M1 運動野 (前回学習)**: 一流ピアニストは指先の force sense と two-point discrimination が非奏者より有意に高い (J Neurophysiol 2020)。M1/S1 拡大、proprioceptive acuity 向上 — 「**タッチ」は皮質地図の精緻化として実装**されている。
- **股関節の球関節 (本日 anatomy)**: 椅子高 → 股関節 90° → 骨盤前傾 → lumbar lordosis 維持 → 上肢自由度確保 → タッチ品質。**下半身 alignment が上肢タッチの天井**を決める同型構造。
- **音響/コンサートホール (既習)**: half-pedal の sympathetic resonance はホール残響と相互作用する。乾いたスタジオでは無効、響きのある場では音色の主成分。
- **認知 (メンタルプラクティス、既習)**: 聴覚予測 → forward model → 運動指令という Schmidt の schema theory 的階層。Murray Perahia, András Schiff の「**音を聴いてから打鍵する**」は、聴覚予測が運動を駆動する逆方向制御の言語化。
- **DuckDB の interpreted vectorized (本日 tech)**: 「物理 (機械) と人間の知覚の橋渡し」は、データベースの「ハードウェア物理 → 抽象計算」と同じ問題構造。両者とも**抽象化の不完全さを認めて道具にする**。
