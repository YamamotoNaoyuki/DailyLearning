# バッハ・フーガの演奏技法：多声部の独立性とウォーキングベース
**日付**: 2026-05-15  
**分野**: piano  
**タグ**: #バッハ #フーガ #多声部 #ボイシング #平均律 #トゥレック #シフ

## 学んだこと

### フーガ演奏が「ピアノで最も難しい」とされる理由

平均律クラヴィーア曲集に含まれる48曲のフーガは、ほとんどが3声または4声、Book I の嬰ハ短調と変ロ短調は5声、Book I の ホ短調は2声と、声部数で構造の密度が変わる。ピアノは本来「打鍵すれば音が出る単音楽器の集合」だが、フーガでは10本の指で**それぞれ独立した方向感・タイミング感・音色を持つ複数の声部**を同時に走らせなければならない。これは単純な技術問題ではなく、**多重注意（multi-stream attention）** という認知問題でもある。

### 4つの独立次元での同時制御（トゥレックの分析）

ロザリン・トゥレックはバッハ演奏の基礎を体系化した。彼女のフーガ演奏では、各声部に対して以下の4次元を**同時に異なる値で**コントロールする：

1. **アーティキュレーション**: legato / non-legato / staccato / 二音スラー
2. **ダイナミクス**: 主題提示中の声部は f、対主題は mp、伴奏的な声部は p
3. **タッチ（音色）**: 指の腹で押し込む vs 指先で掴む、鍵盤に深く沈む vs 表面を撫でる
4. **位相（タイミング）**: 各声部は厳密に同じ拍ではなく、わずかにずらすことで耳が分離できる（mensural displacement）

つまり**4本の指がそれぞれ別の動き方をする** ことを訓練する。これはアンサンブル演奏で複数の弦楽器が独立して演奏するのと同じ状態を、片手や両手の中で再現する技法だ。

### 「Phrasing into the beat, not on the beat」

トゥレックの最重要原則。古典派以降の音楽は拍の上にアクセントが乗るが、バッハの場合は**フレーズが拍に向かって流れ込む**。例：4分音符の連続でも、拍頭ではなく拍頭の手前の音にフレーズの重みを置く。これにより、バッハ特有の和声的緊張と解放のリズムが生まれる。現代の拍中心の演奏感覚を一度解体する必要がある。

### シフのカンタービレ原則とクルターグの影響

シフは恩師のクルターグから「Bach を打楽器的にではなく、常にカンタービレで弾け（spielen sie ihn cantabile, nicht perkussiv）」と教わった。これはバッハの各声部を「人間の声」として扱い、フレーズの呼吸を歌うように作るという原則。フーガでは：

- 各声部の主題提示時には**わずかなテヌート**で「ここから歌い始める」を示す
- 主題が他声部に渡るときには**音量・音色を交代**で受け渡す（リレー）
- 一つの主題が複数声部に重なる**ストレット（stretto）**では、後から入る主題をやや強調して「層が増えた」ことを示す

### ペダリングの逆説

「バッハはチェンバロのためのもので、ピアノでペダルを使うべきでない」は半分正しく半分間違っている。シフ・トゥレックの実践では：

- **メインペダルは原則使わない**：声部の独立性が失われ、和声的混濁が起きる
- ただし**短い"flutter pedal"（フラッターペダル）** や **"finger legato"の補助としての半ペダル** は使う
- 倍音による共鳴を狙う場合（嬰ハ短調フーガ Book I の長大なクライマックスなど）にはペダルで響きを増やす

要は**「ペダルなしを基本にして、ペダルの介入は意図的・例外的に」**。これは無造作なペダリングへの戒めであって、ペダル禁忌ではない。

### キーリリースのコントロール（チェンバロからの転換）

バッハの時代、鍵盤楽器はチェンバロ・クラヴィコード・オルガンであり、いずれもサスティンペダルがなかった。音の終わり方は**キーをいつ離すか**で決まった。現代ピアニストはこれを取り入れる：

- 各音の **release timing（鍵離し）** を意識的にコントロール
- 4声フーガで、ある声部の音を「持続させる」「途中で切る」を意図的に分ける
- 例：バス声部が長い4分音符、テナーが8分音符の動きの場合、テナーをやや切って分離し、バスは指で押さえ続ける（finger legato）

### 物理的訓練：声部の分離

実技的にフーガを習得するには：

1. **声部別に練習**：各声部だけを片手で弾く（4声なら4回）
2. **2声ペアで練習**：S+A, S+T, S+B, A+T, A+B, T+B の全6組み合わせ
3. **主題のみ追跡**：主題が出てくる声部だけを連続的に弾き、他は無視する
4. **片手2声練習**：右手で S+A、左手で T+B など、各手2声を独立に弾く（最も難しい）

## 気づき・洞察

**「指の独立性は脳の独立性」**。物理的に指を独立に動かす訓練は、運動野での独立した運動指令を作る訓練でもある。指は解剖学的に屈筋腱で連結しているため完全な独立は不可能だが、**運動学習を重ねると皮質脊髄路の指令分離が進む**（[[一次運動野M1と運動制御階層]]）。フーガ演奏者の脳画像研究では、各指の運動野表現が非演奏者より明確に分離している。

**フーガは「時間的注意のマルチプレキシング」**。人間の意識的注意は本質的にシングルスレッドだが、フーガ演奏者は**「主題が今どの声部にあるか」**を高速に切り替えながら追跡する。これは認知科学で言う"task switching" の極限訓練。多くのピアニストが平均律を生涯にわたって練習するのは、この能力が衰えやすいから。

**ストレットの哲学的意味**。主題が次々と重なって入るストレットは、「同じ思考が異なる時間軸で並行進行する」状態。これは並列分散システムにおけるパイプライン処理、あるいは複数の自己が同時に存在する仏教的瞑想状態にも通じる。バッハはこれを音楽で実装した。

## 他分野との接続

- **music / シベリウス第7**: 第7番が「主題の有機的変容」で時間を構造化するのに対し、フーガは「主題の重複・反転・拡大・縮小」で時間を構造化する。両者とも**単一胚種からの多声的展開**という共通発想
- **anatomy / 運動制御**: [[一次運動野M1と運動制御階層]] と [[小脳と運動学習タイミング制御]] が、独立した10本指の同時制御を可能にする神経基盤
- **tech / マルチエージェント**: 各声部を独立したエージェントと見れば、フーガは**4エージェント協調プロトコル**のメタファ。主題（メッセージ）が声部間で受け渡されるのは MCP/A2A の構造に似る
- **cs / 並列計算**: 4声部の同時独立進行は、4スレッドの並列処理。ストレットは pipeline parallelism、対位法は thread synchronization
- **deutsch / 言語**: 各声部の独立した「意図」を保つのは、複数の話者が同じ会話で各自の論点を持ち続けるのと同じ。Modalpartikeln が会話の各層に異なるニュアンスを付与するのと類似
