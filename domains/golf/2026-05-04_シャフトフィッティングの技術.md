# シャフトフィッティングの技術

**日付**: 2026-05-04
**分野**: ゴルフ
**タグ**: #シャフト #フィッティング #CPM #トルク #キックポイント #EI #TrackMan #Wishon

## 学んだこと

### 1. フレックスと振動数 CPM
シャフトの硬さは「振動数 (Frequency, CPM = Cycles Per Minute)」で定量化される。クランプした片持ち梁の固有振動数で、L→A→R→S→X の順に高くなる。**TrueTemper 社のスチールシャフト基準では概ね R=240–260 CPM、S=255–275 CPM、X=270–290 CPM**。ただし**業界標準は存在せず**、同じ R 表記でもメーカー間で 10–20 CPM 差がある（True Temper 社内の R フレックスドライバーで 241–281 CPM の幅）。アイアンセットでは「**0.5 インチ短くなるごとに 4 CPM 上昇**」が frequency-matched セットの設計則。

### 2. トルク（捻れ剛性）
シャフトを長軸周りに 1 ft·lb のトルクで捻ったときの捻れ角（度）。**低トルク 2–3°（X 系・Tour 用）vs 高トルク 5–6°（A/R 系・シニア用）**。低トルクはダウンスイングでフェースの開閉誤差を抑え、特に高ヘッドスピード（100 mph 超）プレーヤーでフックフック化を防ぐ。高トルクは「球をつかまえやすく」感じるが、強い切返しではフェースが返り過ぎてフックや引っ掛けの原因になる。

### 3. キックポイント（ベンドポイント）
シャフトが最も大きく曲がる位置。**低キック→高弾道・高スピン**（先端が走る、つかまり良）、**高キック→低弾道・低スピン**（先端硬い、操作性高）、中キックはバランス型。現代の高速カメラ計測では「キックポイント」は瞬時の動的概念に過ぎず、**EI プロファイル（曲げ剛性分布）**で記述する方が正確と分かってきた。

### 4. EI 曲線とベンドプロファイル
EI = Young 率 E × 断面 2 次モーメント I。3 点曲げ試験機でシャフトを Butt（手元）/Mid（中間）/Tip（先端）の 3 区間で計測する。**Tip Stiff（先硬）はレイトリリース・強切返しの選手に**フェース安定をもたらし、**Tip Soft（先軟）は早いリリースに**走り感を与える。Mid Stiff は感触の一貫性、Butt Stiff はグリップ下の安定。Fit2Score 等のソフトウェアは EI を 5 点で取得し「Bend Profile Signature」として可視化する。

### 5. 重量とスイングウェイト
**ドライバートータル重量 290–320 g** が標準。シャフト単体は軽量 50 g 台（高反発系・軽量化）から重量 80 g 台（ツアー系）。**スイングウェイト D0–D5 が男性ドライバーの主流（D2 が中心）**。Lorythmic スケールでは**シャフト 6.5 g 増（バランスポイント 7.75 inch 想定）= スイングウェイト 1 ポイント増**。総重量は同じでも、ヘッド・シャフト・グリップへの質量配分でスイングウェイトは大きく変わる（重量配分こそが本質）。

### 6. シャフト長と MOI
**45 → 47 inch でヘッドスピード +3〜5 mph**（理論値は +1 inch あたり +2 mph）が得られるが、ヘッド軽量化と MOI 低下を伴う。USGA 規制は 48 inch（PGA Tour 競技 2022 〜は 46 inch）。長尺はインパクト点ばらつきが急増し、**center contact 失敗による ball speed loss が swing speed gain を上回る**ケースが多い。Bryson DeChambeau の 47.5 inch 採用例は例外。

### 7. 動的フィッティングと Trackman
TrackMan 4 / GC Quad で Attack Angle、Spin Loft、Club Path、Dynamic Loft、Face to Path を同時計測。シャフト判定の典型手順:
- **Spin Loft が高すぎ（17°超）→ Tip Soft でリリース早すぎ → Tip Stiff へ**
- **Launch 低 + Spin 高 → ギア効果ヒール接触 → Tip 硬めで Face 安定**
- **Side Spin 増 → Torque 過大 or Flex 不足**
- **Smash Factor < 1.45 → Center Contact 不安定 → 重量 or 長さ見直し**
ドライバー最適は**Launch 13–17°、Spin 2200–2700 rpm、Attack Angle +3〜+5°**（Tour 平均）。

### 8. Tom Wishon 4 要素フィッティング
(1) **Clubhead Speed**（ドライバーと 6I 両方を計測、片方だけでは誤判定）、(2) **Transition / Tempo**（Smooth / Average / Forceful の 3 段階観察）、(3) **Shaft Weight**（強くアグレッシブな選手ほど重く）、(4) **Bend Profile / Release Point**（レイトリリース → Tip Firm、アーリーリリース → Tip Soft）。Wishon の主張は「ヘッドスピードだけのフィッティングは guessing」。

### 9. スチールシャフト比較
**Dynamic Gold S300（130 g, 5.8 FCM）= 低弾道・低スピン基準**、**Project X 6.0（120 g, 6.5 FCM）= Tip Stiff、強い切返し向け、中弾道**、**KBS Tour（110–125 g）= Mid Trajectory、滑らかなロード感、Butt が DG より硬い**、**Nippon Modus3 Tour 130 = DG とほぼ同等の弾道、Tour 120 は KBS 系に近い高弾道**。グラファイトはスチール比で**振動減衰が高く（疲労軽減）、軽量化（40–80 g 域）が可能**だが製造分散が大きい。

### 10. ドライバー主要グラファイトモデル
- **Fujikura Ventus** (VeloCore Technology の bias core で先端剛性確保): **Black = 低弾道・低スピン・剛性最高**（Tour で最採用率）、**Blue = 中弾道**、**Red = 高弾道・つかまり**。
- **Mitsubishi Diamana**: **GT = 中弾道・滑らかな走り**、**WB = 中低弾道・球持ち長め**、**PD = 低スピン**、**TB = TourAD 競合ライン**。
- **Graphite Design Tour AD**: **DI = 中元調子・低スピン定番**、**IZ = 中調子・万能**、**XC = 走り感**、**VF = 手元剛性最強・Tip も硬く左を嫌う上級者向け**、**CQ = 中元・球上がり良**。Tour AD は手元剛性が一般に高く、強い切返しの上級者向け傾向。

## 気づき・洞察

- **CPM・トルク・キックポイントは互いに独立しない**。EI プロファイルが唯一の真の記述で、3 つは EI の異なる射影。**「単一スカラーで近似できない多次元空間」**を 3 つの軸で粗視化したのが伝統的フィッティングだった。
- **TrackMan 登場以前の shaft fitting は神話だった**。0.0005 秒のインパクトでヒトの主観 feel は当てにならず、launch monitor が EI 値と球の挙動を客観連結したことで初めて科学化した。D-Plane 理論と同じ「測定機器駆動の科学化」パラダイム。
- **Wishon の Transition 観察は機械学習の特徴量設計**に近い。Tempo と Release Point は launch monitor で直接見えないが、ヘッドスピード・スピンロフトから推定可能。今後 AI フィッティングが台頭する余地。
- **Smash Factor 1.50 の天井は物理（COR 0.83 規制）が決める**が、シャフト適合がそれを実現する条件。

## 他分野との接続

- **解剖学**: 強切返しは前腕回内・手首尺屈の急加速で、橈側手根伸筋腱の伸張速度が EI Tip 区間と共振。**人体の固有振動数（手首 5–10 Hz）とシャフト CPM（4–5 Hz）のカップリング**が「合う/合わない」の感覚の正体。
- **ピアノ**: ハンマーアクションの「キックポイント」（重心位置）の議論と同型。質量分布が打鍵の加速プロファイルを決める点で構造同型。
- **CS**: シャフトの EI プロファイル = 関数の signature、フィッティング = type inference。Wishon の 4 要素は型変数 4 つの type system。
- **テクノロジー**: launch monitor は「観測 → モデル更新」の Bayesian inference そのもの。fitting session は posterior 収束過程。
