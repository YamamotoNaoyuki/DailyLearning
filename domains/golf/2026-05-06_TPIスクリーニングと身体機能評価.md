# TPIスクリーニングと身体機能評価
**日付**: 2026-05-06  
**分野**: golf  
**タグ**: #TPI #BodySwingConnection #PhysicalScreening #SwingFault #GregRose #DavePhillips #Titleist

## 学んだこと

### 起源とBody-Swing Connectionという哲学
**Titleist Performance Institute (TPI)** は、2003年カリフォルニア州オーシャンサイドにて、カイロプラクター/フィジカルトレーナーの **Greg Rose, DC, CSCS** とPGAティーチングプロ **Dave Phillips** によって設立された。Acushnet (Titleist親会社) がスポンサーとなり、ゴルフ史上初の「身体機能とスイングを統合的に評価する研究施設」として始動。当初の研究はLPGA/PGA選手の3Dモーションキャプチャと医学検査の同時取得に注力した。

設立哲学は **"Body-Swing Connection"**——スイングの欠陥はスイング技術の問題ではなく、しばしば身体機能の制限が引き起こす「代償運動 (compensation)」であるという見方。Roseの論文 "There is no one way to swing a club, but there is one efficient way for every player to swing it given their physical limitations" がTPIの命題的背骨である。これはホーガンの「全員に当てはまる唯一の正解」モデルとは正反対の立場である。

### 16-Point Physical Screening Protocol
TPIスクリーニングは16の単純なテストで構成される。各テストは特定の関節可動域・筋機能・神経筋制御を評価し、スイング特性 (swing characteristics) と臨床的に対応づけられている。

| テスト | 評価対象 | 関連スイング欠陥 |
|--------|----------|-----------------|
| Pelvic Tilt Test | 骨盤前後傾の独立制御 (運動分離) | S/C-Posture, Loss of Posture |
| Pelvic Rotation Test | 骨盤の独立回旋 (上半身固定下) | Sway, Slide, Early Extension |
| Torso Rotation Test | 胸郭の独立回旋 (骨盤固定下) | Reverse Spine, Flat Shoulder Plane |
| Overhead Deep Squat | グローバル可動性・スタビリティ | Loss of Posture, Early Extension |
| Toe Touch | 後面チェーン (ハム/腰椎/脊柱) 統合 | C-Posture, Early Extension |
| 90/90 (Cervical Rotation) | 頸椎回旋 (≥70°) | Loss of Posture, Chicken Wing |
| Seated Trunk Rotation | 胸椎回旋 (≥45°/側) | Flat Shoulder Plane, Reverse Spine |
| Bridge with Leg Extension | 殿筋 (Glute) 単独活性 | Sway, Slide, Early Extension |
| Single Leg Balance (eyes open/closed) | 固有受容覚 (前庭・足底) | Sway, Loss of Posture |
| Lat Length Test | 広背筋伸張性 (両腕万歳壁付) | S-Posture, Loss of Posture |
| Lower Quarter Rotation | 股関節内外旋 (≥60°) | Sway, Slide, Hanging Back |
| Forearm Rotation | 前腕回内回外 | Casting, Chicken Wing |
| Wrist Hinge Test | 手関節橈屈/尺屈 | Casting, Early Release |
| Reach, Roll & Lift | 胸椎伸展+肩屈曲 | Flat Shoulder Plane, S-Posture |

各テストは「合格/不合格」の二値ではなく、**Pass / Limited / Fail** の三段階で記録される。Greg Rose曰く「Failは介入の最優先、Limitedは伸びしろ、Passは現状維持」。

### 12 Common Swing Characteristics
TPIは経験的にツアー〜アマチュアに頻出する12のスイング欠陥を分類した。

| Characteristic | 観察される現象 | 主たる身体的原因 |
|----------------|----------------|------------------|
| S-Posture | 過度な腰椎前弯 (アドレス時の反り) | 広背筋短縮、腹横筋弱化 |
| C-Posture | 胸椎後弯増大 (猫背) | 大胸筋短縮、菱形筋弱化 |
| Loss of Posture | バックスイング中の脊柱角喪失 | 胸椎回旋制限、股関節モビリティ不足 |
| Flat Shoulder Plane | バックスイングで肩平面が水平化 | 胸椎回旋・側屈制限 |
| Early Extension | ダウンスイングで骨盤がボール側へ | 股関節屈曲位での回旋不足、Glute活性不足 |
| Over-the-Top | クラブが steep に外側から | 胸椎回旋制限による腕主導の代償 |
| Sway | バックスイングで右下肢へ重心移動超過 | 右股関節内旋制限 |
| Slide | ダウンスイングで左下肢へ過剰スライド | 左股関節屈曲・内旋制限 |
| Reverse Spine Angle | トップで脊柱が左へ反る | 腹斜筋・腹横筋弱化 |
| Hanging Back | フィニッシュで体重が右に残存 | 左下肢安定性不足 |
| Casting / Early Release | 早期リリース | 前腕回外/橈屈制限、手関節制御不足 |
| Chicken Winging | フォロースルーで左肘が引ける | 胸椎伸展不足、肩内旋優位 |

### 臨床ロジック: 制限が代償を生む
TPIの真髄は **「身体制限 → 代償スイング」** という因果連鎖の特定にある。例えば **Toe Touch fail** の選手はハム短縮・腰椎可動性低下を示唆し、アドレスで前傾を維持できず、結果として **C-Posture → Early Extension** に至る。**Pelvic Rotation fail** は骨盤の運動分離不能を意味し、**Sway/Slide** へ直結する。**Lower Quarter Rotation** で右股関節内旋45°以下なら **Sway** がほぼ必発。

これらは100%の因果ではなく、**相関と臨床仮説**である。TPIはこの推論連鎖を明示化し、トレーナーがスイングコーチと共通言語で対話できる基盤を作った——これが業界変革点。

### 認定システムとTour採用
TPIは **Level 1 (Fundamentals, 2日間)、Level 2 (専門深化、Medical/Fitness/Junior/Power/Golf)、Level 3 (応用)** の階層認定を提供。Level 2はカイロプラクター・PT・ATC等の医療従事者向けMedicalトラックと、CSCS等のフィットネス向けトラックが分かれており、相互参照を促す設計。世界で **20,000人以上**が認定保持。

ツアー指導者の代表格として **Joey Hackett** (Justin Thomas, Brooks Koepka 担当)、**Jason Glass** (Lydia Ko 等)、**Dr. Troy Van Biezen** (Adam Scott, Jordan Spieth 担当) が知られ、TPIスクリーニングをツアー帯同で日常的に運用している。

### 限界と批判
学術的視点での指摘も無視できない。
- **相関≠因果**: Toe Touch failと Early Extensionの相関は観察されるが、改善が必ずしもfaultを治す保証はない (Gulgin et al., *Int J Sports Phys Ther* 2014 等)。
- **ATP研究の選択バイアス**: TPI公開データの多くはツアー選手対象で、一般アマへの外挿には限界がある。
- **簡易スクリーニングの限界**: 16テストはGoldStandard (3D Mocap, 等速性筋力計) の代替であって等価ではない。FMS (Functional Movement Screen) との重複も指摘される。
- **査読論文の不足**: 商業認定中心で、システム全体のRCT検証は少ない。Smith et al. (*Sports Medicine* 2018) は「TPI screening の信頼性は中等度、妥当性は要追加研究」と評している。

### アマチュアへの実装優先順位
一般プレイヤーが自宅で自己評価する場合、Greg Roseが推奨する順序は:
1. **Overhead Deep Squat** (グローバル評価、最も情報量が多い)
2. **Toe Touch** (後面チェーン、改善容易)
3. **Pelvic Rotation** (骨盤独立性、Swayの予測子)
4. **Seated Trunk Rotation** (胸椎、X-Factor確保の鍵)

この4つが Pass であれば多くのswing faultが構造的には起こりにくい。Failがある場合、レッスンで技術修正を試みる前に身体側の介入を優先する——これがTPIの実践的ヒューリスティック。

## 気づき・洞察

TPIの本質は「ゴルフスイングを身体の制約条件下での最適化問題と捉える」点にある。ホーガンの "One efficient way" 哲学が普遍解を求めたのに対し、TPIは **個体ごとの制約集合 (constraint set) に応じた局所最適** を求める。これは数理最適化の言葉で言えば、非凸目的関数を制約条件の異なる選手ごとに解いている状態。

もう一つの本質は **「言語の共有化」**。スイングコーチ/フィジカルトレーナー/医師は従来別々の言語で話していたが、TPIの12 Characteristics + 16 Screensは共通プロトコルとして機能する。これはソフトウェア工学でいう **インターフェイス分離** に近く、各専門家が自領域に集中しつつ、整合性のある介入計画を組める。

ただし、TPIをドグマ化する危険もある——「スクリーニングがすべて」の発想は実証的支持を超える。**Pass しているのに Early Extension する選手** や、**Fail しても綺麗に振れる選手** は確実に存在する。臨床ロジックは仮説であって、観察された個別事例を上書きしてはならない。

## 他分野との接続

- **anatomy (本日: 迷走神経HRV)**: TPI screeningはHRVと同様「身体の自律的能力を非侵襲・低コストで定量する」道具。HRVが副交感神経機能の代理変数であるように、Toe Touchは後面チェーン統合の代理変数。代理変数の妥当性は常に検証対象。
- **piano (本日: スロー・プラクティス)**: スロー・プラクティスが「無意識スキルを意識化して再構築する」のと同様、TPI後の身体修正は **既存の代償運動を意識的に分解→再構築** する過程。両者とも「自動化を一度壊す」勇気が必要。
- **cs (本日: HLC)**: HLC (Hybrid Logical Clock) が物理時計と論理時計を統合するように、TPIは技術的観察 (スイング3D) と医学的観察 (関節可動域) を共通フレームに統合。**異種データの整合的時刻付け** の発想が同型。
- **tech (本日: HTMX)**: HTMXがフロント/バックの境界を曖昧化するのと同様、TPIはコーチ/トレーナー/医師の境界を曖昧化する **horizontal integration** のフレームワーク。
- **music (本日: Arvo Pärt)**: Pärtのtintinnabuli技法が「最小要素の制約下で最大表現」を求めるように、TPIは「身体制約下で最効率スイング」を探す。**制約こそが個性を生む** という美学的同型性。
- **過去のbreathing/posture関連エントリ**: 解剖学分野での胸椎モビリティ・股関節モビリティの議論はTPIで言うLevel 2 Fitnessの基礎。「身体側の介入なしにスイング改善は天井に当たる」という結論は両分野で一致。

## 次に深掘りしたいこと

- **TPI Power トラック**: 地面反力 (Ground Reaction Force) とスイング速度の関係。フォースプレート計測と Vertical Ground Reaction Pattern の3類型 (Brian Sullivan の研究)。
- **K-Vest / 4D Motion との統合**: TPIの定性スクリーニングと3D慣性計測 (IMU) による定量データを組み合わせる現代ワークフロー。
- **TPI Junior**: 成長期のwindow of opportunityと長期アスリート発達 (LTAD) モデル。
- **批判的検証論文**: Gulgin (2014), Smith (2018), Lephart研究室のTPI関連論文を一次資料で確認。
- **AMM3D vs TPI**: ツアーで併用されている3Dモーションキャプチャの数値データとTPI screeningの相関係数。
