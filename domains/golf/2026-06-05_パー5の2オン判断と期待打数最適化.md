## 学んだこと

### 1. 期待打数関数 E[S] という土台
Broadie の Strokes Gained は、PGA Tour ShotLink の数百万ショットから「状態（残り距離 d, ライ ℓ）→ ホールアウトまでの平均打数」を回帰した **ベースライン関数** E[S | d, ℓ] を持つ。フェアウェイからの代表値（Tour 基準, ホールアウトまで）：

| 残り距離 | E[S]（FW） |
|---|---|
| 100 yd | ≈ 2.80 |
| 140 yd | ≈ 2.91 |
| 150 yd | ≈ 2.98 |
| 200 yd | ≈ 3.19 |
| グリーン上 20 ft | ≈ 1.87 |
| グリーン上 8 ft | ≈ 1.50 |

go/no-goもレイアップ距離選択も、**すべて「次状態の E[S] を最小化する行動を選ぶ」一つの最適化問題**に帰着する。意思決定は確定的な「打数」ではなく、行動 a を取ったときに到達する状態の確率分布 p(s'|a) 上の期待値 E[S|a] = Σ p(s'|a)·E[S|s'] を比べる、という決定理論の構造を持つ。

### 2. go/no-go の break-even 確率（閉形式）
グリーンに届かせに行く（go）か刻む（lay up）かを、3分岐の期待値で書く。go の結果を「成功＝グリーン周り（確率 p）／失敗＝トラブル（確率 1−p）」に縮約すると：

```
E[S | go] = p · E_success + (1−p) · E_fail
E[S | layup] = E_layup            （ほぼ確定的なので分散小）
```

go を選ぶべき条件 E[S|go] < E[S|layup] を p について解くと：

```
p* = (E_fail − E_layup) / (E_fail − E_success)
```

**数値ケース（230yd 池越えの2オン狙い, アマ上級〜中級）**
- 成功（グリーンエッジ, 残り 15〜30 ft）→ E_success ≈ **2.6**
- 失敗（池, 1罰打→ドロップ 80yd 残り）→ E_fail ≈ 1 + E[S|80yd] ≈ 1 + 2.7 = **3.7**（さらにラフ等で 3.9 まで悪化しうる）
- 刻む（残り 90yd FW から普通に乗せる）→ E_layup ≈ **3.0**

→ p* = (3.7 − 3.0)/(3.7 − 2.6) = 0.7/1.1 ≈ **0.64**

つまり「池に入れずグリーン周りに運べる確率が約64%以上」なら go が +EV。実際のアマが 230yd・池越えで届かせられる確率はしばしば 30〜45% であり、この閾値を割るため **刻むのが正解**になる。逆に E_fail が小さい（池でなくラフで E_fail≈3.2 程度）なら p* = (3.2−3.0)/(3.2−2.6) = 0.33 と激減し、**ハザードの罰の重さが go/no-goを支配する**ことが式から直読できる。失敗時の損失（E_fail−E_layup）が大きいほど、また成功時の利得（E_layup−E_success）が小さいほど、要求確率 p* は上がる。

### 3. 「フルショット距離に刻め」は falsism ─ レイアップ距離最適化
通念「お気に入りウェッジでフルスイングできる距離（例 100yd）に刻め」を、Broadie は **falsism（もっともらしい誤り）** と断じた。理由は近接度（proximity）の単調性にある。Arccos / Shot Scope のアマ実測：

| レイアップ後の残り | 平均近接度（HCP別 or 平均） |
|---|---|
| 50 yd | ≈ 35〜44 ft |
| 60–80 yd | 0–5 HCP 41 ft / 11–15 HCP 50 ft / 20+ HCP 62 ft |
| 100–120 yd | 0–5 HCP 49 ft / 11–15 HCP 62 ft / 20+ HCP 83 ft |
| 110 yd（平均） | ≈ 64 ft |

→ **近いほど近接度が良く E[S] が小さい**。100–120yd を「フルウェッジだから」とわざわざ残すより、可能な限り前へ運んで 60–80yd を残す方が、近接度で 8〜20 ft 得をする。E[S] 差にして概ね 0.05〜0.12 打／ホール。これは「フルスイング vs ハーフショットの分散」議論より、proximity の単調減少が支配的だという主張。

**ただし重要な非単調性（partial wedge の谷）**: 50yd 付近ではアマの近接度が頭打ち〜やや悪化する（3HCP で 50yd=43.9 ft, 18HCP で 44.0 ft とほぼ差が消え、60–80yd と大差ない）。30〜50yd の「半端な距離」はスピンロフトと打点の分散が増え、E[S] 曲線がローカルに平坦化／反転する谷を作る。したがって最適レイアップ距離は「いちばん前」ではなく、**ハザードに捕まらない範囲で、近接度曲線が悪化に転じる手前（多くのアマで 60〜90yd 帯）**。DECADE 系も「ハザードがなければ前へ、グリーン手前にハザードがあれば理想ウェッジ帯（おおむね 80〜100yd）に置く」と整合する。

### 4. 散布（散布楕円）を畳み込んだ期待値
レイアップ距離 t の真の評価は、狙い t に対する着弾の2次元正規分布 N(μ(t), Σ(t)) を E[S] 場に畳み込んだ
**E[S|t] = ∫ E[S | s'] · f(s'; μ(t), Σ(t)) ds'** である。ポイントは E[S] 場が凸でない（ハザード境界で不連続にジャンプする）こと。すると：
- ハザードの縁ギリギリを狙うと、散布楕円の裾が崖（罰打）に落ちる確率が立ち上がり、平均は内側の点推定より大きく悪化する（**Jensen的ペナルティ**）。
- ゆえに最適狙いは「点推定の最良点」より**ハザードから散布σ分だけ内側**にシフトする。これは既習の「狙いはピンでなく散布を考えた安全側へずらす」と同じ原理を、レイアップに適用したもの。
- 距離の散布（縦ブレ）も同様で、グリーン手前ハザードを越える3打目を残すとき、キャリー要求の達成確率を σ_carry で評価し、未達確率×罰を加味した E[S] で刻み距離を決める。

### 5. 集計データによる裏取り
- Shot Scope: アマの「無条件レイアップ」は **約0.6打/パー5の損**。届く見込みがあるのに刻むのは取りこぼし。一方で届かないのに無理に go も損で、判断は p* 閾値が分ける。
- 集計では go-for-it 群が平均 **約0.42打**良いとの報告もあるが、これは「届く位置にいる母集団」のセレクションが効いた条件付き平均であり、母集団を揃えれば p* 判定に帰着する。
- 3打目の green-hit率はおおむね 175yd 以内で急増。「2打目は175yd以内に運べる前進レイアップを」という基準と、近接度単調性は同じことを言っている。

## 気づき・洞察
- go/no-go の本質は **損失の非対称性**。p* 式の分母 (E_fail − E_success) は「賭けのスケール」、分子 (E_fail − E_layup) は「刻みに対する失敗の超過損」。ハザードが池（罰打）かラフ（罰なし）かで E_fail が 0.5打動き、p* が 0.64↔0.33 と倍近く動く。**狙うか刻むかは飛距離の問題ではなくハザードの質の問題**だと式が明言する。
- 「フルウェッジ距離に刻め」の誤りは、人間が**分散（自分の苦手な半端距離）を過大評価し、近接度の単調利得を過小評価する**バイアスの表れ。実データの近接度はほぼ単調なので、分散懸念より前進が勝つ。例外は 50yd 以下のローカルな谷だけ。
- E[S] 場の非凸性（ハザードの崖）こそが「安全側にずらす」最適化の数学的根拠。凸な場なら点推定最良点がそのまま最適だが、崖があるため散布を畳み込むと最適点が内側へ動く。これは在庫管理の newsvendor（過剰在庫 vs 欠品の非対称コストで最適発注点が期待需要からずれる）と同型の構造。

## 他分野との接続
- **house-hunting / 不動産**: go/no-go と break-even 確率は、出口戦略の意思決定（高値挑戦して長期化リスク vs 早期確定）と同型の決定理論。「失敗時の損失の重さ」が閾値を支配する点も共通（[[2026-06-04 出口戦略とリセールバリュー]] の論理）。
- **anatomy / 運動制御**: 散布 Σ は身体側の再現性が決める。戦略（どこを狙うか）と実行（どれだけ散るか）は分離でき、本記録は前者に専念する一方、Σ を縮める努力は anatomy 側の課題。最適化は「Σ を所与とした条件付き最適狙い」。
- **music / piano**: E[S] 場の非凸性ゆえに「点最良より安全側」という考えは、ミスの代償が非対称な箇所（暗譜の不安定な接ぎ目）で安全マージンを取る判断と通じる。
- **横断テーマ**: newsvendor / 期待値最小化 / Jensen の不等式 という確率最適化の骨格が、ゴルフ・不動産・運動制御を貫く。

## 次に深掘りしたいこと
- レイアップ距離の連続最適化を、近接度曲線 prox(d) と散布 Σ(d) を関数フィットして E[S|t] を数値積分し、HCP帯別の最適刻み距離を求める（partial wedge の谷の位置を実データで同定）。
- グリーン手前ハザード越え3打目の **キャリー要求達成確率モデル**：縦散布 σ_carry を距離・クラブ別に与え、未達ペナルティ込みで「刻み距離 × 使用クラブ」の同時最適化。
- ベイズ的「今日の調子」更新（frontier 既掲）を p* 推定に接続：当日の数球から成功確率 p の事後分布を更新し、go/no-go 閾値を動的に調整する手続きと過学習リスク。
