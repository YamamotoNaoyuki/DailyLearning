# 専有部分と共用部分の境界線：玄関ドア・サッシ・バルコニーの権利関係
**日付**: 2026-05-15  
**分野**: house-hunting  
**タグ**: #区分所有法 #専有部分 #共用部分 #管理規約 #標準管理規約 #修繕責任

## 学んだこと

### 「自分の部屋」と思っていても、実はそこの所有権は…

マンションの一室を購入すると「自分が所有する部分」だと感じるが、法的にはそこは複雑な権利構造の上に成り立っている。区分所有法（建物の区分所有等に関する法律、1962年制定、2024-2026年改正）の体系では、マンションの構造は **専有部分**（区分所有権）と **共用部分**（区分所有者全員の共有）に厳密に分けられる。境界線の認識を間違えると、修繕費の負担・リフォームの可否・トラブル時の責任所在で大きな損失を被る。

### 専有部分の定義（区分所有法 2条3項、4条1項）

専有部分とは「区分所有権の目的たる建物の部分」。標準管理規約（国土交通省）に基づく一般的な範囲：

- **住戸内部の壁・床・天井の仕上げ材から内側**
- **室内の設備配管（給水管・排水管のうち専有部分内）**
- **キッチン、浴室、トイレなどの住戸内設備**
- **室内建具（部屋を区切るドア）**

ここで重要なのは **「内側」の境界が建築的構造によって異なる** こと：

1. **躯体内壁説**: コンクリート躯体の内面まで（仕上げ材は専有）
2. **壁芯説**: コンクリート躯体の中心線まで（半分は共有、半分は専有）
3. **躯体外壁説**: 躯体の外面まで（実務上は採用されない）

→ 標準管理規約は**躯体内壁説**を採用しているが、登記簿上の床面積（壁芯）と異なるため、混乱の元になる。

### 共用部分の3分類

#### 1. 法定共用部分（区分所有法上、当然に共用部分）
- 廊下、階段、エレベーター、エントランス、屋上、外壁
- 受水槽、ポンプ室、機械室、共用配管
- 性質上、区分所有の対象とできない

#### 2. 規約共用部分（規約で共用と定めた部分）
- 集会所、管理人室、トランクルーム
- 本来は専有可能だが、規約で共用と定めた

#### 3. 一部共用部分（一部の区分所有者のみが共有）
- 例：1階の店舗用エントランスは店舗区分所有者のみ
- ツインタワーの片側エレベーター

### 専有部分か共用部分か揉めやすい3大ポイント

#### ① 玄関ドア（最大の誤解ポイント）

「自分の家の玄関ドア」は実は**ほぼ共用部分**：
- **ドアの外側塗装、ドア枠、ドアハンドル外側、ドアスコープ** → **共用部分**
- **ドアの内側塗装、室内側ノブ、ドアチェーン** → **専有部分**
- **錠（鍵）** → **専有部分** ←ここだけ専有

→ 玄関ドアの色を勝手に変えると規約違反。シリンダー交換は所有者の自由。

#### ② サッシ・窓ガラス

これも**共用部分**：
- **窓枠（サッシ）、窓ガラス、雨戸、面格子** → **共用部分**
- **網戸** → **専有部分扱い** が多い（規約による）

→ 二重サッシ（内窓）を設置する場合：
- **室内側にもう一つサッシを足す（インナーサッシ）** → 専有部分の追加なので OK
- **既存サッシを断熱性能の高いものに交換** → 共用部分の改変なので原則 NG（管理組合の承認が必要）

これが [[内窓メーカー徹底比較]] や [[断熱リフォーム補助金制度]] で重要：先進的窓リノベ事業の補助金が「内窓設置」中心なのは、外側サッシ交換が区分所有法上ハードルが高いから。

#### ③ バルコニー

バルコニーは**共用部分**だが、当該住戸の区分所有者のみが**専用使用権**を持つ：
- **避難経路としての機能**（火災時の隣戸への避難）→ 共用とせざるを得ない
- **建物外観の一部** → 共用
- **室外機の設置は専用使用権の範囲内で可**
- **物置・ガーデニング**は規約で制限される（避難経路を塞がない範囲）
- **タイル張りや床仕上げ** → 撤去可能であれば原則 OK だが、防水層を傷つける改造は不可

→ バルコニーから飛び降りた・落とした物の責任、隣接バルコニー間の蹴破り板の維持などはすべて管理組合マター。

### 配管問題：専有か共用か

最も実務で揉めるのが配管：
- **共用立て管（縦の幹線）** → **共用部分**
- **共用立て管から分岐して各住戸内に入る枝管** → **専有部分**
- ただし「住戸内にあっても主要な共用設備の一部」とみなされる場合あり

→ 排水管詰まり事故で「自室で水漏れしているが、原因は上階の配管」のようなケースで、責任の押し付け合いが起きる。多くの判例：
- **専有部分内の枝管の漏水** → 当該区分所有者の責任
- **共用立て管の漏水** → 管理組合（全員）の責任
- **特定不能なケース** → 規約や個別事情で判断

### 修繕責任の所在

| 部位 | 区分 | 修繕費負担 |
|------|------|----------|
| 室内の壁紙、フローリング | 専有 | 区分所有者 |
| 玄関ドア（鍵以外） | 共用 | 管理組合（修繕積立金） |
| サッシ・窓ガラス | 共用 | 管理組合 |
| バルコニーの防水層 | 共用 | 管理組合（大規模修繕で） |
| バルコニー内のタイル | 専用使用部分 | 区分所有者（規約による） |
| 室内のエアコン | 専有 | 区分所有者 |
| エアコン室外機 | 専用使用 | 区分所有者 |
| 共用立て管 | 共用 | 管理組合 |
| 室内の給排水枝管 | 専有 | 区分所有者 |

これは **[[マンション管理組合の仕組み]]** と **[[長期修繕計画と大規模修繕]]** で扱う「何を管理組合が直すべきか」の根幹。

### リフォーム可否の判断ルール

専有部分のリフォームは原則自由だが、**管理組合への届出または承認** が必要なケース：

#### 届出のみで可
- 室内の壁紙・床材変更
- システムキッチン交換
- 浴室ユニットバス交換（同等以下の重量）
- 室内建具交換

#### 承認が必要（区分所有法 17 条、規約による）
- 床のフローリング化（遮音等級の確保が必要）
- 室内壁の撤去・新設（構造躯体に手を加えない範囲）
- 配管の経路変更
- インナーサッシ設置（場合による）

#### 原則不可（共用部分への改変）
- 外側サッシ交換
- 玄関ドア交換
- バルコニー手すり変更
- 外壁への穴あけ（エアコン穴は規約による）

### 2024 年区分所有法改正の影響

[[マンション建替え制度と区分所有法2026年改正]] で扱った改正のうち、専有・共用関連の重要点：
- **「専有部分の管理についても管理組合が一定の関与をできる」** 規定の追加：荒廃した専有部分への対応
- **共用部分の変更決議要件の緩和**：4分の3 → 3分の2（一部）
- **所在不明区分所有者の議決権除外**：放置物件への対応

これにより、長く放置された専有部分が共用部分の維持を阻害するケースの解決が進めやすくなった。

## 気づき・洞察

**「所有」の感覚と「法的範囲」の乖離**。多くの居住者は「玄関ドアは自分のもの」と思っているが、実は外側は共用部分。これは「**所有感**」と「**法的所有権**」が違うことを示す好例。マンションは本質的に共同所有体であり、自分の専有部分は実は意外に狭い。「**自分の城ではなく、共有体の一部屋を借りているような状態**」と理解するのが実情に近い。

**「専用使用権」という第三の概念**。バルコニーは共用部分だが、当該区分所有者のみが使う「専用使用」がある。これは所有でも借用でもない第三の状態で、日本の物権法体系では特殊な位置にある。**バルコニーで「私の場所」と感じるのは、所有ではなく専用使用権を持っているから**。

**判断の最終根拠は規約**。区分所有法は「原則」を定めるが、具体的にどこまでが専有・共用かは **各マンションの管理規約で定義** される。同じマンションでも規約改正で境界が動く。**[[重要事項説明の読み方]] で管理規約を読む重要性** はここから来る。

**境界の曖昧さは紛争の源**。配管・防水層・梁・柱など、誰の責任か曖昧な箇所が多数ある。マンション購入時の重要事項説明では「規約と細則」を必ず読み、特に修繕負担区分を確認すべき。**[[マンション専有部と共用部の違いを探る ドアの外側は全部共用部？|KENSOマガジン]]** が指摘するように、「外側は全部共用」のメンタルモデルが実用的。

## 他分野との接続

- **anatomy / 神経分類**: A 線維と C 線維の境界が「直径とミエリンの有無」で明確に分かれるように、専有・共用の境界も「躯体内壁」という明確な物理基準で分かれる。**物理的構造による分類**
- **tech / マイクロサービス境界**: マイクロサービスのドメイン境界（DDD bounded context）は、責任範囲の明確化が肝。マンション区分所有も同じく「責任範囲の明確化」が制度設計の本質
- **cs / OS と権限分離**: カーネル空間（共用部分）とユーザー空間（専有部分）の境界に近い。境界を越える操作にはシステムコール（管理組合承認）が必要
- **music / 演奏会場の利用**: コンサートホールの楽器使用権、リハーサル室の予約権など、共有スペースの専用使用権の例。世界共通の構造
- **golf / コース運用**: ゴルフ場のメンバー権はマンション専用使用権に近い。コース自体は法人所有、メンバーは使用権を持つ
