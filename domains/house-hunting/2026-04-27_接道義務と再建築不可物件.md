# 接道義務と再建築不可物件 — 42条道路とセットバックの落とし穴
**日付**: 2026-04-27
**分野**: house-hunting
**タグ**: #接道義務 #建築基準法42条 #再建築不可 #セットバック #みなし道路 #43条但し書き #位置指定道路

## 学んだこと

### 接道義務とは何か
**建築基準法第43条第1項**:

> 「建築物の敷地は、道路に**2メートル以上接しなければならない**。」

これが「**接道義務 (せつどうぎむ)**」。都市計画区域・準都市計画区域 (=日本の都市部のほぼ全て) 内の建物が対象。**幅員4メートル以上の道路 (建築基準法上の道路)**に**敷地の境界線が連続2メートル以上接していること**を要求する。

立法目的は3つ:
1. **緊急車両のアクセス確保** (消防車・救急車が入れること)
2. **避難経路の確保** (火災・地震時に逃げられること)
3. **採光・通風・防火** (建物が密集しすぎないこと)

戦後の大規模火災 (1945-50年代の都市火災) を踏まえた**消防法的・都市衛生的観点**から1950年に制定。

### 「建築基準法上の道路」の定義 — 第42条の体系
ここで言う「道路」は**民法上の「通れる道」とは別**で、**建築基準法42条**で厳密に定義される7種類:

| 条項 | 通称 | 定義 |
|---|---|---|
| 42条1項1号 | **道路法による道路** | 国道・県道・市道で幅員4m以上 |
| 42条1項2号 | **開発許可道路** | 都市計画法・土地区画整理法等で築造された道路 |
| 42条1項3号 | **既存道路** | 1950年(基準法施行時)以前から幅員4m以上で存在 |
| 42条1項4号 | **計画道路** | 2年以内に事業執行予定で特定行政庁指定 |
| 42条1項5号 | **位置指定道路** | 私道だが特定行政庁が指定した幅員4m以上の道路 |
| **42条2項** | **みなし道路 / 2項道路** | **幅員4m未満**だが基準法施行時に既に建築物が立ち並ぶ道路。**特定行政庁が指定** |
| 42条3項 | 3項道路 | 2項道路で**水平距離2.7m以上**の幅員に「**みなし**」する特例 (商業地の狭隘道路特例) |

このうち**最も日常的に問題になるのが2項道路**。

### 42条2項道路 (みなし道路) — 戦前からの狭い道
1950年の基準法施行時、既に都市部の多くの私道は幅員4m未満だった (戦前の市街地は2-3m道が標準)。これら全てを「不適格」にすると都市が機能不全になるため、**2項道路は「事実上の道路」として基準法上の道路扱い**を受ける。

ただし、その代償として:
- **道路の中心線から両側 2m を「**道路内**」とみなす** (4m幅を確保したことにする)
- 建物建築・建替え時に**敷地を後退させる必要**がある——これが**セットバック (壁面後退)**

### セットバックの実装
具体的には2方式:

**(a) 中心振り分け方式**: 道路中心線から両側に2mずつ。両側の所有者が**それぞれ後退**する。
```
[家A]——|—中心線—|——[家B]
       2m         2m
```

**(b) 一方後退方式**: 対面が崖・河川・線路など物理的に後退できない場合、片側だけが**4m後退**する。
```
[崖|川]======|===4m===[家B] (家Bが4m後退)
```

セットバック部分は:
- **建物・塀・門・植栽・物置・駐車場** などの配置不可
- **建ぺい率・容積率の計算からも除外** (敷地面積に算入できない)
- 所有権は元の所有者だが**実質的に道路用地**——固定資産税は減免されるが**売却は実質困難**

### セットバックの隠れたコスト
中央値的な狭小住宅地を想定 (敷地100㎡、間口10m、2項道路に接する):
- 中心線から1.5m地点に道路境界がある場合、**0.5m × 10m = 5㎡を後退**
- 実効敷地は **95㎡** (5%減)
- 容積率150%なら、**建てられる延床面積が 150㎡ → 142.5㎡** (▲7.5㎡)
- 物件価値は**敷地面積に比例**するため、**約 5% の資産価値減**

加えて、道路工事時に**自費で舗装を行う事例**もあり (自治体により異なる)、**売り出し時にセットバック未実施の物件は値引き要因**。

### 再建築不可物件
**接道義務を満たさない既存建物**は、現状のまま住めるが**建替え不可**——これが**再建築不可物件**。代表的パターン:

1. **完全に道路に接しない (袋地)**: 周囲を他人の土地に囲まれた **無接道地**
2. **接道幅が2m未満**: 1.8m や 1.5m で接道
3. **接している道が42条道路でない**: 通称「**通路**」「**里道**」「**赤道**」(法定外公共物) しか接していない

特に**2号物件 (袋地)** は、**民法上の囲繞地通行権**で通行は保証されるが、**建築基準法上の接道義務は満たさない**——つまり**通れるが建てられない**という奇妙な状態。

### 43条但し書き (現43条第2項) — 救済規定
**建築基準法第43条第2項**は、接道義務の例外を定める:

> 「特定行政庁が、その敷地の周囲に広い空地があることその他**国土交通省令で定める基準に適合する**と認めて建築審査会の同意を得て**許可**したものについては、第1項を適用しない。」

つまり**特定行政庁の個別許可**を得れば、再建築不可物件でも建替え可能になることがある。この許可を得る通称「**43条許可**」「**但し書き許可**」と呼ぶ。許可されやすいパターン:

- 周囲が公園・農地・河川などで広く空地が確保されている
- 接道はないが**隣地通路を緊急車両が通れる程度に確保**されている
- 既存の通路を**位置指定道路 (42条1項5号)** に格上げできる

ただし**自治体ごとに運用基準が大きく異なる**ため、購入前に**特定行政庁 (市町村の建築指導課) で個別相談**が必須。「**買ってから許可申請、不許可で住宅ローンが破綻**」という事例が多発する。

### 価格と流動性
再建築不可物件の市場価値は**周辺相場の30-70%**。投資家視点では:
- **賃貸用としては激安で利回りが高い** (利回り15-20%超もある)
- **建替え不能のため出口戦略が極めて限定的** (買い手が少ない)
- **住宅ローン審査が厳しい** (担保価値が低い)
- **火災・倒壊で更地化したら、二度と建てられない** (=投資が消失する)
- **隣地買収で接道を確保** できれば一気に正常物件化、5-10倍値上がり可能

### 位置指定道路 (42条1項5号道路) の実務
**位置指定道路**は、私人が築造して**特定行政庁の指定**を受けた私道。指定の要件:
- 幅員 4m 以上
- 2項道路と同じく中心線から両側 2m が道路敷
- 排水設備・舗装が一定基準
- 路上に建物・塀がないこと
- 隅切り (角を斜めに切る) があること

新規分譲開発で**ミニ開発(2-4区画)**を行う際、**位置指定道路を引いて各区画を接道させる**手法が主流。「**位置指定道路の私道持分**」が物件の付随権利として登記される——**修繕費負担・通行掘削権の規定**を売買契約で確認することが重要。

### 「敷地延長 (旗竿地)」の接道判定
**旗竿地 (はたざおち / flag-shaped lot)** は、**細い通路 (竿)** で接道する変則敷地。例:

```
┌───────┐
│       │
│  家   │
│       │
└┬──────┘
 │ (竿: 通路、幅2m)
 │
 道路
```

**竿の幅が2m以上であること**、**竿が接道道路にも2m以上接していること**が必須。竿の長さ (奥行) には法律的な制限はないが、**自治体条例で「20m以上の竿は接道幅3m以上が必要」など追加規制**がある場合多数 (神奈川県・横浜市は厳しい)。

旗竿地の評価は通常の整形地より**10-30%安い**——竿部分は実質的な利用が困難 (駐車場くらい)、採光通風が悪い、リフォーム・修繕の搬入が大変。

### 接道義務の最新動向 — 2025年以降
2025年4月施行の**建築基準法改正 (2022年公布)** で:
- **省エネ基準適合義務化** に伴う既存建物の増改築が増加 → 接道問題の再検討事例急増
- **空き家対策特別措置法**との連動で、再建築不可空き家の**特定空き家**指定→**強制撤去**事例が増加
- 一部自治体で**43条許可の運用緩和** (例: 横浜市の43条運用基準改定 2024)
- **接道義務の規制緩和** (4m → 2.7m など) は議論されたが、消防庁の反対で見送り

## 気づき・洞察

### 「建てられる」と「通れる」の制度的乖離
民法上の通行権 (囲繞地通行権・地役権・通行地役権) は**個人間の関係**。建築基準法の接道義務は**社会的安全規制**。この二つは**全く別の論理体系で動く**——「通行権は確保されているから建てられる」と勘違いして購入した投資家が破産する事例は枚挙にいとまがない。**法律のレイヤー分離**を理解することが不動産投資の基本——**民法と公法は別世界**。

### 戦後の都市計画の「沈澱物」としての2項道路
2項道路は**1950年の現状追認**で生まれた歴史的経過措置だが、**75年経った今も大量に残存**。これは**変えられない過去**を制度に組み込んだ事例で、**法律が現実に妥協する瞬間**。完全に「正しい」道路だけを認めれば都市が動かなくなる——**現実主義的な立法**の代表例。同時に「**そろそろ更新すべき制度**」とも言える——次世代の都市政策の課題。

### 再建築不可物件 = 「時間制限つき資産」
再建築不可物件は**物理的存続期間と経済的存続期間が一致**する稀有な資産——建物が朽ちた瞬間に**経済価値が消失**する。これは通常の不動産が「土地は永続、建物は減価」という二元論で評価されるのと根本的に違う。**会計的には負債性資産**——維持費が増えるたびに**清算価値が減少**する。**時間が敵となる資産**は通常の資産評価モデルが適用できない。

### 43条但し書き = 「裁量行政の窓」
43条但し書きの存在は、**機械的な法適用の限界**を認める制度的知恵。建築基準法は厳密な数値基準で動くが、**例外事象には特定行政庁の裁量で対応**する余地を残す。これは**法律と運用の二重構造**——法律が硬直的だから運用が柔軟になる。**裁量行政は腐敗の温床**にもなりうるが、**機械的法執行の硬直性**を救う必要悪。

### セットバック = 「私有財産の徐々の社会化」
セットバック制度は、所有権を保ちつつ**実質的に道路用地として供出**させる——**完全国有化なき公共化**の精緻な装置。これは**コモンズ理論**の制度実装の一例で、**私的所有と公共利用の境界をグラデーション化**する。**所有 vs 利用**の二項対立を超える法的工夫として興味深い。

### 横浜エリアの特殊性 (家探し文脈)
横浜市は**戦前からの旧市街地 (中区・西区・南区)** で2項道路の比率が高く、**港北・都筑・青葉区** などの戦後開発地は位置指定道路が主流。横浜市建築局は**43条許可の運用が比較的厳格** (神奈川県の中でも厳しい部類) で、**建築指導課への事前相談が必須**。一方で**準工業地域**や**第一種低層住居専用地域**で接道判定が異なる場合があり、**用途地域 + 接道判定**の二段階チェックが実務。

## 他分野との接続

- **house-hunting (定期借地権・媒介契約)**: 「**所有権は持てるが利用に制約がある**」構造として相同——再建築不可は**時間軸の制約**、定期借地権は**期間の制約**。**所有の質的多様性**
- **cs (MVCC + VACUUM)**: VACUUM は dead tuple を回収する——再建築不可物件は**回収不能な dead asset**。**メモリ管理の物理的不可逆性**と相同
- **tech (eBPF verifier の制約)**: 「**問題を解析可能領域に収める**」設計——基準法は**全建物の安全性を保証する代わりに、可能な建築範囲を制限**する。**制約の受容で安全性を得る**
- **golf (Pelz 17インチ・失敗時許容)**: 接道義務は「**緊急時の避難・消防アクセスを保証する**」——**普段は使わないが、失敗時に必須**な余裕設計。**冗長性が安全を生む**点で同型
- **anatomy (筋膜の張力ネットワーク)**: 道路 = 都市の**筋膜**——建物 = 個別筋肉、道路網 = 連絡組織。**個別では機能せず、ネットワーク全体の連続性が機能を支える**
- **bread (湯種法・水分マトリックス)**: 「**全体の機能を保つために、局所の構造を強制する**」点で同型——湯種は水分の局所固定、接道義務は安全のための形状規制
- **other (印欧祖語のグリムの法則)**: 「**過去の状態が現在の構造を規定する**」歴史的経路依存——2項道路の存在は1950年の都市状態の化石、グリムの法則は数千年前の音韻変化の化石
