# 中古マンション断熱リノベーション
**日付**: 2026-03-09
**分野**: 家探し
**タグ**: #断熱リノベ #内窓 #UA値 #費用対効果

## 学んだこと

### 1. 内窓（インナーサッシ）の仕組みと効果

**基本構造**: 既存窓の室内側にもう1枚の窓（サッシ+ガラス）を取り付け、二重窓化する。既存窓と内窓の間に空気層（中空層）が生まれ、これが断熱・防音の緩衝帯として機能する。

**なぜマンションで有効か**: マンションの窓・サッシは「共用部分」であり、管理規約上、勝手に交換できない。内窓は室内側に新たに取り付けるため「専有部分」の工事となり、管理組合の承認なしで施工可能なケースが多い（※規約により届出が必要な場合あり）。

**断熱効果（熱貫流率の劇的改善）**:

| 窓の仕様 | 熱貫流率 U値 [W/㎡K] |
|---------|---------------------|
| アルミサッシ＋単板ガラス（築古マンションの典型） | 6.51 |
| アルミサッシ＋複層ガラス | 4.07 |
| 内窓追加（樹脂サッシ＋Low-E複層ガラス） | **1.75** |
| 内窓追加（樹脂サッシ＋Low-E複層・アルゴンガス） | **約1.5以下** |

→ 単板アルミサッシ（6.51）から内窓Low-E複層（1.75）への改善は、**約3.7倍の断熱性能向上**。窓が熱損失の40-50%を占めるため、住戸全体のUA値を約14%改善できるとの試算がある。

**副次的効果**:
- **結露抑制**: 室内側ガラスの表面温度が上がり、結露が大幅に減少
- **防音**: 二重窓の空気層が遮音性を高め、交通騒音を10-15dB低減（体感で半分程度）
- **防犯**: 窓が二重になることで侵入抑止効果

### 2. 施工費用の相場

**内窓1箇所あたりの費用**:

| 項目 | 費用目安 |
|------|---------|
| 内窓本体（樹脂サッシ＋Low-E複層ガラス） | 5万〜12万円 |
| 施工費 | 3万〜7万円 |
| **1箇所あたり合計** | **8万〜15万円** |
| ふかし枠（窓枠の奥行き不足時に必要） | +数万円/箇所 |

**一般的なマンション一戸あたり（70㎡・3LDK想定）**:
- 窓の数: 6〜8箇所（リビング2、各居室1×3、キッチン1程度）
- 全窓内窓化: **約50万〜120万円**
- リビング＋主寝室のみ: **約25万〜45万円**（費用対効果の高い選択）

**注意**: マンションでは窓枠の奥行きが浅い場合が多く、ふかし枠の追加費用が発生しやすい。事前の現地調査が重要。

### 3. UA値の改善シミュレーション

中古マンション（築20年・70㎡・6地域=横浜）を想定:

| 状態 | 推定UA値 | 断熱等級 |
|------|---------|---------|
| 改修前（アルミサッシ単板、無断熱〜薄い断熱） | 1.5〜2.0程度 | 等級2〜3 |
| 内窓のみ追加 | 1.2〜1.5程度 | 等級3〜4 |
| 内窓＋床壁天井断熱 | 0.6〜0.8程度 | 等級4〜5 |
| フル断熱改修（高性能窓+充填断熱） | 0.5以下 | 等級5〜6 |

→ **内窓だけではZEH基準（UA値0.60）には届かない**が、省エネ基準（0.87）には近づける。ZEH水準を目指すなら床・壁・天井の断熱も必須。

### 4. 床・壁・天井の断熱改修オプション

| 部位 | 工法 | 費用目安（6畳間） | 特徴 |
|------|------|-----------------|------|
| 壁 | 充填断熱（グラスウール等） | 約30万円 | コスト安だが施工精度に依存 |
| 壁 | 断熱パネル貼り | 約40万円 | 室内が数cm狭くなる |
| 床 | 充填断熱 | 約30万円 | 1階や最下階で効果大 |
| 天井 | 充填断熱（吹き込み） | 約15万円 | 最上階で効果大 |
| 一体 | 発泡ウレタン吹付け（壁+床+天井） | 30万〜40万円/室 | 隙間なく施工でき断熱効果が最も高い |

**マンション特有の制約**:
- **外壁側のみ施工可能**: 隣戸との界壁は断熱不要（隣も暖房している前提）
- **最上階・1階が不利**: 屋上や地面に接する面の熱損失が大きい→天井・床断熱の優先度が上がる
- **中住戸が有利**: 上下左右を他住戸に囲まれた「中住戸」は外気に接する面が少なく、窓の断熱だけでも効果が大きい
- **躯体（コンクリート）は共用部**: 外壁のコンクリート自体には手を加えられない。室内側から断熱材を貼る「内断熱」が唯一の選択肢
- **天井高が下がる**: 天井・壁に断熱材を入れると室内寸法が縮小する（天井で3〜5cm、壁で3〜5cm程度）

### 5. 補助金制度：先進的窓リノベ2026事業

**概要**:
- 環境省の補助金事業。令和7年度補正予算で**1,125億円**を計上
- 対象: 既存住宅の窓断熱改修（戸建て・マンション問わず）
- 補助上限: **一戸あたり最大100万円**（2025年の最大200万円から半減）

**2026年の重要な変更点**:
- **Aグレード内窓が対象外に**: 2025年まではAグレード（Uw 2.3以下）も対象だったが、2026年は**Sグレード（Uw 1.5以下）またはSSグレードのみ**が対象
- 「特大サイズ」カテゴリの新設
- → Low-E複層ガラス以上の高性能窓を選ばないと補助金が出ない

**補助金額のイメージ（内窓・Sグレード）**:
- 大サイズ（リビング掃き出し窓等）: 約3万〜5万円/箇所
- 中サイズ（腰窓等）: 約2万〜4万円/箇所
- 小サイズ（トイレ・浴室等）: 約1万〜2万円/箇所
- 3LDK全窓で: **約15万〜30万円の補助**が見込める

**他の補助金との併用**:
- 「子育てグリーン住宅支援事業（みらいエコ住宅2026）」との併用が可能
- 窓リノベが必須工事で、それに加えて給湯器交換・バリアフリー改修なども補助対象に

### 6. 費用対効果シミュレーション

**ケース: 築25年・70㎡・3LDKマンション（横浜市）を全窓内窓化**

| 項目 | 金額 |
|------|------|
| 内窓施工費（7箇所・Low-E複層） | 約80万円 |
| 先進的窓リノベ補助金 | ▲約20万円 |
| **実質負担** | **約60万円** |
| 年間光熱費削減（冷暖房費） | 約1.8万〜2.4万円/年 |
| **回収期間** | **約25〜33年** |

**考察**:
- 光熱費削減だけでの投資回収は25年以上かかり、純粋な経済計算では「元が取れにくい」
- ただし以下の**定量化しにくいメリット**が大きい:
  - 結露によるカビ・ダニ抑制 → 健康面のメリット（特に子育て世帯）
  - ヒートショック予防 → 室内温度差の縮小
  - 快適性の向上 → 窓際の冷輻射（コールドドラフト）が消える
  - 防音効果 → 幹線道路沿いでは生活の質が大幅改善
  - **将来の資産価値**: 断熱等級が住宅性能の重要指標に。2028年以降はZEH水準以上でないとローン減税対象外になるため、**断熱性能が資産価値に直結**する時代が来る

**フル断熱改修（内窓+壁+床+天井）の場合**:
- 施工費: 200万〜350万円
- 補助金: ▲30万〜50万円
- 実質負担: 170万〜300万円
- 光熱費削減: 約4万〜6万円/年
- → スケルトンリノベーション（間取り変更等）と同時施工すれば、断熱のみの追加コストは抑えられる

## 気づき・洞察

- **「内窓だけ」は最もコスパが高い断熱改修**。施工が1箇所1〜2時間と手軽で、壁や床を壊す必要がない。リノベーション予算が限られるなら、まず内窓から
- **中住戸（上下左右が他の住戸）なら内窓だけでもかなり快適になる**。外気に接する面が窓のみなので、窓を制すればほぼ断熱を制する
- **最上階・1階・角住戸は断熱的に不利**だが、その分中古価格が安い場合がある。断熱リノベの費用を見込んで「安い不利住戸＋断熱改修」という戦略もある
- **2026年の補助金基準厳格化（Aグレード除外）は実は追い風**: Sグレード以上を選ぶことで、より高い断熱効果が得られる。中途半端な改修を避ける方向にインセンティブが設計されている
- **2028年問題**: 省エネ基準適合レベルでは住宅ローン減税が受けられなくなる。中古マンション購入＋断熱リノベでZEH水準に引き上げれば、減税メリットを維持しつつ快適性も確保できる——**購入戦略と断熱戦略はセットで考えるべき**

## 他分野との接続

- **解剖学**: ヒートショック（急激な温度変化による血圧変動）は断熱不足の住宅で起きやすい。入浴時の脱衣所-浴室間の温度差が特に危険。内窓による室内温度の均一化は循環器系への負担軽減に直結する
- **パン（サワードウ）**: 発酵に最適な温度管理（25-28℃）は住宅の断熱性能に依存する。高断熱住宅ほど室温が安定し、長時間発酵の温度管理が容易に
- **哲学・美学**: 日本の伝統的住宅は「夏を旨とすべし」（吉田兼好）で通気性重視だった。現代の高断熱志向は西洋的な「環境を制御する」思想。しかし内窓は既存の窓を残しつつ重ねる——和洋折衷の実用的解法とも言える
- **テクノロジー**: スマートホームのHEMS（Home Energy Management System）と断熱改修を組み合わせれば、エネルギー消費の「見える化」と「削減」を同時に実現。IoT温湿度センサーで断熱効果を定量的に検証できる
- **ゴルフ**: コースマネジメントの「リスクとリターンの最適化」は、断熱リノベの費用対効果分析と同じ思考構造。限られた予算をどの部位に配分するか＝どのショットにリスクを取るか
