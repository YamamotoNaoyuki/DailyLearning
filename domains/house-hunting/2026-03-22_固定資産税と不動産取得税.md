# 固定資産税と不動産取得税
**日付**: 2026-03-22
**分野**: 家探し
**タグ**: #固定資産税 #不動産取得税 #都市計画税 #税金 #固定資産税評価額

## 学んだこと

### 不動産に関わる「4つの価格」を理解する

不動産購入を検討する際、まず混乱するのが「土地の値段が複数ある」という事実だ。日本の不動産には4つの公的価格が存在し、それぞれ目的が異なる。

| 価格指標 | 決定機関 | 公示地価との比率 | 用途 |
|---------|---------|----------------|------|
| **公示地価** | 国土交通省 | 100% | 土地取引の指標 |
| **実勢価格**（時価） | 市場 | 90〜150%（エリアによる） | 実際の売買価格 |
| **相続税路線価** | 国税庁 | 約80% | 相続税・贈与税の算定 |
| **固定資産税評価額** | 市区町村 | 約70% | 固定資産税・不動産取得税の算定 |

重要なのは、固定資産税評価額は実勢価格の5〜7割程度に設定されているという点だ。横浜のように地価が上昇しているエリアでは、実勢価格との乖離がさらに大きくなることがある。公示価格は毎年更新されるが、固定資産税評価額は3年に1度の「評価替え」でしか変わらないため、上昇局面では評価額が実勢価格に追いつかないタイムラグが生じる。直近の評価替えは令和6年度（2024年度）に実施されており、次回は令和9年度（2027年度）だ。

### 固定資産税の仕組み——「毎年払う税金」の全貌

固定資産税は、毎年1月1日時点の所有者に対して課税される地方税（市町村税）だ。横浜市では固定資産税に加えて都市計画税も課税される。

**基本計算式:**

```
固定資産税額 = 課税標準額 × 1.4%（標準税率）
都市計画税額 = 課税標準額 × 0.3%（制限税率）
合計税率 = 1.7%
```

横浜市はいずれも標準税率・制限税率の上限で課税しており、減税している自治体もあるが横浜市は満額だ。

### 住宅用地の特例——マンション購入者にとっての最大の恩恵

住宅用地（住宅の敷地に供されている土地）には、課税標準額を大幅に引き下げる特例がある。これは総務省が定める制度で全国一律に適用される。

| 区分 | 面積要件 | 固定資産税の課税標準 | 都市計画税の課税標準 |
|-----|---------|-------------------|-------------------|
| **小規模住宅用地** | 200㎡以下の部分 | 評価額 × **1/6** | 評価額 × **1/3** |
| **一般住宅用地** | 200㎡超の部分 | 評価額 × **1/3** | 評価額 × **2/3** |

マンションの場合、敷地全体の面積を戸数で割った「持分面積」で判定される。例えば敷地面積3,000㎡のマンションが100戸なら、1戸あたり30㎡。200㎡以下なので小規模住宅用地に該当し、土地の課税標準額が評価額の1/6になる。マンションはほぼ例外なく小規模住宅用地の特例が適用されるため、土地分の税負担が極めて軽い。これは戸建てに比べたマンションの税制上の優位点だ。

### 新築住宅の固定資産税減額措置

新築住宅（床面積50㎡以上280㎡以下）の家屋部分について、一定期間、税額が1/2に減額される。

| 住宅タイプ | 減額期間 |
|-----------|---------|
| 一般住宅（戸建て等） | 3年間 |
| マンション（3階建以上の耐火・準耐火構造） | **5年間** |
| 認定長期優良住宅（一般） | 5年間 |
| 認定長期優良住宅（マンション） | **7年間** |

中古マンションを購入する場合、この減額措置の「残り期間」を引き継ぐことになる。築10年の中古マンションを買えば、減額期間はとっくに終了しているため恩恵はない。逆に、築3年以内の中古マンションなら残り2年分の減額を受けられる可能性がある。

### 家屋の経年減点補正率——RC造マンションの減価の実態

家屋の固定資産税評価額は「再建築価格 × 経年減点補正率」で算出される。RC造（鉄筋コンクリート造）マンションの経年減点補正率は以下のように推移する。

| 経過年数 | 経年減点補正率 | 新築時比 |
|---------|-------------|---------|
| 新築 | 1.0000 | 100% |
| 5年 | 約0.8714 | 約87% |
| 10年 | 約0.7397 | 約74% |
| 15年 | 約0.6225 | 約62% |
| 20年 | 約0.5054 | 約51% |
| 25年 | 約0.3884 | 約39% |
| 30年 | 約0.3059 | 約31% |
| 45年以上 | **0.2000**（下限） | **20%** |

木造の下限が0.2に到達するのが築25年程度であるのに対し、RC造は築45年まで下がり続ける。ただし下限は同じ0.2であり、RC造マンションは築45年以降、建物の固定資産税評価額が新築時の20%で底を打つ。つまり、築古マンションでも建物の固定資産税がゼロになることは決してない。

### 都市計画税——横浜市では避けられないもう一つの税

都市計画税は都市計画事業（道路整備、下水道整備、区画整理等）の財源として、市街化区域内の土地・家屋に課税される。横浜市は全域が市街化区域に含まれるため、横浜でマンションを買う限り都市計画税は必ず課税される。税率は0.3%で、固定資産税と合わせて1.7%が実効税率となる。

### 不動産取得税——購入時に一度だけ課税される都道府県税

不動産取得税は不動産の取得時に一度だけ課税される都道府県税だ。神奈川県の場合、取得後おおむね6ヶ月〜1年後に県税事務所から納税通知書が届く。購入直後ではなく「忘れた頃にやってくる」税金であり、資金計画に含めておかないと痛手になる。

**基本計算式:**

```
不動産取得税 = 固定資産税評価額 × 税率
```

**税率（2027年3月31日まで）:**
- 土地: 3%（本則4%から軽減中）
- 住宅用家屋: 3%（本則4%から軽減中）
- 非住宅用家屋: 4%

**宅地の課税標準特例:**
宅地の取得については、固定資産税評価額が1/2に軽減される（2027年3月31日まで）。

**中古住宅の建物の軽減措置:**

自己居住用の中古住宅（床面積50㎡以上240㎡以下）を取得した場合、新築年月日に応じた控除額が固定資産税評価額から差し引かれる。新耐震基準適合が条件だが、1982年1月1日以降に新築された住宅は適合とみなされる。

| 新築年月日 | 控除額 |
|-----------|-------|
| 1997年4月1日以降 | **1,200万円** |
| 1989年4月1日〜1997年3月31日 | 1,000万円 |
| 1985年7月1日〜1989年3月31日 | 450万円 |
| 1982年1月1日〜1985年6月30日 | 420万円 |

横浜で中古マンションを買う場合、1997年4月以降の新築物件なら控除額1,200万円が適用される。築25年程度のRC造マンション（70㎡）であれば、建物の固定資産税評価額が1,200万円を下回ることも多く、建物分の不動産取得税が実質ゼロになるケースが珍しくない。

**土地の軽減措置（住宅用土地）:**

以下のいずれか大きい方の額が税額から控除される:
- 45,000円
- 土地1㎡あたりの評価額 × 1/2 × 住宅の床面積の2倍（上限200㎡）× 3%

### 認定長期優良住宅の税制優遇

認定長期優良住宅は、固定資産税と不動産取得税の両方で追加的な優遇を受けられる。

- **固定資産税**: 新築マンションの減額期間が5年→**7年**に延長
- **不動産取得税**: 建物の控除額が1,200万円→**1,300万円**に増額
- **登録免許税**: 所有権保存0.15%→0.1%、所有権移転0.3%→0.1%（いずれも2026年3月末まで。延長の可能性あり）

ただし中古住宅を取得する場合、長期優良住宅の認定はあくまで新築時の認定がベースであり、認定を受けていない中古住宅を後から認定取得することは現実的には困難だ。購入前に認定の有無を確認する意味は、固定資産税減額の残存期間と、住宅ローン減税の借入限度額（長期優良は5,000万円）の判定にある。

### 購入前に固定資産税評価額を確認する方法

固定資産税評価額は税額を直接左右するため、購入前に把握しておくことが重要だ。しかし、評価額は原則として所有者にしか開示されない。購入検討者が確認する手段は以下の通り。

1. **売主に課税明細書の写しを依頼する**: 毎年4月に届く納税通知書に添付される課税明細書に評価額が記載されている。売主に提供を依頼するのが最も確実
2. **仲介業者を通じて確認**: 重要事項説明書に固定資産税・都市計画税の額が記載される。仲介業者が売主から取得するのが通常の流れ
3. **固定資産税路線価から推定**: 「全国地価マップ」（一般財団法人資産評価システム研究センター）で固定資産税路線価を閲覧可能。路線価 × 面積 × 補正率で概算可能
4. **固定資産評価証明書の取得**: 売買契約締結後、買主の委任を受けた司法書士が取得するのが一般的

### 横浜マンション税額シミュレーション

**物件条件:**
- 横浜市都筑区、築15年、RC造マンション、専有面積70㎡
- 物件価格（実勢価格）: 4,500万円
- 土地持分面積: 25㎡

**評価額の推定:**
- 土地の固定資産税評価額: 実勢価格の約70% → 土地全体から按分して約600万円
- 建物の固定資産税評価額: 新築時約1,400万円 × 経年減点補正率0.62 ≒ 約870万円

**固定資産税（年額）:**

```
【土地】
課税標準額 = 600万円 × 1/6（小規模住宅用地特例）= 100万円
固定資産税 = 100万円 × 1.4% = 14,000円
都市計画税 = 100万円 × 1/3 ÷ 1/6 × 0.3%
  → 特例適用後: 600万円 × 1/3 × 0.3% = 6,000円
  ※都市計画税の住宅用地特例は1/3（固定資産税の1/6より緩い）

【建物】
課税標準額 = 870万円（経年補正後、特例なし）
固定資産税 = 870万円 × 1.4% = 121,800円
都市計画税 = 870万円 × 0.3% = 26,100円

【年額合計】
固定資産税: 14,000 + 121,800 = 約135,800円
都市計画税: 6,000 + 26,100 = 約32,100円
合計: 約167,900円 → 年額約17万円（月額約14,000円）
```

**不動産取得税（一度だけ）:**

```
【建物】
固定資産税評価額 870万円 − 控除額 1,200万円 = マイナス → 税額0円

【土地】
税額 = 600万円 × 1/2 × 3% = 90,000円
軽減額 = 土地㎡単価(600万÷25) × 1/2 × 住宅面積の2倍(140㎡、上限200㎡) × 3%
       = 12万円 × 1/2 × 140 × 3% = 252,000円
→ 軽減額 > 税額 のため、土地分も税額0円

【不動産取得税合計: 0円】
```

このシミュレーションが示すのは、自己居住用の中古マンション（特に築15年程度以上）では、軽減措置のおかげで不動産取得税が実質ゼロになるケースが多いという事実だ。一方、固定資産税+都市計画税は毎年約17万円が恒久的にかかり続ける。月額にすると約14,000円で、管理費・修繕積立金（合計約3万円）と合わせると毎月約4.4万円のランニングコストになる。

### 新築 vs 中古の税負担比較

同じ4,500万円の物件でも、新築と築15年中古では税負担が大きく異なる。

| 項目 | 新築マンション | 築15年中古 |
|-----|-------------|-----------|
| 建物評価額 | 約1,400万円 | 約870万円 |
| 固定資産税+都市計画税（年額） | 約23万円 | 約17万円 |
| 新築減額（5年間） | 建物分1/2 → 約16万円に | なし |
| 不動産取得税 | 建物0円、土地0円 | 建物0円、土地0円 |

新築は減額期間中は税負担が軽いが、6年目以降に約1.5倍に跳ね上がる。中古は初年度から安定的。30年間の累計で見ると、新築の方が建物評価額の減少が大きいため長期的には中古より高い税額を払うことになる。ただし、新築は建物の品質や設備のメリットがあるため、税金だけで判断すべきではない。

## 気づき・洞察

### 固定資産税評価額は「購入判断の裏チャンネル」

固定資産税評価額は公的機関が独自に算定した「お墨付きの価格」だ。売主の言い値や仲介業者のポジショントークとは独立した客観的指標として使える。実勢価格が固定資産税評価額の1.4倍（=公示地価相当）を大きく超えるなら、市場が過熱している可能性がある。逆に1.2倍程度なら割安感がある。特に横浜のようにエリア間の価格差が大きい市場では、評価額との比率を比較することで、どのエリアが「割安」でどのエリアが「プレミアム過多」かを判別できる。

### 「忘れた頃に来る」不動産取得税のメンタルモデル

不動産取得税が購入後6ヶ月〜1年後に届くという仕組みは、行動経済学的に見て巧妙（あるいは厄介）だ。購入時の高揚感が薄れ、住宅ローン返済が日常化した頃に予期しない出費が来る。軽減措置で実質ゼロになるケースが多いとはいえ、申告しなければ軽減が適用されない自治体もあり（神奈川県は原則申告制）、「知っていれば払わなくて済んだ税金」が発生するリスクがある。

### 3年ごとの評価替えとタイミング戦略

固定資産税評価額は3年に1度しか見直されない。次回の評価替えは2027年度（令和9年度）。もし2026年中に購入すれば、2024年度の評価額（地価上昇前の水準）がベースとなった税額で少なくとも1年は過ごせる。逆に2027年以降の購入は、上昇後の評価額が適用される可能性が高い。微妙なタイミングだが、地価上昇局面では「評価替え前の購入」が税負担上有利になる場合がある。

## 他分野との接続

### cs（コンピュータサイエンス）: 評価アルゴリズムの類似性
固定資産税評価額の算定は、再建築価格方式という「コストアプローチ」を採用している。これはソフトウェアの見積もりにおけるCOCOMO（Constructive Cost Model）に類似する。部材ごとの標準単価を積み上げ、経年補正を掛けて現在価値を算出するボトムアップ手法だ。一方、実勢価格は市場の需給で決まるトップダウン的な価格。この二つのアプローチが常に乖離するのは、ソフトウェアの「開発コスト」と「市場価値」が一致しないのと全く同じ構造だ。

### golf: コースレーティングと固定資産税評価の共通構造
ゴルフのコースレーティングは、距離・障害物・地形など複数のファクターを標準化されたアルゴリズムで数値化する。固定資産税評価も同様に、構造・用途・築年数・立地などを要素分解し、補正率を掛けて最終的な数値を導き出す。どちらも「主観を排除した定量評価」を目指しているが、実際のプレー体感（＝住み心地）や実勢価格（＝市場の評価）とはしばしば乖離する。評価基準の定期的見直し（評価替え3年/コースレーティング定期更新）も共通している。

### tech: データ駆動型不動産バリュエーション
PropTech分野では、機械学習による不動産自動査定（AVM: Automated Valuation Model）が進化している。Zillowの「Zestimate」やSONYグループの「おうちダイレクト」が代表例だ。固定資産税評価額の「再建築価格方式」が古典的なルールベース評価だとすれば、AVMは膨大な取引データからパターンを学習する統計的アプローチ。日本では、固定資産税評価額・路線価・成約価格・賃料データを特徴量としてXGBoostやLightGBMで査定するモデルが研究されている。

## 次に深掘りしたいこと

- **確定申告と住宅関連控除の連携**: 住宅ローン減税・固定資産税・不動産取得税の軽減措置を漏れなく適用するための確定申告チェックリスト
- **マンションの「タワマン節税」封じ**: 2024年1月以降のタワーマンション固定資産税の階層別補正の仕組みと影響
- **固定資産税の過徴収問題**: 全国で頻発する評価ミス・過大徴収の実態と、課税明細書の自己チェック方法
- **償却資産税**: マンション共用部の設備（エレベーター、受水槽等）に課税される償却資産税の管理組合での取り扱い
- **相続時の不動産評価**: 固定資産税評価額と相続税路線価の関係、マンションの相続税評価額見直し（2024年以降の通達改正）
