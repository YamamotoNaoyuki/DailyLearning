# 賃貸 vs 持ち家のNPV分析と機会費用 — 投資理論で住まいを考える
**日付**: 2026-05-08
**分野**: house-hunting
**タグ**: #NPV #IRR #opportunity-cost #rent-vs-own #real-estate-finance #DCF

## 学んだこと

### 「総コスト比較」の罠

巷の比較記事は「**50年で持ち家1.2億円 vs 賃貸1.4億円、持ち家の勝ち**」のような単純な総額比較が多い。これには重大な欠陥が3つある：

1. **時間価値の無視**: 30年後の100万円を今の100万円と同列に扱っている
2. **機会費用の無視**: 頭金1000万円を「投資せず家を買う」場合と「賃貸で頭金を運用する」場合の差
3. **リスク調整の無視**: 不動産は売却時の流動性リスク、修繕の不確実性、災害リスクを伴う

正しい比較は**NPV (Net Present Value, 正味現在価値)**または**IRR (Internal Rate of Return, 内部収益率)**で行う。

### NPV分析の枠組み

NPVは「**将来のキャッシュフローを現在価値に割引いた合計**」。割引率 r で：

```
NPV = Σ CFₜ / (1+r)ᵗ
```

持ち家のキャッシュフロー（年間）：
- **流出（−）**: ローン返済、固定資産税、都市計画税、修繕積立金、管理費、火災保険、リフォーム費用
- **流入（+）**: 帰属家賃（自分で住むことで支払わずに済む賃料）、最終売却価格

賃貸のキャッシュフロー（年間）：
- **流出（−）**: 家賃、更新料、火災保険
- **流入（+）**: 頭金相当額の運用益（持ち家を買わずに投資した場合）

「**頭金を投資した場合の運用益**」が**機会費用**で、ここを無視すると賃貸が不当に不利になる。

### 機会費用の具体例

**シナリオ**: 横浜・5000万円のマンション。頭金1000万円、ローン4000万円、35年固定2.0%

**持ち家を買う場合**:
- 頭金1000万円は不動産に固定（流動性ゼロ）
- 月返済 約13万2千円、35年で総支払額 約5550万円（うち利息1550万円）
- 35年後の物件残存価値（仮に新築価格の40% = 2000万円）
- + 維持費（修繕積立金、管理費、固定資産税）月3万円相当 = 35年で1260万円

**賃貸の場合**:
- 同等物件の家賃 月15万円（家賃 ≒ 物件価格×0.3%/月 が横浜の目安）
- 35年で総家賃 6300万円（更新料・値上げ込みでもう少し）
- ただし**頭金1000万円を株式indexで5%/年運用**すると35年後は **約5500万円**（複利）

**割引率5%でのNPV比較**:
- 持ち家: 流出の現在価値 ≒ 4500万円、流入の現在価値（売却価格2000万円→現在価値340万円）
- 賃貸: 流出の現在価値 ≒ 4900万円、流入の現在価値（運用益5500万円→現在価値990万円）

総合すると、5%運用前提では**賃貸が約100-200万円有利**。3%運用なら持ち家有利。**運用利回り（機会費用）の前提で結論が逆転**するのが核心。

### IRR比較（投資判断としての家）

三菱UFJ不動産販売の論考では、**「同等物件の賃貸 + 頭金をJ-REITに投資」**のIRRと「**マンション購入**」のIRRを比較する手法を提案している：

- 持ち家のIRR = 「自分が払うべき家賃を払わずに済む」効果と「物件価値の変動」の総合内部収益率
- 賃貸+投資のIRR = J-REIT等の運用利回り

横浜の中古マンション（築20年、駅10分）の典型値：
- 持ち家IRR: 2-4%（物件によって大きく変動、立地が悪いと-1%もあり得る）
- J-REIT平均IRR: 4-6%（2026年現在）

**IRRの観点では、立地・物件によっては賃貸+投資が合理的**。ただしIRRは確率分布の中央値で、テールリスク（暴落、災害）の評価は別途必要。

### 持ち家特有のリスク要因（NPVに反映すべき）

NPV計算に組み込むべきが見落とされがちな項目：

**正のリスク（持ち家有利）**:
- インフレーション・ヘッジ: 不動産は実物資産なのでインフレで名目価値上昇
- 心理的安定: 強制貯蓄効果、引越しのコスト・心理負担
- 老後の住居安定: 高齢者の賃貸契約困難（生命保険系の家賃保証契約も2026年で限定的）

**負のリスク（賃貸有利）**:
- 流動性リスク: 売却に半年以上、3-5%の手数料
- 災害リスク: 地震保険でカバーできない部分（最大50%まで）
- 修繕積立金不足リスク: 古いマンションの値上げラッシュ（過去エントリ）
- 流動性プレミアム: 株式と比較して約2-3%の超過リターンが必要との理論値（Stulz 2005）
- 集中リスク: 個人資産の70-90%が一物件に集中

### 帰属家賃（Imputed Rent）の正しい扱い

経済学的には、持ち家所有者は**「自分から自分に家賃を支払っている」**。この帰属家賃は所得税法上は非課税だが、**経済的には実在する所得**である。

NPV計算では：
- 持ち家側の流入 = 帰属家賃（同等賃貸の市場家賃）
- これがないと「同等の住居サービス」を比較できない

日本の税制では帰属家賃が非課税であることが、**持ち家を相対的に有利にしている**（米国等では mortgage interest deduction で持ち家を優遇している国もあるが、日本は住宅ローン控除のみ）。

### 横浜エリアの実勢分析（2026年5月時点）

- **新築マンション平均価格**: 横浜駅周辺7000万円、武蔵小杉8000万円超
- **中古マンション築20年**: 横浜駅10分3500-4500万円
- **賃貸相場**: 同等物件で月14-18万円
- **賃料倍率（年家賃 / 物件価格）**: 4-5%（東京都心は3%台で持ち家のIRRが低くなりがち、横浜は若干賃貸有利な水準）
- **2026年の金利上昇局面**（過去エントリ参照）でローン金利が変動0.7%→1.2%へ。これが持ち家側のキャッシュフローを圧迫

**結論的に**: 横浜は東京都心と比較して**賃料倍率が高い** = 持ち家のIRRが相対的に良い。ただし駅徒歩10分超、築古、再開発予定なし、のような物件は**マンション建替え制度の対象になる前に売り抜く**戦略のほうがNPV的には優位。

## 気づき・洞察

「**家を買うのは投資ではなく消費**」という考え方が経済学的に正しい場合が多い。**家は通常、減価する消費財**（建物部分は減価償却、土地は地価次第）であり、投資としては流動性・リスク調整後リターンで他の資産（株式、債券、REIT）に劣る。

**例外**: 都心一等地、再開発エリア、極端に割安な中古物件、または「**自己利用 + ライフスタイル価値**」を含めた効用関数で評価する場合。

「**機会費用**」は経済学的には自明だが、住宅選択では**心理的に過小評価されがち**。なぜか：
- 「家賃は捨て金」という日本の文化的バイアス
- 頭金を「使わない」と運用されない（行動経済学的な mental accounting）
- 株式投資の心理的ハードルが高い

**正しい賃貸戦略**: 賃貸を選ぶなら、**頭金相当額を必ず運用する**。さもなくば「最悪の選択」（消費に流用）になる。

「**50年で1億円超**」という日経新聞の見出しは**インフレ調整後**で見るべき。年2%インフレ前提なら50年後の1億円は**現在価値で3700万円程度**。額面の数字に惑わされない。

NPV分析の結論は「**運用利回り前提に強く依存**」する。これは**「家を買うべきかは、株を買うべきかと表裏一体**」を意味する。**「持ち家 vs 賃貸」ではなく「持ち家 vs 賃貸+運用」が正しい比較軸**。

## 他分野との接続

- **CS (GC pause time target = 連続パラメータ)**: NPV計算で割引率を変えると結論が変わるのは、GCのpause time target を変えると最適アルゴリズムが変わるのと同型。**前提値依存の最適化**
- **golf (DECADE Strategic思考)**: 「**安全策（賃貸）も選択肢、ただし機会費用がかかる**」というStrategic設計の発想と、住宅選択は完全に同型。リスクとリワードの選択
- **bread (生態系の動的平衡)**: 持ち家の「ローン返済 + 修繕費 + 固定資産税 + 管理費」のキャッシュフロー構造は、サワードウのLAB:酵母比のように**多変数の動的平衡**で最適点が決まる
- **anatomy (横隔膜の二重機能)**: 家は「**居住の場 + 資産**」の二重機能。横隔膜の「呼吸 + 姿勢」と同じく、**両機能の最適化が競合する**ことがある（投資効率を最大化するなら通勤に不便な郊外、居住快適性なら高い都心）
