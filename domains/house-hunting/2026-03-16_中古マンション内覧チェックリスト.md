# 中古マンション内覧チェックリスト——見落としやすい重要ポイント
**日付**: 2026-03-16
**分野**: 家探し
**タグ**: #内覧 #チェックリスト #中古マンション #デューデリジェンス #見落とし防止

## 学んだこと

### 1. 内覧前——事前に入手すべき書類（書面の戦い）

内覧は当日始まるのではなく、**書類の事前入手**から始まる。不動産仲介業者に以下の書類を事前請求すること：

| 書類 | 確認すべきポイント |
|------|-------------------|
| **重要事項説明書** | 権利関係、用途地域、建築制限、瑕疵情報 |
| **管理規約** | ペット可否、リフォーム制限、民泊禁止条項、楽器演奏ルール |
| **修繕積立金の推移** | 段階増額方式か均等積立か。直近5年の値上げ履歴 |
| **長期修繕計画書** | 次回大規模修繕の時期と予算。積立金残高との乖離 |
| **総会議事録（過去3年分）** | 滞納問題、住民トラブル、管理会社変更、建替え議論の有無 |
| **修繕履歴** | 過去の大規模修繕の実施内容と費用 |

**最重要チェック**: 修繕積立金が国交省ガイドライン（200〜300円/平米/月）を大幅に下回っていないか。著しく安い場合は将来の大幅値上げまたは一時金徴収のリスクが高い。

### 2. 建物外観チェック——構造の健康診断

#### クラック（ひび割れ）の判定基準
- **0.3mm未満**: ヘアークラック。乾燥収縮による生理的なもの。経過観察で可
- **0.3mm以上**: **構造クラックの可能性**。毛細管現象で水が浸入し、鉄筋腐食→爆裂→耐力低下の連鎖が始まる
- **1.0mm以上**: 深刻な構造問題。建物全体の安全性に関わる可能性

#### 白華現象（エフロレッセンス）
コンクリート内部のアルカリ成分（水酸化カルシウム）が水分とともに表面に析出する現象。
- **局所的な白華**: クラック周辺のみなら軽度
- **広範囲の白華**: コンクリート中性化が進行している深刻なサイン
- **茶褐色の滲み**: 鉄筋が腐食している可能性が高く、最も危険

#### その他の外壁チェック
- 防水塗膜の浮き・剥離（特にバルコニー下面）
- 排水溝・ドレンの詰まり・水溜まり跡
- タイルの浮き（叩くと空洞音がする箇所）
- 基礎部分の水染み跡

---

### 3. 共用部——管理の質を測る「リトマス試験紙」

共用部の状態は、管理組合と住民の質を映す鏡。以下の項目を観察する：

| チェック箇所 | 良い兆候 | 悪い兆候 |
|-------------|---------|---------|
| **廊下・階段** | 清掃が行き届き照明が点灯 | ゴミ放置、電球切れ |
| **エレベーター** | 検査済証掲示、内部清潔 | 落書き、異臭 |
| **ゴミ置場** | 分別ルール掲示、清潔 | 分別不良、悪臭、不法投棄 |
| **駐輪場** | 整理整頓、番号札あり | 放置自転車山積み |
| **掲示板** | 管理組合のお知らせが最新 | 古い掲示物が放置 |
| **郵便受け** | 全戸きちんと施錠 | チラシ溢れ、壊れたボックス |
| **植栽** | 手入れされている | 雑草繁茂、枯れた植木 |

**プロのコツ**: エレベーター内の掲示物と駐輪場を見れば、そのマンションの管理レベルの8割がわかる。

---

### 4. 室内チェック——表面の下を読む

#### 床の水平（傾き）チェック
- **ビー玉テスト**: 床にビー玉を置いて転がるか確認。6/1000（1mで6mm）以上の傾きは構造的問題の可能性
- **水準器アプリ**: スマートフォンの水準器で複数箇所を計測。特に部屋の対角線上
- 建具（ドア・引き戸）の開閉具合も傾きの指標。複数箇所で不具合があれば建物全体の歪み

#### 壁・天井の異常
- **クラック**: 窓枠の四隅から斜めに走るクラック（構造的応力のサイン）
- **水染み**: 天井の茶色いシミ（上階からの漏水履歴）
- **壁紙の浮き**: 結露やカビを壁紙で隠蔽しているケースあり
- **窓周辺の結露跡**: 黒カビの痕跡は断熱性能不足のサイン

#### 建具の動作確認
- 全てのドア・引き戸の開閉
- サッシの開閉とクレセント錠の動作
- 網戸の状態

---

### 5. 水圧・排水チェック——最も見落とされるポイント

**必ず実施すべきテスト**:

1. **同時通水テスト**: キッチン・洗面・浴室の全蛇口を同時に全開にする。高層階ほど水圧が低下しやすい
2. **トイレの流れ**: 水を流して勢いと補給速度を確認
3. **排水速度**: 洗面台・浴室に水を溜めて一気に流す。ゴボゴボ音や逆流は配管の詰まりサイン
4. **給湯器の応答**: お湯が出るまでの時間と温度安定性
5. **配管材質の確認**: 2000年以前の物件は金属管（鉄管・銅管）の可能性。腐食リスクが高い。架橋ポリエチレン管・ポリブテン管なら安心

**築年数と配管の関係**:
- 築20年以上: 給水管の劣化が本格化する時期
- 築30年以上: 排水管も含めた全面交換が視野に入る
- 床スラブ貫通配管: リノベ時の配管移設が困難（排水勾配の制約）

---

### 6. 遮音性チェック——住んでからの後悔No.1

#### 床構造の違い
| 項目 | 直床（じかゆか） | 二重床（にじゅうゆか） |
|------|-----------------|---------------------|
| 構造 | スラブに直接フローリング貼り | スラブ上に支柱＋パネル＋仕上材 |
| 遮音性（重量衝撃音） | スラブ厚次第 | 低音域でやや不利な場合も |
| 遮音性（軽量衝撃音） | クッション材で対応 | 空気層が緩衝 |
| リノベ自由度 | 配管移設に制約 | 空間内で配管ルート変更可 |
| コスト | 安い | 高い |

#### 遮音性の確認方法
- **壁ノック**: コンクリート壁はゴツゴツと硬い音。石膏ボード間仕切りはコンコンと軽い音
- **スラブ厚の確認**: 200mm以上が望ましい。150mm以下は重量衝撃音が懸念
- **戸境壁の厚さ**: 180mm以上のコンクリート壁が理想。乾式壁（石膏ボード＋グラスウール）はやや劣る
- **窓を開けた状態と閉めた状態**: 外部騒音の遮断度合いを比較
- **静かに耳を澄ます**: 上階の足音、隣室のテレビ音、外部の交通騒音
- **内覧時間の工夫**: 平日夜や休日など生活音が多い時間帯に再訪

---

### 7. 日照・採光チェック

- **内覧時の実際の日差し**: 時間帯により印象が大きく変わる。可能なら午前と午後の両方を確認
- **周辺建物の影響**: 南側の建物の高さと距離。冬至の太陽高度（東京で約31度）で最悪ケースを想定
- **将来の建設計画**: 南側に空き地や駐車場があれば建設リスクあり。都市計画図・用途地域を確認
- **バルコニーの奥行き**: 深すぎると室内に日が入りにくい（2m以上は要注意）
- **天井高**: 2,400mm以上で開放感。2,500mm以上は贅沢

---

### 8. 臭いチェック——五感の中で最も正直なセンサー

#### 空室特有の問題：封水切れ（Pトラップの乾燥）
排水トラップ内の水（封水）が蒸発すると、下水管からの臭気が室内に逆流する。空室期間が3ヶ月以上の物件では高確率で発生。

**確認方法**:
1. 玄関ドアを開けた瞬間の第一印象（嗅覚は最初の数秒が最も鋭敏）
2. 各排水口に鼻を近づける（キッチン、洗面、浴室、洗濯パン）
3. トイレの水面が下がっていないか確認
4. 換気扇を回して臭気の変化を確認

**臭いの種類と原因**:
| 臭い | 原因 | 深刻度 |
|------|------|--------|
| 下水臭 | 排水トラップ封水切れ | 低（水を流せば解決） |
| カビ臭 | 壁内部・天井裏の結露カビ | **高**（構造的原因） |
| タバコ臭 | 前居住者の喫煙 | 中（壁紙交換でも残る場合あり） |
| ペット臭 | 前居住者のペット | 中（床下地まで染込みの場合あり） |
| 接着剤臭 | リフォーム直後のVOC | 低（換気で軽減、シックハウスに注意） |

### 9. 収納チェック

- **クローゼット奥行き**: ハンガー収納に最低60cm必要（標準的なスーツの肩幅+余裕）
- **収納率**: 専有面積の**8〜10%**が理想。70平米なら5.6〜7.0平米の収納面積
- **玄関収納**: シューズボックスの容量。家族4人なら最低40足分
- **パントリー**: キッチン周辺の食品庫。あれば大きなプラス
- **収納の配置**: 使う場所の近くにあるか（動線効率）
- **天袋・枕棚**: 高い位置の収納は日用品には不向きだが季節物には有効

### 10. リノベーション可能性の見極め

#### 壊せる壁と壊せない壁
| 判定方法 | 間仕切り壁（撤去可能） | 躯体壁（撤去不可） |
|---------|----------------------|-------------------|
| **叩いた音** | コンコン（軽い・空洞感） | ゴツゴツ（硬い・詰まった感じ） |
| **壁厚** | 70〜90mm程度 | 150mm以上 |
| **図面表記** | 二重線 | 塗りつぶしまたは太線 |
| **構造** | 石膏ボード＋軽量鉄骨 | 鉄筋コンクリート |

#### ラーメン構造 vs 壁式構造
- **ラーメン構造**: 柱と梁で支える。室内に柱型が出るが**間取り変更の自由度が高い**
- **壁式構造**: 壁で支える。柱型がなくスッキリだが**壁を撤去できない制約**。主に5階建て以下

#### 配管ルートの確認
- **パイプスペース（PS）の位置**: 図面で「PS」表記。移動不可
- **床スラブ上配管 vs 床スラブ貫通配管**: 上配管なら水回り移設が比較的容易
- **二重床なら配管ルート変更の余地あり**: 空間内でルーティング可能
- **電気容量**: 分電盤のアンペア数確認。30Aは現代では不足。60A以上が望ましい

### 11. 周辺環境チェック——物件の外にある価値

#### 生活利便施設の距離
| 施設 | 理想的な距離 |
|------|------------|
| スーパー | 徒歩5分以内 |
| コンビニ | 徒歩3分以内 |
| 病院・クリニック | 徒歩10分以内 |
| 小学校 | 徒歩10分以内 |
| 公園 | 徒歩5分以内 |
| 最寄駅 | 徒歩10分以内 |

#### ハザードマップの確認
- **洪水浸水想定区域図**: 国土交通省ハザードマップポータルサイト
- **土砂災害警戒区域**: 横浜は丘陵地が多く、がけ崩れリスクあり
- **液状化リスク**: 埋立地・河川流域低地は要注意（みなとみらい、鶴見川流域等）
- **津波浸水想定**: 臨海部は確認必須

#### 時間帯を変えた再訪
- **夜間**: 街灯の有無、人通り、飲食店の騒音
- **朝**: 通勤ルートの混雑、日当たり
- **雨天**: 敷地内の排水状況、周辺道路の冠水

### 12. 売主・仲介業者への質問リスト

必ず聞くべき質問：

1. **売却理由**: 「住み替え」は良い兆候。「近隣トラブル」「瑕疵」が隠されていないか
2. **居住年数**: 短期間での売却は何か問題がある可能性
3. **過去の修繕・リフォーム履歴**: 水回りの交換時期、配管の更新有無
4. **近隣トラブルの有無**: 騒音、ペット、ゴミ問題
5. **管理費・修繕積立金の値上げ予定**: 直近の総会での議論
6. **空室率**: マンション全体の空室率が高いと管理費滞納リスク増
7. **駐車場の空き状況と料金**: 機械式駐車場は維持費が高額
8. **インターネット環境**: 光回線の導入状況、マンション全体の回線速度
9. **固定資産税・都市計画税の実額**: 年間の税負担を具体的に

## 気づき・洞察

### 「見える部分」と「見えない部分」の非対称性
内覧で最も危険なのは、リフォーム済みの綺麗な表面に騙されること。新しい壁紙の下にカビが、美しいフローリングの下に傾いた床がある可能性を常に意識すべき。**表面的な美しさと構造的な健全性は全く別物**。

### 内覧は「減点法」で臨む
気に入った物件ほど良い点ばかりに目が行く（確証バイアス）。意識的に「問題を探す」モードで臨み、チェックリストを機械的に消化することが重要。感情と分析を分離する。

### 五感の全てを使う
視覚（クラック、水染み）、聴覚（遮音性、外部騒音）、嗅覚（カビ、下水）、触覚（壁の叩き音、床の感触）——味覚以外の全感覚を動員する「全身検査」が内覧の本質。

## 他分野との接続

### 解剖学との接続——「見えない構造体」の重要性
人体の**筋膜（fascia）**がそうであるように、マンションにも目に見えない構造体がある。配管は「血管」、電気配線は「神経」、躯体は「骨格」、防水層は「皮膚」。表面のリフォーム（化粧）に惑わされず、インフラの健全性を見る視点は、解剖学で表層筋の下にある深層筋や筋膜を意識するのと同じ。

### テクノロジーとの接続——オブザーバビリティの思想
ソフトウェアの**OpenTelemetry**が「メトリクス・ログ・トレース」の三本柱でシステムの内部状態を可視化するように、内覧チェックリストは建物の「可観測性（Observability）」を高めるツール。書類（ログ）、外観・共用部（メトリクス）、室内の詳細検査（トレース）の三層で建物の健康状態を診断する。

### 耐震知識との接続——構造の読み方
前回学んだ耐震基準の知識が直接活きる。クラック0.3mmの判定基準、ラーメン構造と壁式構造の違い、Is値の概念——これらは内覧時の「構造を読む目」を支える基礎知識。耐震の「隠れた構造」を知っていることが、内覧の質を根本的に変える。

### 哲学との接続——「いき」の構造と空間の質
九鬼周造の「いきの構造」が「媚態・意気地・諦め」の三契機を分析したように、住空間の質も「機能性・美観・制約の受容」の三層で構成される。完璧な物件は存在せず、何を受け入れ何を譲れないかを自覚することが、住まい選びの「いき」である。
