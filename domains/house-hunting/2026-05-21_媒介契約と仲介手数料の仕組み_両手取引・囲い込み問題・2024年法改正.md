# 媒介契約と仲介手数料の仕組み — 両手取引・囲い込み問題・2024年規制動向
**日付**: 2026-05-21
**分野**: 家探し
**タグ**: #媒介契約 #仲介手数料 #両手仲介 #囲い込み #REINS #宅建業法 #情報の非対称性 #エージェント問題

## 学んだこと

[[媒介契約3類型（2026-04-26）]] で制度の骨格は押さえたので、今日はその制度が**生み出すインセンティブ構造**——なぜ買主が損をする方向に力が働くのか、そしてそれをどう自衛するか——を経済学的に掘り下げる。これは買主にとって、物件選びそのものよりも実損に直結するテーマである。

### 1. 媒介契約3類型を「監視の強度」で読み直す

売主が仲介業者と結ぶ契約（標準媒介契約約款＝国交省告示583号）は3種類だが、本質は「**売主が業者をどれだけ縛り、業者にどれだけ義務を課すか**」の強度差である。

| | 一般媒介 | 専任媒介 | 専属専任媒介 |
|---|---|---|---|
| 複数社依頼 | 可 | 不可（1社） | 不可（1社） |
| 自己発見取引 | 可 | 可 | **不可**（業者を通す） |
| REINS登録義務 | なし | 7営業日以内 | 5営業日以内 |
| 業務報告義務 | なし | 2週に1回以上 | **1週に1回以上** |
| 契約期間上限 | （法定なし、行政指導3か月） | 3か月 | 3か月 |

ここに**非対称な罠**がある。売主から見ると専属専任は「業者が一番頑張ってくれる契約」に見える（報告が手厚く、登録も早い）。しかし業者から見ると専属専任・専任こそが**「この物件を独占でき、両手を狙える」契約**である。一般媒介は他社と競合するため両手化が難しい。つまり**業者が最も推奨してくる契約（専任系）が、囲い込みの温床になる契約**でもある——この利害のねじれが今日の出発点。

### 2. 仲介手数料の上限と、その「分母」の問題

宅建業法46条＋報酬告示による上限（速算式）：
- **取引額×3% + 6万円 + 消費税**（400万円超の部分）
- 正確には3段階の積み上げ（200万円以下5%、200〜400万円4%、400万円超3%）を一本化した近似式
- **2024年7月改正**：「低廉な空家等の媒介特例」を400万円以下→**800万円以下**に拡大。売主・買主それぞれから**最大33万円（30万円＋消費税）**受領可。6年ぶりの報酬規定改正で、地方・空き家流通を阻んでいた「手数料が安すぎて業者が動かない」問題への対応

ここで効いてくるのが**「片手か両手か」で業者の取り分が2倍になる**という事実。5,000万円の物件なら片手で約171万円、両手なら売主＋買主で**約342万円**。同じ1件の労力で報酬が倍——これが囲い込みの経済的駆動力そのものである。

### 3. 両手仲介はなぜ「合法」なのか — 準委任という抜け穴

直感的には「同じ業者が売主と買主の両方を代理するのは利益相反では？」と思える。民法108条は**双方代理を原則禁止**している（一方に有利な条件は他方に不利、という構造的矛盾のため。弁護士は弁護士法25条でも禁止）。にもかかわらず日本で両手仲介が合法なのは、**媒介の法的性質が「代理」ではなく「準委任」だから**である（不動産流通推進センターの整理）。

- 媒介業者は本人に代わって**意思表示（売る・買うの判断）を行わない**。契約の成立に向けて尽力する「双方の取次ぎ」にすぎない
- これは**法律行為ではなく事実行為** → 民法656条（準委任）が適用され、644条・645条の善管注意義務・報告義務を負うが、108条の代理規定は適用されない

つまり「代理人ではないから双方代理禁止に当たらない」という論理。**法形式上は綺麗だが、経済実態としては利益相反が温存される**——この形式と実態の乖離が、日本の不動産取引の構造的歪みの核心。米国では多くの州が Dual Agency を禁止・厳格規制し、buyer's agent と seller's agent が完全分離するのと対照的。

### 4. 囲い込みの構造 — 不正のメカニズム

**囲い込み**＝専任を受けた業者が、両手（＝報酬2倍）を実現するために、**他社からの買主紹介・物件確認を意図的に妨害**し、自社の抱える買主に売る不正。具体的手口：

1. 他社からの問い合わせに「**申込みが入っています**」「商談中です」と虚偽を伝えて遮断
2. 「**鍵がない**」「売主の都合がつかない」と内見を物理的に断る
3. REINSに**登録しない**、または「申込あり」等の**虚偽ステータス**を登録する

被害は売主が直接受ける（買主候補が絞られ、本来より安く・遅く売れる）。だが買主にも影響する——市場に出ているはずの物件が「見えない」状態になり、選択肢が人為的に狭められる。

### 5. 2024-2025 規制動向 — REINSステータス管理の義務化

長年「業界の悪しき慣習」だった囲い込みに、初めて**法的根拠のある是正手段**が入った。

- **2024年6月公布 / 2025年1月施行**の宅建業法施行規則改正。REINSの大規模改修に合わせ、専任・専属専任媒介物件の**「取引状況の登録（ステータス管理機能）」を義務化**
- ステータス3区分：**①公開中 ②書面による購入申込みあり ③売主都合で一時紹介停止中**
- 物件の実際の販売状況とREINS登録ステータスに相違がある場合（＝故意の虚偽登録・情報非開示）、**宅建業法65条1項の指示処分の対象**であることが明確化された。これが囲い込みに対する**初の明示的な法的根拠**。悪質・反復なら業務停止、免許取消も
- 売主には登録時に**登録証明書（ID/PW/2次元コード）**が交付され、売主専用画面で自分の物件のステータスをリアルタイム確認できる

ただし**専門家の評価は「前進だが限定的」**（さくら事務所等）。残る抜け穴：
- ステータスを「公開中」にしたまま、内見依頼を「鍵がない」「担当者不在」で**物理的に断る**のはREINS上は違反にならない
- 申込みが無いのに「**書面による購入申込みあり**」と虚偽登録すれば他社を遮断できる。登録内容が事実かを**客観的に検証する手段がない**のが実情

## 気づき・洞察

### これは典型的なプリンシパル・エージェント問題である

経済学のフレームで見ると驚くほど綺麗に当てはまる。**プリンシパル（依頼人＝売主／買主）がエージェント（仲介業者）に行動を委任するが、エージェントの行動を完全には監視できず、両者の利害が一致しない**ときに発生する非効率——まさにこれ。

- **情報の非対称性**：売主は「業者が本当にREINSに登録したか」「他社の内見を断っていないか」「広告を十分打ったか」を完全には観測できない。業者は圧倒的に多くの情報を握る
- **隠れた行動（moral hazard）**：囲い込みは「サボり」ではなく逆に「自社買主を見つける努力」だが、依頼人の利益に反する方向への努力という点で隠れた行動の一種
- **報酬構造がインセンティブを歪める**：両手で報酬2倍 → エージェントの最適行動が「早く高く売る（売主の利益）」ではなく「自社買主に売る（業者の利益）」にずれる

2025年の規制は、この問題への古典的処方箋である**「モニタリングの強化」**（REINSステータスの可視化＝売主が観測できる情報を増やす）に相当する。だが報酬構造（両手容認）という根本のインセンティブには手を付けていないため、効果が限定的なのは理論的にも予想どおり。**根治には「報酬構造の変更（両手禁止）」か「エージェントの分離（buyer's agent制度）」が必要**——というのが経済学の含意。

### 日本市場最大の特殊性 — バイヤーズエージェントの不在

米国・シンガポールでは、買主には**100%買主だけの味方をする buyer's agent** がつき、その報酬は慣行上**売主が支払う**（成約時に売主側手数料から分配）。買主は実質無料で「自分の代理人」を得られる。

日本にはこれが構造的に存在しない。理由は連鎖している：
1. 両手仲介が合法 → 1社が売主・買主双方から手数料を取れる
2. 業者の最大の収益機会が両手 → 「片手で買主だけの味方」をするビジネスが育ちにくい
3. REINSは業者専用で一般人がアクセスできない → 買主は「業者を通さないと物件全体を見られない」
4. 結果、買主の前に現れる業者は**たいてい売主側（または両手狙い）**であり、純粋な買主代理ではない

つまり買主は、**自分の「敵」になりうる相手から物件を紹介され、その相手に手数料を払う**という、よく考えると奇妙な立場に置かれている。「正直不動産」が題材にする業界の闇はここに根がある。

### 買主の自衛策（エンジニア的に整理すると「監視と検証」）

情報の非対称性に対抗する手段は、結局「**観測可能性を上げる**」ことに尽きる：

1. **物件確認テスト**：気になる物件を、紹介してきた業者**とは別の業者**にも問い合わせさせ、「内見できます」と返ってくるか確認する。別業者だと「申込みあり」と断られるなら囲い込みの強いシグナル
2. **複数チャネルで露出を確認**：SUUMO/at home等のポータルに長期間出ているのにREINSステータスや業者対応が不自然 → 警戒
3. **両手物件への警戒**：紹介業者が「売主側でもある」場合、価格交渉で買主の味方になりにくい。指値（値引き交渉）の通りやすさは片手物件の方が一般に良い
4. **手数料の妥当性検証**：上限が法定なので「3%+6万円」を超える請求は違法。値引き交渉の余地もある（特に高額物件）
5. **重要事項説明での確認**：囲い込みリスクは [[重要事項説明書（35条書面）の読み方と告知義務 (2026-05-09)]] とセットで——情報を出す側のインセンティブを常に疑う姿勢が共通の防御原理

## 他分野との接続

- **[[CS]]／分散システム — 真実性の検証できない台帳**：REINSステータス義務化は「ログを取れ」という規制だが、**ログの内容が事実かを暗号学的・客観的に検証する仕組みがない**。これは [[不動産登記簿（登記事項証明書）の読み方と相続登記義務化2024 (2026-05-12)]] で見た「対抗力はあるが公信力はない弱い台帳」と同型で、Sigstore/Rekorの透明性ログのような**改ざん検知性が欠落した monitoring**。観測可能性を上げても、観測値の真正性を保証しなければ moral hazard は残る、という設計上の教訓
- **[[tech]]／メカニズムデザイン**：囲い込み問題は、相続登記義務化で見た「**個々の合理性が社会的非効率を生む**」構造の再演。2025年規制＝モニタリング強化は処方の半分にすぎず、報酬構造（インセンティブ）に触れないと均衡は動かない。これはMoEの負荷分散やトークン経済設計と同じ「**インセンティブを変えずにルールだけ足しても行動は変わらない**」原理
- **[[賃貸 vs 持ち家のNPV分析と機会費用 (2026-05-08)]]**：仲介手数料（売買代金の約3%）は売却時にも発生し、持ち家の流動性コスト・取引コストの主要項目。NPV計算で「往復6%の取引摩擦」として効いてくる
- **音楽／演奏家とマネジメント**：演奏家とエージェント、レーベルとの関係も典型的なプリンシパル・エージェント問題（演奏家の利益とマネジメントの手数料最大化のずれ）。利益相反を「専属契約」で縛る点も媒介契約と相似

## 次に深掘りしたいこと

- **2025年規制の実効性データ**：施行から1年強が経過し、実際に65条1項の指示処分が出た事例はあるか。処分件数・内容を国交省の処分情報で追跡したい
- **buyer's agent型サービスの実態**：日本でも「片手・買主専属」を謳う不動産エージェント（らくだ不動産等）が増えているが、報酬を誰が払うか・本当に中立かを検証したい。両手構造の中で片手を貫くビジネスモデルが成立するのか
- **米国の Dual Agency 規制の最新動向**：2024年のNAR（全米リアルター協会）の集団訴訟和解で買主側手数料の慣行が大きく変わったと報じられている。日米の制度比較で日本の将来像を予測したい
- **REINS APIの開放可能性**：一般消費者がREINSに（匿名化して）アクセスできれば情報の非対称性は劇的に縮小する。なぜ業界はクローズドを維持するのか、政策論として追いたい
