# リバースモーゲージとリースバック — 高齢期の住宅資産活用の二大スキームを徹底比較
**日付**: 2026-05-16
**分野**: house-hunting
**タグ**: #リバースモーゲージ #リースバック #老後資金 #リバース60 #住宅資産

## 学んだこと

「老後 2000 万円問題」以降、自宅を **「持ったまま現金化したい」** ニーズが顕著になっている。中古マンション購入者にとっても **「将来この物件をどう現金化するか」** は購入時の出口戦略として重要。日本で利用できる二大スキームが **リバースモーゲージ** と **リースバック** で、表面的には似ているが構造はまったく違う。

### 一行で言えば

- **リバースモーゲージ**：自宅を **担保にして借金** する（所有権は維持）
- **リースバック**：自宅を **売却して賃貸として住み続ける**（所有権を失う）

格言化すれば「**リバースモーゲージは資産を担保に生かす、リースバックは資産を現金化する**」。

### リバースモーゲージの仕組み

(1) 自宅を担保に金融機関から融資枠を設定
(2) 一括または分割で現金を受け取り、生涯に渡って利息のみ返済（または据置）
(3) **借入人死亡時**に自宅を売却して元本一括返済

**主な提供主体**：
- メガバンク（三菱 UFJ、三井住友、みずほ、りそな）— 商品名は各行で異なる
- 都市銀行・地銀
- **東京スター銀行「充実人生」** — リバースモーゲージの草分け
- **住宅金融支援機構「リ・バース 60」** — 60 歳以上、ノンリコース型（後述）

**典型的な条件**：
- **年齢**：満 55 歳〜80 歳（金融機関で異なる）
- **物件**：戸建て中心、マンションは制限あり（ある金融機関では「築年数 20 年以内」「専有面積 50 ㎡以上」「都心部のみ」など厳しい）
- **担保評価**：路線価ベースで保守的に評価、**評価額の 50-70% が融資上限**
- **金利**：変動金利 2-3.5%（2026 年現在）、毎月利息のみ返済

**リ・バース 60 の特殊性** — ノンリコース型を選べる：死亡時に自宅売却額が残債を下回っても、**遺族に追加請求しない**。利息は通常型より若干高くなるが、相続人への債務移転リスクを切る。2026 年現在、Reverse 60 利用件数は累計 8,000 件超で年々増加。

### リースバックの仕組み

(1) 自宅を不動産会社・投資家に **売却**
(2) 売却代金を **一括受領**
(3) 同時に **賃貸契約** を結び、家賃を払って住み続ける
(4) 任意で **買戻し特約** を付ける（数年以内に同価格で買い戻せる）

**主な提供主体**：
- ハウスドゥ「リースバック」— 上場会社の主力商品
- セゾンファンデックス
- 大手不動産会社の専門部署

**典型的な条件**：
- **年齢制限なし** — 何歳でも利用可能（リバースモーゲージとの最大の違い）
- **物件種類問わず** — マンション・戸建て・収益物件まで
- **売却価格**：**市場価格の 60-80%** が一般的（業者が転売利益を確保するため安く買う）
- **家賃**：売却価格の **7-10% / 年**（つまり 10-14 年で売却額を回収する設定）
- **賃貸契約**：定期借家契約（2-3 年が多い、再契約で延長）

### 4 つの軸での比較

| 軸 | リバースモーゲージ | リースバック |
|---|---|---|
| **所有権** | 維持 | 失う |
| **資金受取** | 融資枠から都度引出 | 売却代金を一括 |
| **月々支出** | 利息（借入額の 2-3.5%/年） | 家賃（売却額の 7-10%/年） |
| **死亡時** | 自宅売却で元本返済 | 既に他人の所有物 |
| **年齢制限** | 55-80 歳 | なし |
| **物件制限** | 戸建て中心 | ほぼ全種類 |
| **得られる金額** | 評価額の 50-70% | 市場価格の 60-80% |
| **税金** | 借入金なので非課税 | **譲渡所得税が発生**（条件次第） |
| **相続人** | 遺品なし or 残額相続 | 相続資産なし |
| **手続き期間** | 2-3 ヶ月 | 1-2 ヶ月 |
| **解約・買戻し** | 借入返済で解約 | 買戻し特約で再取得可能（任意） |

### 「向いている人」のセグメント

**リバースモーゲージが向く**：
- **生活費の補填**を月々得たい高齢者
- 持ち家を **手放したくない** 心情がある
- 都市部の **担保評価が高い戸建て** 所有者
- 相続人がいない or 自宅を残す意思がない

**リースバックが向く**：
- **まとまった資金**が必要（事業資金、医療費、住宅ローン残債一括返済）
- **若年〜中年で困窮**している（年齢制限がない）
- **物件種類で制限を受けたくない**（マンションでも可）
- 将来 **買い戻したい**（買戻し特約で柔軟性）

### 注意点・落とし穴

**リバースモーゲージ**：
- **金利上昇リスク** — 変動金利なので将来の返済負担が膨らむ
- **不動産価値下落リスク** — 担保評価額が下がると追加担保要求
- **長生きリスク（リコース型の場合）** — 想定より長生きすると借入総額が担保価値を超え、遺族に債務移転
- **配偶者居住問題** — 借入人死亡後、配偶者の居住権が消える金融機関がある（配偶者連帯型を選ぶ必要）
- **マンションは敬遠される** — 戸建てと違い管理費・修繕積立金があり、長期保有での評価減リスクが大きい

**リースバック**：
- **売却価格が安い** — 市場価格の 60-80% で買い叩かれる
- **家賃が高い** — 売却代金を 10-14 年で回収するため周辺相場より割高
- **定期借家契約の再契約リスク** — 業者の判断で再契約拒否されうる（実際は稀だが）
- **家賃滞納で退去リスク** — 通常の賃貸と同じ
- **譲渡所得税** — 売却益が出れば課税。マイホーム特例（3,000 万円控除）は適用可能だが、低い売却価格で逆に **譲渡損失** で他所得との損益通算は不可
- **悪質業者問題** — 国民生活センターに苦情多数、2022 年に消費者庁が注意喚起

### 第三の選択肢

- **住宅ローン残債のある場合の特殊スキーム** — リバースモーゲージは原則「無借金物件」要件あり。残債がある場合は「リバースモーゲージで残債一括返済」スキームの一部金融機関で対応
- **不動産担保ローン** — リバースモーゲージより自由度高い、通常返済型
- **マイホーム借上げ制度（JTI）** — 50 歳以上の持ち家を JTI（移住・住み替え支援機構）が借り上げ、転貸する。空室時も家賃保証。リースバックの賃貸版

## 気づき・洞察

リバースモーゲージとリースバックは **「資産流動化の二様式」** — 担保型と売却型。これは [[cs/B木とデータベースインデックス]] の B 木と LSM ツリーの関係（同じ「キーバリュー」を異なるトレードオフで実装）に近い。どちらが優れているかではなく、**どんな状況・目的にどちらが適合するか** という問題。

**「持ち家信仰」のコストが可視化される** — 中古マンションを買うとき、多くの人は「老後の安心」を理由にする。しかし老後に資金が必要になったとき、**マンションは流動性の悪い資産** で、リバースモーゲージでは戸建てより低評価、リースバックでは家賃が高くなる。**「持ち家がある」と「老後の現金がある」は別物**。これは [[house-hunting/賃貸_持ち家_NPV分析と機会費用]] の延長で、購入時の意思決定に「出口戦略」を組み込む必要がある。

最も重要な観点 — **リースバックの「家賃 7-10% / 年」は実質的に 10-14 年の家賃割賦** で、これは **隠れた金利** として年率 6-9% 相当（築古マンションの実質利回りに近い）。**「現金を急ぐ代償としての高利」** が支払われている。一方リバースモーゲージは利息 2-3% で長期借入できる。**「時間があるか緊急か」** が判断軸になる。

**マンション所有者の選択肢が狭い** — リバースモーゲージは戸建て中心、リースバックも区分所有マンションでは家賃設定が難しい（管理費・修繕積立金を家賃に含めると割高に見える）。**マンションを買う時点で「老後の現金化が難しい」コストを織り込むべき**。これは中古マンション購入者にとって重要な認識。

業界の倫理問題 — **悪質リースバック業者** が高齢者を狙う事案が増加。買戻し特約の小さい文字、3 年後の家賃値上げ、定期借家契約での退去要求など。**「資金が必要な状況の高齢者」は情報非対称性で不利に立たされる**。2026 年現在、宅建業法改正で重要事項説明にリースバック特有の説明義務が拡充される予定。**[[house-hunting/重要事項説明書の読み方と告知義務]] の知識が老後にも効く**。

[[house-hunting/共有名義と贈与税相続税の落とし穴]] との連動 — リースバックで自宅を売却すると、相続資産が現金に変わる。これは相続税対策として **「現金は不利、不動産は有利」** という従来常識を反転させる選択。一方リバースモーゲージは自宅という不動産形態で相続させるが、債務付き。**相続戦略との整合性**が必須。
