# 引き渡し後のやることリスト（Post-Purchase Checklist After Property Handover）
**日付**: 2026-03-23
**分野**: 家探し
**タグ**: #引き渡し #手続き #転居届 #ライフライン #住宅ローン控除 #リフォーム #横浜市

## 学んだこと

### 全体タイムライン：優先度順チェックリスト

引き渡し（決済・鍵受領）後の手続きは、法的期限のあるものから順に着手する。以下、時系列で整理する。

---

### Phase 1: 引き渡し当日〜1週間以内（最優先）

#### 1. 登記識別情報通知の保管
- 決済日に司法書士が法務局へ登記申請。約1〜2週間後に**登記識別情報通知**（12桁のパスワード形式）が届く
- 従来の権利証に相当するもので、**再発行不可**。金庫や貸金庫での厳重保管が必須
- 目隠しシール（折込み方式）を不用意に剥がさない。剥がすと再封不可

#### 2. ライフライン開通手続き
引き渡し前（1週間前まで）に連絡しておくのが理想だが、当日以降でも対応可能。

| ライフライン | 手続き方法 | 注意点 |
|---|---|---|
| **電気** | 電力会社のWebまたは電話で申込。スマートメーターなら立会不要 | 旧住所の停止手続きも忘れずに |
| **ガス** | 東京ガス等にWeb/電話で開栓予約 | **立会い必須**（ガス漏れ点検・開栓確認）。予約が混む時期は2週間前推奨 |
| **水道** | 横浜市水道局（045-847-6262）またはWeb | 横浜市は開栓依頼不要の場合あり（元栓を開けて蛇口をひねればOK）。ただし使用開始届は必要 |

**横浜市水道局**: 使用開始届はWebから可能（https://www.city.yokohama.lg.jp/kurashi/sumai-kurashi/suido-gesui/）

#### 3. 管理組合への届出
- 区分所有者変更届（入居届）を提出。通常は売買決済時に仲介業者が手配済みだが、未提出の場合は速やかに
- 管理組合所定の書式で、**緊急連絡先・車両情報・同居人情報**等を届出
- 管理人への挨拶を兼ねて直接持参するのが望ましい。ゴミ出しルール、宅配ボックスの使い方、自転車置場のルール等の実務情報を収集する好機

#### 4. 近隣への挨拶
- **対象**: 両隣＋真上＋真下の4戸（最低限）。加えて管理人
- **タイミング**: 引越し作業の前日まで（騒音・トラック等で迷惑をかける前に）。不在なら引越し後1週間以内に再訪
- **手土産**: 500〜1,000円程度。日持ちする菓子・タオル等。のし紙は「ご挨拶」で苗字を記載
- リフォーム工事を行う場合は、**工事の挨拶と入居の挨拶を分ける**（工事挨拶はリフォーム業者と連携して工事開始1週間前に実施）

---

### Phase 2: 引き渡し後14日以内（法定期限あり）

#### 5. 転入届・転居届（住民票の異動）
**法定期限: 引越し日から14日以内**（住民基本台帳法第22条・第23条。正当な理由なく届出を怠ると5万円以下の過料）

| パターン | 手続き |
|---|---|
| 横浜市外→横浜市 | 旧住所の市区町村で**転出届**→横浜市の新住所の区役所で**転入届** |
| 横浜市内の別の区へ | 新しい区の区役所で**転出届＋転入届を同時提出** |
| 横浜市内の同一区内 | 同じ区役所で**転居届** |

**横浜市の窓口受付時間**:
- 月〜金: 8:45〜17:00
- 第2・第4土曜: 9:00〜12:00（祝日・年末年始を除く）
- **3〜4月は混雑**。横浜市は一部手続きでマイナポータルからの転出届オンライン提出に対応

**必要書類**:
- 転出証明書（市外からの転入の場合）またはマイナンバーカード（マイナンバーカードによる転出の場合は転出証明書不要）
- 本人確認書類（マイナンバーカード、運転免許証等）
- 印鑑

**同時に行う関連手続き**（区役所ワンストップ）:
- マイナンバーカードの住所変更（継続利用手続き）
- 国民健康保険の加入届（会社員は勤務先で手続き）
- 国民年金の住所変更届（第1号被保険者の場合）
- 児童手当の認定請求（新規）——横浜市は「パマトコ」アプリでも電子申請可
- 印鑑登録（旧住所のものは転出で自動廃止。新住所で再登録が必要な場合）

#### 6. 郵便局の転送届（転居届）
- **郵便局窓口**またはe転居（https://welcometown.post.japanpost.jp/etn/）でオンライン申請
- 転送期間は届出日から**1年間**。届出から実際の転送開始まで3〜7営業日かかる
- **届出は早めに**（引越し前でも届出可能。転送開始日を指定できる）
- 1年後に転送延長届を再提出すれば延長可能

---

### Phase 3: 引き渡し後1ヶ月以内（早めに対応）

#### 7. 運転免許証の住所変更
- **届出先**: 新住所を管轄する警察署または運転免許センター
  - 横浜市内: 神奈川県警察運転免許センター（二俣川）、または最寄りの警察署
- **必要書類**: 免許証＋新住所の住民票（マイナンバー記載なし）またはマイナンバーカード
- 法的な届出期限は明記されていないが、「速やかに届け出る」義務あり（道路交通法第94条）
- 免許証は本人確認書類として最も頻繁に使うため、**住民票取得後すぐに変更するのが実務的**

#### 8. 銀行・金融機関への届出
- 住宅ローン契約の金融機関: **新住所の住民票を提出する義務**がある（契約書に記載）。一定期間内に未提出だと契約違反のおそれ
- 給与振込口座、クレジットカード会社、証券口座等も順次住所変更

#### 9. 火災保険・地震保険の確認
- 決済日に加入済みのはず（ローン実行の条件）。保険証券の内容を改めて確認
- 住所表記の確認、補償内容（水災の要否など）の最終確認
- 既存エントリで検討した「水災補償の要否」「個人賠償責任特約」等の判断を反映

#### 10. NHK受信料の届出
- **NHK受信料の窓口**（https://www.nhk-cs.jp/）からオンラインで住所変更
- 電話でも可（0120-151-515、9:00〜20:00、土日祝対応）
- 既に契約がある場合は**住所変更**、新規の場合は**新規契約**

#### 11. インターネット・TV
- 光回線の移転手続きまたは新規契約。工事が必要な場合は2〜4週間かかることが多い
- **マンションのインターネット環境を管理組合に確認**: VDSL方式（最大100Mbps）か光配線方式（最大1Gbps〜）か。VDSL物件では個別に光ファイバー引込みが困難な場合あり
- モバイルWi-Fiで仮対応しつつ、固定回線の開通を待つのが現実的

### Phase 4: リフォーム工事（入居前に実施するのが理想）

#### 12. リフォーム工事の順序と注意点

引き渡し後・入居前にリフォームを行うのが最も効率的（家具なし・騒音気にせず・工期短縮）。

**工事準備の手順**:

1. **管理規約の確認**（工事着手の1.5〜2ヶ月前）
   - リフォーム細則の確認: 床遮音等級（LL-45等）、工事可能時間帯、搬入ルート
   - 共用部（窓枠・玄関ドア・外壁・PS）は工事不可

2. **工事申請書の提出**（工事着手の1ヶ月以上前）
   - 管理組合所定の書式で、工事内容・期間・施工業者を記載し理事長の承認を得る
   - 承認まで10日〜2週間。余裕を持って1ヶ月以上前に提出

3. **近隣承認の取得**（工事着手の2〜3週間前）
   - 多くの管理規約で、上下左右の住戸の**書面承諾**が必要
   - 承諾書用紙は管理組合が用意する場合と、施工業者が用意する場合がある

4. **近隣への工事挨拶**（工事着手の1週間前）
   - 両隣＋上下階に、工事期間・作業時間帯・施工業者連絡先を記載した書面を配布
   - 手土産（500〜1,000円）を添えるのがマナー

5. **工事実施の一般的な順序**
   - 解体 → 電気・給排水の先行配管 → 大工工事（間仕切り・床下地） → 断熱工事（内窓含む） → 内装仕上げ（壁・天井・床） → 設備設置（キッチン・UB・洗面） → クリーニング → 竣工検査

6. **竣工検査**
   - 施工業者と一緒に全箇所を確認。不具合があれば手直しリストを作成し、入居前に完了させる

---

### Phase 5: 引き渡し翌年の確定申告（住宅ローン控除）

#### 13. 住宅ローン控除の確定申告

**初年度は確定申告が必須**（2年目以降は年末調整で可能）。

**申告時期**: 入居した年の翌年2月16日〜3月15日（還付申告は1月1日から提出可能）
- 例: 2026年に入居 → 2027年2月16日〜3月16日に確定申告

**必要書類チェックリスト**:

| 書類 | 入手先 |
|---|---|
| 確定申告書（第一表・第二表） | 国税庁HP / 税務署 |
| (特定増改築等) 住宅借入金等特別控除額の計算明細書 | 国税庁HP / 税務署 |
| 住宅取得資金に係る借入金の**年末残高等証明書** | 金融機関（10月〜翌1月に郵送） |
| 売買契約書の写し | 手元保管 |
| 土地・建物の登記事項証明書 | 法務局（オンライン請求500円） |
| 住宅用家屋証明書または認定長期優良住宅の認定通知書写し | 市区町村 / 手元保管 |
| 本人確認書類 | マイナンバーカード等 |

**2026年の中古住宅ローン控除** (既存エントリとの連携):
- 控除率0.7%、借入限度額3,500万円、控除期間13年（大幅拡充）
- 最大控除額409.5万円（2025年の210万円からほぼ2倍）
- 所得要件2,000万円以下、床面積50㎡以上

**e-Tax推奨**: マイナンバーカードがあればスマートフォンから申告可能。税務署窓口の混雑を回避できる

---

### Phase 6: その他（随時）

#### 14. 不動産取得税の軽減申告
- 取得後60日以内に都道府県税事務所へ申告（神奈川県の場合）
- **申告しなくても6ヶ月〜1年後に納税通知が届く**が、軽減措置の申告を忘れると満額課税されるリスク
- 既存エントリで学んだ通り、中古住宅控除（1997年4月以降新築で1,200万円控除）で建物分が実質ゼロになるケースが多い

#### 15. 住所変更登記
- **2026年4月1日以降**（令和8年4月施行）、住所変更から2年以内の変更登記が義務化
- 決済時に新住所で登記した場合は不要。旧住所で登記した場合は変更登記が必要
- 法務省「スマート変更登記」でオンライン申請可能

#### 16. その他の住所変更
- 勤務先への届出
- 各種保険（生命保険・自動車保険等）
- 自動車の車検証の住所変更（陸運支局）
- 選挙人名簿（転入届で自動登録されるが、転入後3ヶ月以上経過しないと選挙権行使不可）

## 気づき・洞察

- 手続きの順序には**依存関係**がある。住民票が起点となり、免許証変更（住民票が必要）→銀行届出（住民票または免許証が必要）と連鎖する。**住民票の移動を最初に完了させるのが全手続きのクリティカルパス**
- リフォームを入居前に行う場合、引き渡し〜入居の間に「住んでいない期間」が生じる。住宅ローン控除は「取得後6ヶ月以内に入居」が要件なので、リフォーム工事期間を考慮したスケジュール管理が必要
- 不動産取得税の軽減申告は「知っている人だけが得をする」制度。申告しなくても納税通知は届くが、軽減措置は自動適用されない場合がある
- 管理組合への届出と近隣挨拶は、法的義務というより**コミュニティへの参入儀礼**。特にリフォーム工事を予定している場合、この段階での良好な関係構築が工事の近隣承認をスムーズにする

## 他分野との接続

- **解剖学**: 引越し作業時の腰椎保護。重い荷物の持ち上げはスクワットの要領（膝関節屈曲、脊柱中立位）で行い、腰椎椎間板への過負荷を避ける
- **テクノロジー**: 住所変更手続きの非効率さはまさに「分散システムの一貫性問題」。住民票を正（Single Source of Truth）として各サービスに伝播させるが、結果整合性しか保証されず、全システムの同期完了まで数週間かかる
- **ドイツ語**: ドイツでもAnmeldung（住民届出）は転入後14日以内が義務。Einwohnermeldeamt（住民登録局）での手続きは日本の転入届と驚くほど似た構造
