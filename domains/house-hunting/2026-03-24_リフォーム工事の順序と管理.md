**日付**: 2026-03-24
**分野**: 家探し

## 学んだこと

### 1. マンションリノベーション工事の標準工程

フルリノベーション（スケルトンリフォーム）の工事は、以下の順序で進行する。各工程には明確な依存関係があり、順序の入れ替えは基本的にできない。

#### Phase 1: 解体工事（1〜2週間）
- **最初に行う**: 既存の間仕切り壁、床材、天井材、設備機器を全て撤去
- 解体の順序: 建具・サッシ取り外し → 設備機器・電気機器撤去 → 間仕切り壁解体 → 床材撤去 → 天井材撤去
- 解体後に躯体（コンクリート）が露出し、**ここで初めて隠れた問題が判明する**（配管腐食、クラック、アスベスト含有建材等）
- 産業廃棄物の分別処理が必要。廃材処分費は面積・内容により15〜40万円程度

#### Phase 2: 墨出し・現場確認（1〜2日）
- 新しい間取りの壁位置、設備位置をコンクリート躯体に直接マーキング
- **施主立会い推奨ポイント**: 図面上の間取りを実際の空間で確認する最後のチャンス
- 寸法の微調整（コンセント位置、スイッチ高さ等）はこの段階が最終

#### Phase 3: 先行配管・先行電気配線工事（1〜2週間）
- **給排水配管**: 水回りの新しい位置に合わせた配管ルートの敷設。排水勾配（50分の1以上）の確保が最重要
- **電気配線**: 分電盤からの幹線、各部屋への分岐配線、スイッチ・コンセントの位置決め
- **ガス配管**: ガスコンロ使用の場合のみ。IH化なら不要だが200V配線が必要
- 壁・天井が塞がる前に完了させる必要があるため「先行」と呼ぶ
- **LAN配線・TV配線もこの段階**: 後からの追加は壁の一部解体が必要

#### Phase 4: 大工工事・下地造作（2〜4週間）
- **床下地**: 置き床工法（二重床）の場合、支持脚の設置と合板下地の敷設。遮音等級LL-45確保
- **間仕切り壁下地**: LGS（軽量鉄骨）またはスタッドによる壁骨組み。断熱材（グラスウール等）の充填
- **天井下地**: 既存天井高に応じて野縁組み。ダウンライト・空調の位置を反映
- **造作家具・収納**: 造り付けの棚、カウンター等の造作
- 最も工期が長く、遅延が全体に波及する**クリティカルパス上の工程**

#### Phase 5: 断熱工事（Phase 4と並行）
- 外壁面への断熱材（スタイロフォーム等）の内貼り
- マンションでは外壁側の壁面と天井面（最上階の場合）のみ
- 中住戸は外気接触面が窓のみのため、内窓設置が主な断熱対策
- 断熱工事は大工工事と並行して進めることが多い

#### Phase 6: ユニットバス設置（Phase 4中盤）
- ユニットバスは**壁・天井が完成する前**に搬入・設置する必要がある（搬入経路の確保）
- 大型パネルを分割搬入し、現場で組み立て
- 設置後、周囲の壁・天井を仕上げていく

#### Phase 7: 内装仕上げ工事（1〜2週間）
- **クロス（壁紙）貼り**: パテ処理（下地の平滑化）→ 下塗り → クロス貼り
- **塗装**: 塗装仕上げの場合は下地処理→下塗り→上塗りの3工程
- **床仕上げ**: フローリング・タイル・クッションフロア等の施工
- **タイル工事**: キッチン・洗面等のタイル貼り

#### Phase 8: 設備据付・器具取付（3〜5日）
- キッチン・洗面台・トイレの設置
- 照明器具・スイッチプレート・コンセントカバーの取付
- エアコンの設置（配管穴は事前に確保済み）
- 建具（室内ドア・引戸）の吊り込みと調整

#### Phase 9: クリーニング・検査（2〜3日）
- 専門業者によるハウスクリーニング
- 施工業者による社内検査（自主検査）
- 施主立会いによる竣工検査

### 2. 工事期間の目安

| リフォーム規模 | 工事期間 | 準備含む全体期間 |
|---|---|---|
| 部分リフォーム（水回り1箇所） | 1〜2週間 | 1〜2ヶ月 |
| 間取り変更あり | 1〜2ヶ月 | 3〜5ヶ月 |
| フルリノベーション（スケルトン） | 2〜3ヶ月 | 4.5〜8.5ヶ月 |

70㎡のマンション・フルリノベーションで**着工から完成まで約40〜60日（6〜9週間）**が一般的。ただし以下で変動する：
- 造作家具が多い → +2〜3週間
- 水回り大幅移動 → +1〜2週間
- 解体後の想定外問題 → +1〜3週間
- 資材の納期遅延（特注品） → 数週間単位で遅延の可能性

**準備期間の内訳**:
- 業者選定・相見積もり: 1〜2ヶ月
- 設計・プランニング・打合せ: 1〜2ヶ月
- 管理組合への申請・承認: 2週間〜1ヶ月
- 近隣への挨拶・書面承諾: 1〜2週間

### 3. 施工管理者（現場監督）との連携

#### 施工管理者の役割
- **工程管理**: 各職種（解体屋、電気工、配管工、大工、クロス職人等）のスケジューリング
- **品質管理**: 各工程の施工品質チェック、是正指示
- **安全管理**: 労働安全衛生法に基づく現場安全確保
- **原価管理**: 予算内での工事完了、追加費用の抑制

#### 施主としての関わり方
- **定例打合せ**: 週1回程度の現場打合せを設定。進捗確認と決定事項の確認
- **現場見学**: 可能なら週1〜2回。ただし職人の作業を妨げないよう配慮
- **連絡手段**: LINEグループやチャットツールで施工管理者と連絡。写真共有が容易
- **議事録**: 決定事項は必ずテキストで残す。「言った・言わない」の防止

#### 施主が立会うべき重要タイミング
1. **解体完了後**: 躯体の状態確認、隠れた問題の発覚共有
2. **墨出し時**: 間取りの最終確認
3. **先行配管・配線完了後**: コンセント・スイッチ位置の確認（壁が塞がると変更不可）
4. **ユニットバス設置前**: 搬入するUBの仕様最終確認
5. **クロス貼り前**: 下地の平滑度確認
6. **竣工検査**: 全体の最終チェック

### 4. 追加工事（変更指示）の管理

追加工事はリノベーションで最もトラブルになりやすい領域。

#### 追加工事が発生する主な原因
- **解体後の発覚**: 配管の腐食、コンクリートのクラック、想定外のアスベスト、スラブ下配管の発覚
- **施主の仕様変更**: 「やっぱりここにも棚が欲しい」「コンセントを追加したい」
- **図面との不整合**: 既存図面の不正確さ（築古物件で多い）、PSの実際の位置が図面と異なる
- **法令・規約による制約発覚**: 管理規約の詳細条件が着工後に判明

#### 追加工事管理の鉄則
1. **必ず書面で**: 口頭の「やっておきますね」は絶対にNG。変更指示書（追加工事確認書）に内容・金額・工期影響を明記
2. **事前見積もり**: 追加工事の都度、金額と工期への影響を書面で提示してもらう
3. **承認プロセス**: 施主の書面承認なしに着工しないルールを契約時に取り決め
4. **写真記録**: 変更箇所のBefore/After写真を必ず撮影。日付入り
5. **予備費からの充当**: 総予算の10〜15%を予備費として確保しておく（前回エントリで学習済み）

#### 追加工事の費用相場
- コンセント追加: 1箇所5,000〜15,000円
- 照明位置変更: 1箇所10,000〜30,000円
- 配管経路変更: 5〜20万円
- 間仕切り壁追加: 1箇所10〜20万円
- 解体後のアスベスト処理: 30〜100万円（レベルにより大幅変動）
- 配管全面交換（解体後に劣化判明）: 30〜80万円

### 5. 品質チェックポイント

#### 中間検査（施工中）
- **配管の水圧テスト**: 給水管に加圧し漏水がないことを確認
- **電気配線の絶縁抵抗測定**: 配線の絶縁不良がないことを確認
- **防水処理**: ユニットバス周辺、キッチン背面の防水シート施工確認
- **断熱材の充填状態**: 隙間なく充填されているか、防湿シートの施工
- **下地の平滑度**: クロス貼り前の下地パテ処理。照明を当てて陰影チェック

#### 竣工検査チェックリスト
- [ ] クロス: 剥がれ、浮き、ジョイント部のずれ、汚れ
- [ ] 床: 傾き（水準器アプリで確認）、きしみ、フローリングの隙間
- [ ] 建具: 開閉のスムーズさ、戸当たり、鍵の動作
- [ ] 水回り: 全蛇口の通水確認、排水速度、水漏れ（配管接合部）
- [ ] 電気: 全スイッチ・コンセントの動作確認、照明の点灯
- [ ] 換気: 24時間換気システムの動作、レンジフードの排気
- [ ] 設備: エアコン・食洗機・IH等の動作確認
- [ ] 収まり: 巾木、見切り材、コーキングの仕上がり
- [ ] 清掃: ハウスクリーニングの仕上がり

**不具合発見時**: 竣工検査で見つかった不具合は「ダメ出しリスト」としてまとめ、是正完了後に再検査。引渡し前に全項目クリアが原則。

### 6. 近隣対応と管理組合手続き

#### 管理組合への工事申請
1. **管理規約の確認**: リフォーム細則の有無。工事可能時間帯、使用材料の制限（遮音等級等）
2. **工事申請書の提出**: 通常1ヶ月前まで。設計図面、工程表、施工業者情報を添付
3. **理事会の承認**: 管理組合によっては理事会の承認が必要。総会決議は通常不要（専有部リフォーム）
4. **近隣住民の書面承諾**: 上下左右の住戸から承諾書を取得する規約が多い

#### 近隣への挨拶
- **挨拶範囲**: 両隣（左右）＋上下階（真上・真下）＋斜め上下 = 最大8戸。エレベーター共用の同階全戸にも挨拶が理想
- **挨拶のタイミング**: 工事開始の1〜2週間前
- **伝える情報**: 工事内容の概要、工事期間、作業時間帯、騒音・振動が大きい時期、施工業者の連絡先
- **挨拶品**: 500〜1,000円程度の日用品（タオル、洗剤等）
- **書面の配布**: 口頭だけでなく、工事概要を記載した書面を投函

#### 騒音・振動管理
- **最も騒音が大きい工程**: 解体工事（1〜2週間目）。コンクリートへの斫り（はつり）が最大
- **作業時間の制限**: 多くのマンションで9:00〜17:00（土日祝休み）。管理規約で異なる
- **防音対策**: 養生シートによる廊下・エレベーター内の保護、防音パネルの設置
- **共用部の使用**: エレベーターの養生（台車使用で傷防止）、廊下への資材仮置きは事前許可
- **搬入経路**: 大型資材（ユニットバス、キッチン）の搬入はエレベーターサイズを事前確認

### 7. 工事保証とアフターサービス

#### 工事保証の種類
- **施工保証**: 施工業者が施工した箇所の不具合を保証。期間は1〜10年（業者により異なる）
- **設備保証**: 取り付けた設備機器の不具合保証。メーカー保証（通常1〜2年）とは別に施工業者が延長保証を提供する場合あり
- **防水保証**: ユニットバス周辺等の防水工事に対する保証。通常5〜10年

#### 保証書で確認すべき事項
- 保証期間の始期と終期
- 保証の対象範囲（施工部分のみか設備含むか）
- 免責事項（経年劣化、施主の過失、天災等）
- 保証の実行方法（連絡先、対応期限）

#### アフターサービス
- **定期点検**: 引渡し後3ヶ月、6ヶ月、1年、2年が一般的
- **初期不具合**: クロスの乾燥収縮によるジョイント開き、建具の季節変動による調整は保証範囲内
- **記録の保管**: 保証書、図面、仕様書、取扱説明書を一式保管。後のメンテナンスで必要

### 8. 横浜エリア特有の留意点

- **横浜市の工事届出**: 特定建設作業（杭打ち・解体等）は騒音規制法・振動規制法に基づき事前届出が必要。ただしマンション室内リフォームは通常該当しない
- **アスベスト事前調査（2022年4月〜）**: 大気汚染防止法改正により、一定規模以上の解体・改修工事でアスベスト事前調査が義務化。報告対象は請負金額100万円以上or解体面積80㎡以上
- **横浜市のマンション管理規約**: 横浜市は管理組合運営の支援制度があり、リフォーム細則が整備されているマンションが多い
- **坂の多い横浜**: 資材搬入経路の確認が重要。丘陵地のマンションはトラック進入に制約がある場合あり
- **築古マンションの注意**: 横浜エリアは築30年以上の物件が多い。スラブ下配管の可能性、アスベスト含有建材の可能性が高く、解体後の追加費用リスクが大きい

## 気づき・洞察

### 工事工程はDAG（有向非巡回グラフ）
各工程の依存関係はソフトウェアのビルドシステムと同じ構造。解体→配管/配線→大工→内装→設備→検査という依存チェーンがあり、上流の遅延は全て下流に波及する。だからこそ「クリティカルパス」の概念がそのまま適用でき、大工工事がボトルネックであることを理解しておく必要がある。

### 「壁が閉じる前」が最後の分岐点
配管・配線は壁の中に隠れる。壁が塞がった後の変更は、壁の解体→再施工→再仕上げという3倍のコストがかかる。この「不可逆性の壁」が存在する工事では、前半の決定精度が後半のコストを決定的に左右する。ソフトウェアにおける「設計段階のバグ修正コストは実装後の1/10」と同じ構造。

### 追加工事管理はChange Managementそのもの
追加工事の管理プロセス（書面での変更要求→影響分析→承認→実行→記録）は、ITILのChange Managementプロセスと構造的に同一。「口頭のやっておきますね」は未承認のhotfixと同じで、後からの追跡が困難になる。

## 他分野との接続

- **tech（プロジェクト管理）**: 工事の工程管理はアジャイル/ウォーターフォールのハイブリッド。全体はウォーターフォール（工程の順序は固定）だが、各工程内はイテレーティブ（現場での微調整）。スクラムの「スプリントレビュー」は「施主立会い」に対応する
- **anatomy（身体の層構造）**: 建物のリフォームも身体のように「層」で構成されている。躯体（骨格）→配管配線（神経・血管）→下地（筋膜）→仕上げ（皮膚）。深い層の問題ほど修復コストが高い
- **cs（DAG・トポロジカルソート）**: 工程間の依存関係を有向非巡回グラフとして表現し、トポロジカルソートで最適な実行順序を決定するのは、コンパイラのタスクスケジューリングと同じ問題
- **bread（発酵の不可逆性）**: パン作りの「オーブン投入後は戻せない」と同じく、壁を閉じた後の変更は困難。成形前に生地の状態を完璧にすることが、リフォームの配管完了時点の確認に対応する
