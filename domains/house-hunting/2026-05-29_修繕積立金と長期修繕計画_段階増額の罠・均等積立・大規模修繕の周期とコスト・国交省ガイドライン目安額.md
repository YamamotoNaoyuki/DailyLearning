> 補足: 本テーマは [[修繕積立金値上げ問題と均等積立方式 (2026-05-02)]] で制度の骨格（2方式・0.6/1.1倍ルール・認定制度連動）を扱った。本エントリは重複を避け、**(a) ガイドラインの一次資料の正確な数値**、**(b) 0.6/1.1倍ルールと「均等＝段階」等価性の数理**、**(c) キャッシュフロー／現在価値（NPV）の観点**、**(d) 中古購入時に積立不足を見抜く定量診断**に焦点を絞って深掘りする。

## 学んだこと

### 1. 長期修繕計画 — 「30年・大規模修繕2回」を見据える文書
国交省「長期修繕計画作成ガイドライン（平成20年6月策定、令和6年6月改定）」の骨格：
- **計画期間**: 30年以上、かつ**大規模修繕工事が2回含まれる期間以上**。新築分譲時は「30年以上かつ大規模修繕2回以上」、既存は「30年以上」が標準。
- **大規模修繕の周期**: 従来は固定値だったが、令和6年改定で**幅をもたせた**——外壁塗装・屋上防水等の大規模修繕は「**一般に12〜15年程度**」。部材・工法・仕様で変わるため固定値→範囲表記に。
- **見直し**: **おおむね5年ごと**に建物調査・診断を行い、その結果と**物価動向**を踏まえて計画を改訂。「作りっぱなし」が最大の劣化要因。令和6年改定で**省エネ改修の有効性**と「昇降機の適切な維持管理に関する指針」に沿ったEV点検が追記された。
- 計画は単なる工事スケジュールではなく、**資金収支表**（年度ごとの支出・収入・残高）を伴う。「残高がマイナスになる年がある」計画は破綻計画。

### 2. 大規模修繕の周期が「12〜15年」である本当の理由
技術的最適ではなく**法定調査との同期**が大きい。建築基準法12条の特定建築物定期調査では**10年に一度、外壁全面打診等**が必要。足場・ゴンドラの仮設費は大規模修繕費の15〜20%を占めるため、**全面打診のための足場と外壁・防水の修繕足場を共有**すれば総費用を圧縮できる。よって「打診義務の10年」と「外壁劣化の進行（おおむね12年前後で塗膜・シーリング劣化が顕在化）」のすり合わせ点として12〜15年が選ばれる。費用の構成は概ね **足場15〜20% / 外壁25〜35% / 防水15〜20% / 鉄部・その他＋諸経費**。

工事内容は周期で質的に変わる:
- **1回目（12〜15年）**: 外壁・屋上防水・鉄部塗装・シーリング更新（足場系が主）。
- **2回目（24〜30年）**: 上記に加え**給排水管（更生 or 更新）・玄関ドア・サッシ・機械式駐車場リニューアル・エレベーター更新**が重なる。「**設備更新の津波**」がここに集中し、戸当たり費用が最高化（[[タワーマンションの長期保有リスク (2026-04-28)]]）。給排水管は「更生（ライニングで延命、安い）」か「更新（引き直し、高いが恒久）」かで桁が違う。
- エレベーター更新は1基あたり数千万円規模、機械式駐車場は[[機械式駐車場の維持コストと将来問題 (2026-05-20)]]で別途整理。

### 3. ガイドライン目安額（令和6年6月改定・一次資料の正確な値）
国交省「マンションの修繕積立金に関するガイドライン」の核心は、**「計画期間全体での修繕積立金の平均額（円/㎡・月）」**＝Z＝(A+B+C)÷X÷Y で評価する点（A=計画当初残高、B=計画期間で集める総額、C=専用使用料等繰入総額、X=総専有床面積、Y=計画期間〔月〕）。これと下表の目安を比較する。

**計画期間全体での平均額の目安（機械式駐車場分を除く・円/㎡・月）**

| 地上階数 / 建築延床面積 | 事例の3分の2が包含される幅 | 平均値 |
|---|---|---|
| 20階未満・5,000㎡未満 | 235〜430 | **335** |
| 20階未満・5,000〜10,000㎡未満 | 170〜320 | **252** |
| 20階未満・10,000〜20,000㎡未満 | 200〜330 | **271** |
| 20階未満・20,000㎡以上 | 190〜325 | **255** |
| 20階以上（超高層） | 240〜410 | **338** |

- 幅は標準偏差ではなく**「事例の上下1/6を除外した中央寄り2/3」**で表現。理由はガイドライン自身が明記する通り「事例のばらつきが**正規分布にならない**ため」。＝右に裾を引く分布（一部の高コスト物件が平均を押し上げる）を前提に、平均より**中央値的な頑健性**を優先した設計。
- **20,000㎡以上で平均値がむしろ下がる（255<271）**のは**規模の経済**（工事量がまとまると単価が下がる）。一方**20階以上は338と最高**＝特殊足場・ゴンドラ・共用部比率増のコスト。規模効果と高層プレミアムが逆向きに効く。
- **2011年版→2021年版→2024年版で水準が約+50%に切り上がった**。これは劣化が早まったのではなく、**建設物価（労務費・資材費）の継続上昇**を反映した「事例ベースの更新」。＝目安額自体がインフレ連動する動的基準。

**機械式駐車場の加算額**（駐車場使用料で別会計処理していない場合に上乗せ）
加算額 ＝ 機種別の1台月額修繕工事費 × 台数 ÷ 総専有床面積。機種別単価（令和6年・117事例）:

| 機種 | 1台月額 |
|---|---|
| 2段(ピット1段)昇降式 | 6,450円 |
| 3段(ピット2段)昇降式 | 5,840円 |
| 3段(ピット1段)昇降横行式 | 7,210円 |
| 4段(ピット2段)昇降横行式 | 6,235円 |
| エレベーター方式・垂直循環方式 | 4,645円 |
| その他 | 5,235円 |

ガイドラインのモデルケース（10階・延床7,000㎡・総専有4,900㎡＝70㎡×70戸・3段昇降式30台）では、本体目安252＋駐車場加算36＝**288円/㎡・月**が中心、幅206〜356。実績241円/㎡・月は幅の下限近くだが「範囲内」と判定される。

### 4. 0.6D≤E かつ 1.1D≥F ——「初期0.6倍・最終1.1倍」ルールの数理
令和6年改定の目玉。均等にした場合の平均額をD（=A÷B÷C）とし、段階増額の**最低額E≥0.6D、最高額F≤1.1D**を満たすよう求める（A=総額、B=総専有床面積、C=計画期間〔月〕）。含意:
- 最大/最小の比は **F/E ≤ 1.1/0.6 ≈ 1.83倍**。＝「初期から最終で最大1.8倍まで」。改定前の典型的計画は初期0.4〜0.5D・最終1.5〜2.0Dで、**比が3.5〜5倍**（国交省調査で249事例平均3.58倍、上位42事例5.3倍）あった。これを1.8倍に圧縮する。
- **なぜ「均等そのもの（1.0倍固定）」を強制しないか**。CのDは平均なので、面積×期間の制約上、**初期を下げた分は必ず後期で上げて帳尻を合わせる必要がある**（∫(積立)=総額Bが不変）。完全均等は新築販売を阻害するため政治的に通らない。0.6〜1.1という「狭い段階」は、**販売側の都合（初期負担を見せたい）と資金安定（合意形成リスク低減）のナッシュ的妥協点**。
- 重要な保存則: **どの方式でも計画期間で集める総額Bは同じ**（同じ工事をするなら）。違うのは**徴収のタイミング分布だけ**。＝積立方式の選択は「**いくら払うか」ではなく「いつ払うか（CFの時間配分）**」の問題。ここが次の節の核心。

### 5. キャッシュフローと現在価値（NPV）の視点——「均等≒段階」が崩れる4つの摩擦
工学/数理の読者向けに本質を言えば：**摩擦のない世界では、均等方式と段階方式は区分所有者にとって等価ですらない——むしろ段階方式が有利**である。

- 集める総額B（名目）が同じなら、**段階方式は支払いを後ろ倒し**にする。割引率r>0の下では、後ろ倒しのキャッシュアウトは**現在価値が小さい**。同額の名目負担なら、支払いを遅らせる段階方式の方が**NPV的には軽い**。これは「段階方式が悪」という通説と逆。住宅ローンの繰上返済を遅らせるのと同じ論理（手元資金を市場でrで運用できるなら）。
- ではなぜ国交省は均等を「望ましい」とするのか。**4つの摩擦が割引メリットを食い潰す**から:
  1. **合意形成リスク（最大）**: 値上げには総会の普通決議（区分所有法・標準管理規約）が要る。所有者は新築時から入れ替わり、「最初の合意」を引き継がない人が増える。**値上げ決議の不成立確率**が時間とともに上がり、計画が実行されない＝**実質デフォルト**。段階方式の「未来の値上げ」は無リスク債ではなく**信用リスクのある約束**。
  2. **インフレ／工事費高騰リスク**: 後ろ倒しした分、将来の工事費は上振れしうる（建設物価13年連続上昇）。名目で固定した低い初期積立は、実質購買力で見ると不足が拡大。
  3. **滞納・所有者構成リスク**: 高齢化で年金生活者が増えると値上げ余地が縮む。「払える人が払えるうちに」均す方が頑健。
  4. **資金不足時の調達コスト**: 不足すれば**一時金徴収（数十〜数百万円/戸）か管理組合借入（金利数%）**。借入金利は区分所有者が市場で運用できる利回りを通常上回り、**段階方式の割引メリットを逆ザヤで打ち消す**。
- 結論の数理的整理: **均等方式の優位性は「割引率」ではなく「リスク調整後の割引率」と「実行確率」で語られるべき**。段階方式の名目NPVメリットは、合意形成不成立確率p（年々増加）と借入逆ザヤを織り込むと**期待値で負ける**。＝双曲割引（[[行動経済学とプロスペクト理論 (2026-05-24)]]）に加え、**約束の履行不確実性**が均等優位の真因。

### 6. 中古購入時に「積立不足」を見抜く定量診断
重要事項説明書に載るのは**現在の月額と残高だけ**。それでは不十分。以下を取り寄せて計算する:
- **充足率 ≒ (現在残高 + 計画期間の積立見込総額) ÷ 計画期間総工事費**。80%以上で健全、60%未満は深刻。ただし分母（工事費見積）が古い物価だと過小評価され**充足率が過大に出る罠**——見直し年次を必ず確認（5年超未更新は警戒、10年超は危険）。
- **㎡あたり平均額を逆算してガイドライン幅と比較**: 上表の幅の**下限を下回る**なら、長寿命化促進税制の閾値（下限値）も満たさず、将来値上げ確定のシグナル。
- **段階増額の傾斜を見る**: 計画書で初期/最終の比が1.8倍を大きく超える（旧来の3〜5倍）なら、未実行の値上げが将来に積み残されている。
- **過去5年の臨時徴収・一時金の有無**、**管理組合借入残高**（将来世代へのツケ＝隠れ負債）、**滞納率（5%超は警戒）**。滞納は[[管理組合と区分所有法]]の先取特権・特定承継人への承継（前所有者の滞納が買主に承継される）という形で直接降りかかる。
- **過去3〜5年の総会議事録**: 値上げ議案が「否決され続けている」なら、段階方式が機能不全に陥っている動かぬ証拠。
- **管理計画認定**: 認定マンションは積立基準を満たすシグナルで、**マンション長寿命化促進税制（固定資産税1/6〜1/2減額・参酌1/3、大規模修繕実施年の翌年度、100㎡相当まで）**の対象になりうる。積立金引上証明書（マンション管理士/建築士発行）が要件。

### 7. 「安すぎる中古」の正体
表面の月額が相場（70㎡で均等基準なら月17,500〜25,000円≒250〜350円/㎡）より**大幅に安い物件は、ほぼ例外なく段階増額の初期段階 or 計画破綻**。買った瞬間は得に見えるが、(1)近い将来の値上げ決議、(2)大規模修繕時の一時金徴収、(3)組合借入の返済負担、のいずれか/複数が待つ。**「月々の安さ」は将来CFの前借り**であり、購入価格に織り込まれるべきディスカウント要因。逆に**適正に積み立てている物件は月額が高い分、隠れ負債が小さく、出口（売却）でも認定・健全性が評価される**。

## 気づき・洞察

- **積立方式の選択は「金額」ではなく「キャッシュフローの時間分布」の選択**。総額（積分値）は工事内容で決まり方式不変。動かせるのは支払いタイミングだけ——これは資本予算編成（capital budgeting）そのもの。マンション管理組合は実質、**30年スパンのインフラファンドのCF設計**をしている。
- **段階方式の名目NPVは有利、しかしリスク調整後は不利**という反転が本質。通説「段階＝悪」は理由づけが雑で、正しくは「**未来の値上げは信用リスクのある約束であり、その不履行確率と調達逆ザヤが割引メリットを上回るから均等が頑健**」。ゴルフのDECADE的に言えば（[[ゴルフの期待値とコースマネジメント]]）、**期待値が僅かに高くても分散が大きすぎる賭けは選ばない**——均等方式は低リターン低分散の堅実プレー。
- **ガイドライン目安額が+50%に動いた**事実は、修繕積立金が**名目固定の罠**を持つことを示す。年金保険料が物価スライドするのと同様、積立も実質連動が要る。「均等にしたから安心」も誤りで、ガイドラインは**均等でも5年ごと見直し必須**と明記。＝静的な解は存在せず、**動的に再計算し続ける制御問題**。
- **保存則の視点**: ∫(積立CF)＝総工事費 という制約は、面積×期間で正規化したDを不変量にする。0.6/1.1倍ルールはこの不変量まわりの**許容変動帯（バンド）**を定めたもの。物理の保存量＋摂動許容範囲という構造で、骨のリモデリング（[[骨のリモデリングとウォルフの法則 (2026-05-25)]]）の「定常状態まわりの適応的増減」と同型。

## 他分野との接続

- **golf（期待値・分散管理）**: 均等 vs 段階は「期待値（NPV）」と「分散（実行リスク）」のトレードオフ。名目期待値が高い段階方式を退け、分散の小さい均等を選ぶのは、ボギーを避けてパーを積むコースマネジメントと同じリスク選好。
- **anatomy（骨リモデリング／恒常性）**: 「総量は保存しつつ需要に応じて配分を動的に再計算する」制御は、ウォルフの法則の骨のリモデリング（応力に応じ吸収/形成を平衡させる）と同型。5年ごと見直し＝メカノスタットのフィードバック周期。
- **行動経済学（双曲割引・フレーミング）**: 段階増額の人気は[[行動経済学とプロスペクト理論 (2026-05-24)]]の双曲割引（近い負担を過大に嫌い遠い負担を過小評価）の産物。0.6/1.1倍ルールは**選択肢のフレーム（許容バンド）を絞ることで、双曲割引を法制度で補正**するナッジ＋ハードルール。間取りの壁芯/内法フレーミング（[[マンションの間取りと専有面積 (2026-05-24)]]）と同じ「フレームが意思決定を支配する」系。
- **house-hunting内**: [[修繕積立金値上げ問題と均等積立方式 (2026-05-02)]]（制度骨格）、[[タワーマンションの長期保有リスク (2026-04-28)]]（設備更新の津波・高層プレミアム338円）、[[機械式駐車場の維持コストと将来問題 (2026-05-20)]]（加算単価の出所）、[[第三者管理方式とマンション管理 (2026-05-27)]]（発注ガバナンスが工事費＝積立消費効率を左右）、[[賃貸 vs 持ち家のNPV分析 (2026-05-08)]]（積立金は持ち家CFの確定流出項で、その過小評価はNPV比較を歪める）。
- **music（通奏低音／ラモーの基礎低音）**: 月額の表面値は「旋律」、長期修繕計画と充足率は「隠れた根音（[[ラモーと和声理論の誕生]]）」。表面の安さに惑わされず、CF構造という基礎低音を読むのが賢い買い手。

## 次に深掘りしたいこと

- 充足率の分母（工事費見積）の**物価更新が遅れることで充足率が過大表示される系統誤差**の実証——古い計画ほど見かけ健全に見える逆説の定量化。
- 段階増額の値上げ決議の**不成立確率p(t)の実証モデル**（築年数・所有者交代率・高齢化率を説明変数に）。期待CFをp(t)で割り引いた「リスク調整後充足率」の設計。
- 管理組合借入の**実際の金利水準と貸出基準**（住宅金融支援機構のマンション共用部分リフォーム融資等）、逆ザヤの定量。
- マンション長寿命化促進税制の**取得実績と中古市場プレミアムの実証**——認定×積立健全がいくらの価格差を生むか。
- 海外（シンガポールHDB/コンドのsinking fund、米コンドミニアムのreserve study義務化〔Surfside崩落後の州法強化〕）との**準備金制度比較**——日本の「目安」型 vs 米のreserve study義務型のガバナンス強度差。
