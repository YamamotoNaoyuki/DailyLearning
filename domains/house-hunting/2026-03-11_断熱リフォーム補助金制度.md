# 断熱リフォーム補助金制度（2026年版）
**日付**: 2026-03-11
**分野**: 家探し
**タグ**: #補助金 #断熱 #内窓 #先進的窓リノベ #横浜市 #省エネ

## 学んだこと

### 1. 国の補助金制度（2026年版）

#### 先進的窓リノベ2026事業
- **予算**: 1,125億円（2025年の約半分）
- **上限**: 1戸あたり**100万円**（2025年は200万円から半減）
- **変更点**: Aグレード除外、**Sグレード（Uw 1.5以下）以上のみ対象**
- **補助額目安**: 内窓1箇所あたり約2〜5万円（サイズ・グレードによる）
- **3LDK（7箇所）の場合**: 補助額約15〜30万円
- **申請期限**: 2026年12月31日（予算消化次第終了。2025年は8月に予算枯渇）

#### 子育てグリーン住宅支援事業（こどもエコすまい後継）
- **対象**: 子育て世帯・若者夫婦世帯（夫婦いずれかが39歳以下）
- **リフォーム上限**: 45万円（子育て世帯）/ 30万円（一般）
- **内窓も対象**: ただし先進的窓リノベとの**重複申請は不可**
- **他の省エネ工事と合算可能**: 高効率給湯器、節水トイレなどと組み合わせ

#### 既存住宅の省エネリフォーム減税
- **工事費の10%を所得税から控除**（上限25万円）
- **住宅ローン減税との併用不可**（どちらか有利な方を選択）
- **固定資産税の減額**: 工事翌年度の固定資産税が1/3減額（120m²まで）

### 2. 神奈川県の補助金

#### 神奈川県既存住宅省エネ改修費補助金
- **補助率**: 工事費の1/3（上限50万円）
- **対象**: 県内の既存住宅の省エネ改修（内窓含む）
- **条件**: 改修後にUA値がZEH基準（0.60以下）または断熱等級5以上を満たすこと
- **注意**: 国の補助金との**併用可能**（ただし補助金合計が工事費の8割を超えない範囲）

### 3. 横浜市独自の制度

#### 横浜市住まいのエコリノベーション補助制度
- **補助率**: 工事費の1/5（上限20万円）
- **対象工事**: 窓の断熱改修、壁・天井・床の断熱改修、高効率給湯器等
- **条件**: 横浜市内の既存住宅。施工前に申請が必要
- **特徴**: 国・県の補助金と**併用可能**
- **注意**: 予算が少なく**先着順**で終了するため、年度初め（4月）の申請が望ましい

#### 横浜市の省エネ住宅相談
- 横浜市建築局が無料の省エネ住宅相談窓口を設置
- 補助金申請のサポート、施工業者の紹介、UA値計算の助言

### 4. 補助金の併用パターン（最適戦略）

**中古マンション・内窓7箇所の場合（3LDK想定）**:

| 制度 | 補助額（目安） |
|------|-------------|
| 先進的窓リノベ2026 | 15〜30万円 |
| 神奈川県省エネ改修費補助 | 最大約17万円（工事費1/3） |
| 横浜市エコリノベーション | 最大10万円（工事費1/5） |
| **合計** | **約42〜57万円** |

**工事費50〜70万円に対して42〜57万円の補助→実質自己負担8〜28万円**

ただし：
- 国+県+市の補助金合計が工事費の8割を超えないよう調整が必要
- 先進的窓リノベはSグレード以上が条件→インプラスならスチール骨材でクリアしやすい
- 全制度とも**施工前の申請**が必須。施工後の申請は不可

### 5. 申請手続きの流れ

```
1. 見積もり取得（複数業者推奨）
2. 補助金対象グレードの確認（Sグレード以上か）
3. 横浜市補助金の申請（先着順、4月が理想）
4. 神奈川県補助金の申請
5. 先進的窓リノベの申請（施工業者が代行）
6. 施工
7. 完了報告書の提出
8. 補助金受領（施工後2〜3ヶ月）
```

**重要**: 各制度で申請タイミングが異なる。横浜市は年度初め、先進的窓リノベは予算消化前に申請すべき。2025年は8月に予算枯渇した実績あり。

### 6. 断熱改修による光熱費削減効果

- **東京大学前研究室の実証データ**: 内窓設置により冬季の暖房エネルギー消費が**約20-30%削減**
- **年間光熱費削減**: 約1.8〜2.4万円（電気代単価による）
- **投資回収期間**: 補助金適用後の自己負担額から計算→**5〜15年**（従来の25〜33年から大幅短縮）
- **結露減少**: 結露によるカビ発生が減少→健康面でのメリット（アレルギー、喘息リスク低減）
- **ヒートショック予防**: 浴室・トイレの温度差が縮小。65歳以上の入浴中死亡は年間約19,000人

### 7. 2028年問題との連動

2028年以降の住宅ローン減税では、省エネ基準適合住宅（断熱等級4）ではローン減税が受けられなくなる。**ZEH水準（等級5）以上が必須**。

中古マンションを断熱リノベでZEH水準に引き上げれば：
- 住宅ローン減税の適用を維持（最大409.5万円の控除）
- 内窓設置だけで住戸全体のUA値を約14%改善→中住戸なら等級5到達の可能性
- **補助金で内窓を付ける＋ローン減税を維持**という二重のメリット

### 8. 断熱改修と資産価値

- **BELS評価の向上**: 断熱性能が可視化され、売却時の差別化要因に
- **ZEH水準の資産プレミアム**: 築10年・ZEH水準のマンションは同等の非ZEH物件より**約5-8%高い売却価格**（不動産経済研究所、2025年調査）
- **ランニングコスト開示**: 2024年から新築マンションの省エネ性能表示が義務化。中古でもBELSがあれば比較可能

## 気づき・洞察

補助金を3層（国＋県＋市）で併用すると、実質自己負担が8〜28万円まで下がる。工事費の60-80%が補助金で賄える計算。これは「知っているか知らないか」で数十万円の差が出る情報格差の典型例。

2028年問題との連動は特に重要。内窓設置は「断熱性能の向上」と「住宅ローン減税の維持」という二重の目的を同時に達成できる。中古マンション購入時のリノベ計画に内窓を組み込むことが、2026-2027年の購入者にとって最適戦略となりうる。

先進的窓リノベのSグレード要件（2026年の変更点）は、前回学んだインプラス（スチール骨材）が有利という結論を強化する。

## 他分野との接続
