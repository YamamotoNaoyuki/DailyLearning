# Beim Arzt — 医者での会話
**日付**: 2026-05-08
**分野**: deutsch
**タグ**: #B1 #medical #doctor #symptoms #appointment #neutral

## 学んだこと

ドイツでの医者通いは生活の必須シーン。Hausarzt（家庭医）→ Facharzt（専門医）への紹介システム、Krankenkasse（健保）の手続き、薬局（Apotheke）での処方薬受取りなど、**システム特有の語彙**を覚える必要がある。教科書的フレーズを脱して、**ネイティブが実際に診察室で使う**4つを学ぶ。

---

### Phrase 1: Mir geht's seit Tagen mies, ich krieg den Kopf nicht frei.
**日本語訳**: ここ数日、調子悪くて頭がぼーっとしてる。

**単語リスト**:
- mir geht's = 私の調子は（"es geht mir" の口語形）
- mies = ひどい・最悪な（口語、umgangssprachlich）
- seit Tagen = ここ数日来
- den Kopf nicht frei kriegen = 頭をスッキリさせられない、もやもやする

**使う場面**: 友人や同僚に体調を訴える時、または医者の前で雑談的に状態を伝える導入。"Ich fühle mich nicht gut" よりずっと自然で、感情の重さが伝わる。

**フォーマル度**: umgangssprachlich（口語的）

**キーワード解説**:
- *mies* は形容詞「ひどい」だが、副詞的に使うと「気分が最悪」のニュアンス。"Mir ist mies" は身体的な不調、"Es geht mir mies" は精神的な落ち込みも含む
- *kriegen* は "bekommen"（手に入れる）の口語形。話し言葉では圧倒的に *kriegen* が多用される
- *den Kopf frei kriegen / haben* は「頭をクリアにする・気分転換する」の意味で、否定形だと「集中できない、もやがかかったように感じる」

**関連表現**:
- "Ich fühl mich total schlapp." = ぐったりして気力がない（schlapp = ぐったりした）
- "Mir ist heute irgendwie komisch." = なんか今日変な感じ（komisch の曖昧表現）
- "Ich bin völlig im Eimer." = 完全にダウンしてる（口語強調、"im Eimer" = バケツの中 = ダメ・壊れた）

### Phrase 2: Können Sie mich krankschreiben? Ich schaff's nicht zur Arbeit.
**日本語訳**: 病欠の診断書を出していただけますか？仕事に行ける状態じゃないんです。

**単語リスト**:
- krankschreiben = 病欠の診断書を発行する（分離動詞）
- schaffen = 成し遂げる、ここでは「（仕事に）行ける状態にする」
- zur Arbeit = 仕事へ（"zu der Arbeit" の縮約、Dativ）

**使う場面**: 診察の終盤、医者に診断書（**Arbeitsunfähigkeitsbescheinigung, AU**、通称 *gelber Schein* = 黄色い紙）を依頼する時。ドイツでは病気で3日以上休む場合、雇用主にこれを提出する義務がある（最近は電子化されつつある）。

**フォーマル度**: neutral（医者へのSie呼びかけ、ただし文末は口語的）

**キーワード解説**:
- *krankschreiben* は「**病気と書く（書類を作成する）**」の合成動詞。分離動詞なので "Ich schreibe Sie krank" のように動詞が分離する。文中で必ず "schreibe ... krank" の語順
- *Ich schaff's nicht* = "Ich schaffe es nicht" の口語縮約。**schaffen + zu不定詞 / 場所** の構造で「〜まで辿り着く・成し遂げる」
- *zur Arbeit gehen / fahren* は「通勤する」の決まり文句。冠詞は通常付けず "zur Arbeit"（die Arbeit + zu = zur）

**関連表現**:
- "Brauchen Sie eine AU?" = 病欠証明書必要ですか？（医者から）
- "Wie lange werden Sie mich krankschreiben?" = どのくらいの期間、診断書出してもらえますか？
- "Ich brauche einen gelben Schein für meinen Arbeitgeber." = 雇用主用の病欠証明書が必要なんです

### Phrase 3: Das zieht so komisch im Rücken, vor allem wenn ich mich bücke.
**日本語訳**: 背中が変な感じに引っ張られる、特に前かがみになる時。

**単語リスト**:
- ziehen = 引く、（痛みが）ピリピリする・引きつる
- komisch = 変な、奇妙な
- vor allem = 特に
- sich bücken = 前かがみになる、身をかがめる（再帰動詞）

**使う場面**: 痛みの**性質**を医者に伝える時。「痛い」だけでなく「どう痛いか」を表現できると診断精度が上がる。整形外科（Orthopäde）や Hausarzt で頻出。

**フォーマル度**: neutral

**キーワード解説**:
- ドイツ語の痛み表現は**痛みのタイプで動詞が変わる**：
  - *Es zieht*（引きつる、神経痛・筋肉痛系）
  - *Es sticht*（刺すように痛む、急性痛）
  - *Es brennt*（燃えるように痛む、皮膚炎・神経痛）
  - *Es pocht / pulsiert*（ズキズキ脈打つ、頭痛・炎症）
  - *Es drückt*（押されるように痛む、圧迫感）
- *sich bücken* は **再帰動詞 + Akkusativ**（自分自身を曲げる）。日本語の「かがむ」と一対一対応
- *vor allem* は副詞句で文中の任意の位置に置けるが、強調したい要素の直前が自然

**関連表現**:
- "Es tut weh, wenn ich tief einatme." = 深く息を吸うと痛い
- "Der Schmerz strahlt ins Bein aus." = 痛みが脚に放散する（坐骨神経痛の典型表現、*ausstrahlen* = 放射する）
- "Ich hab so ein Stechen in der Brust." = 胸にチクチクする痛みがある

### Phrase 4: Muss ich das Antibiotikum wirklich zu Ende nehmen, auch wenn's mir besser geht?
**日本語訳**: 抗生剤って、調子良くなっても本当に最後まで飲み切らなきゃダメですか？

**単語リスト**:
- das Antibiotikum (pl. Antibiotika) = 抗生物質
- zu Ende nehmen = 最後まで飲み切る
- auch wenn = 〜であっても（譲歩節を導く接続詞）
- es geht mir besser = 私の調子は良くなっている

**使う場面**: 処方薬について医者に確認する時の典型疑問。患者の自主的な質問はドイツでは**好意的に受け取られる**（医療パターナリズムを避ける文化）。"Compliance"（服薬遵守）の確認会話。

**フォーマル度**: neutral（敬語のSieは省略されているが、文構造はフォーマル）

**キーワード解説**:
- *das Antibiotikum* はラテン語起源で、**中性名詞**。複数形は *die Antibiotika*（不規則）
- *zu Ende nehmen* は分離可能でない決まり文句。"die Packung zu Ende nehmen" / "die Tabletten zu Ende nehmen" の形で使う
- *auch wenn* は**従属接続詞**で、後続文は**動詞文末**："auch wenn es mir besser **geht**"
- *es geht mir besser* は **es+geht+mir+形容詞** の決まり構文。"Mir geht's gut/schlecht/besser" のように **mir はDativ**

**関連表現**:
- "Wie oft muss ich das einnehmen?" = どのくらいの頻度で服用しますか？
- "Gibt's Wechselwirkungen mit anderen Medikamenten?" = 他の薬との相互作用はありますか？（die Wechselwirkung = 相互作用、医療文脈で頻出）
- "Soll ich's mit oder ohne Essen nehmen?" = 食事と一緒に飲むべきですか、別々に？

## 気づき・洞察

ドイツ語の医療会話には**「痛みの性質を動詞で表現する」**という他言語にあまりない特徴がある。日本語は「ズキズキ・チクチク・ジンジン」のオノマトペ、英語は "throbbing, stabbing, dull" のような形容詞、ドイツ語は **動詞**（zieht, sticht, brennt, pocht）で表現する。これは医療面談を**動詞の選択 = 痛みの分類**として体系化していると言える。

**Krankschreibung（病欠診断書）**の制度は、ドイツの労働文化を反映している。「病気は診断書で証明する」という形式主義と、「3日までは自己申告でOK」というプラグマティズムの折衷。日本の「気合で出社」文化とは対照的で、**「具合悪いなら休むのが正しい」**という社会的合意がある。

抗生剤を最後まで飲み切るかの質問（**Phrase 4**）は、近年「**過剰使用が耐性菌を生む**」というEUの公衆衛生キャンペーンで議論された。医者は「症状が消えても処方された量は飲み切れ」と指導し続けているが、最近の研究（BMJ 2017等）では**「症状改善で中止も可」とする見解**が出ている。患者がこの質問をすることは医学的に意味がある。

**口語の縮約**（Ich schaff's, mir geht's, geht's mir）は教科書ではあまり扱われないが、診察室では普通。医者自身も**敬語と口語を混ぜる**ことが多く、堅い "Wie kann ich Ihnen helfen?" の後すぐに "Was haben Sie denn?"（どうしたの？）と砕ける場合も。

## 他分野との接続

- **anatomy (横隔膜と呼吸)**: "tief einatmen"（深呼吸）の話法は、横隔膜呼吸の臨床評価そのもの。ドイツの理学療法（Physiotherapie）は世界的に体系化されており、Bewegungsapparat（運動器）の語彙が豊富
- **music (Monteverdiの seconda prattica = 規則の柔軟化)**: 「**症状改善で抗生剤中止可**」という近年の動きは、長年の医学規則を**新しい証拠で柔軟化**するもので、Seconda Prattica の「規則を破ってよい正当な理由」と思考構造が同じ
- **bread (Postbiotic = 微生物由来の健康効果)**: 抗生物質の話題は、サワードウマイクロバイオームや腸内細菌叢の議論と直結。**「殺菌するか共生するか」**は医療パラダイムの分岐点
- **golf (Strategic = 患者の選択)**: 「最後まで飲むか中断するか」は、ゴルフのStrategic設計同様、**情報を持って自分でリスクを選ぶ**という近代医療倫理の核心
