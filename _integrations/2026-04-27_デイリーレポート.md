**eBPF** は cBPF (1992) のパケットフィルタから出発し、Alexei Starovoitov の拡張 (2014) でカーネル内に**安全に拡張可能な汎用VM**となった。**verifier** は CFG 検証＋抽象解釈＋メモリ安全＋終了保証を組み合わせた静的検証器で、bounded loop と型付きポインタによる**意図的なチューリング不完全性**で安全を担保する。**CO-RE** が BTF 経由でカーネルバージョン間配布を可能にし、**sched_ext** (6.12) はスケジューラ自体を BPF で完全置換できる時代を開いた。前日の io_uring と対照的に、**Google が io_uring を全面停止する一方で eBPF は verifier の継続改善に賭ける**——カーネル境界拡張の二つの戦略。

### music: 機能和声の臨界点 [[詳細]](https://github.com/YamamotoNaoyuki/DailyLearning/blob/main/domains/music/2026-04-27_ワーグナーのトリスタン和音と機能和声の崩壊.md)
**Wagner『Tristan und Isolde』** (1865) 前奏曲第2小節の **F-H-Dis-Gis** 4音は、150年解決のつかない五分析を生んだ——Riemann の French 6度、Schenker の対位法事象、Schoenberg の **vagierender Akkord (さまよう和音)**。本質は**意図的な構造的曖昧性**で、複数の機能を同時重ね合わせる。**4時間の前奏曲を通じて A 短調主和音は来ない**——Schopenhauer の意志哲学を和声構文に直接翻訳した、**機能和声の臨界点**。Debussy の Golliwog 引用、Berg の Lyric Suite 暗号、Schoenberg の無調への正統化——**20世紀音楽の文法は1865年に始まった**。

### piano: 暗譜は「保存」でなく「冗長性」 [[詳細]](https://github.com/YamamotoNaoyuki/DailyLearning/blob/main/domains/piano/2026-04-27_暗譜の四記憶とChaffinの演奏キュー理論.md)
**Roger Chaffin** がコンサートピアニスト Imreh と共に Bach イタリア協奏曲 Presto の 10ヶ月学習過程をビデオ記録し、**performance cues (PC)** を発見。**意識的に注意を向けた小節は2年後の再演でも記憶残存、PC のない小節は忘却**。古典4記憶 (聴覚/視覚/運動/分析) に**第5の構造記憶 (PC)** を加える Chaffin パラダイムの本質は、**「失敗時の復帰経路を多重化する」設計**——暗譜は完璧な記憶ではなく、自動運動の連鎖が切れた時の**意識的なリカバリーポイント網**。これは航空業界の冗長設計や Pelz の "17インチ過ぎる速度" と同じ**失敗時許容の哲学**。

### golf: 物理学が直感を救う [[詳細]](https://github.com/YamamotoNaoyuki/DailyLearning/blob/main/domains/golf/2026-04-27_Pelzの17インチ理論とBroadie批判.md)
元 NASA エンジニア **Dave Pelz** の "17インチ過ぎる速度" は、**lumpy donut**（プレー後半のスパイクマーク蓄積）と **capture diameter の速度依存減少**から導かれる物理学的最適点。**Penner (2002, Canadian J. Physics)** が定量化:速度1.5m/sでホール有効直径は半減、2.0m/sで実質ゼロ。**Mark Broadie の Strokes Gained Putting** (Columbia, 2011) は ShotLink データから**距離別 in 確率の正規化**を達成、Tour 8ft 50% vs Am 30%。両者は対立的に語られるが**実は補完**——Pelz=個別パット物理、Broadie=戦略統計。Tour 一線級 (Tiger, Spieth) はどちらも実装している。

### anatomy: 個別筋肉は便宜的虚構 [[詳細]](https://github.com/YamamotoNaoyuki/DailyLearning/blob/main/domains/anatomy/2026-04-27_筋膜とアナトミートレイン.md)
**Tom Myers の Anatomy Trains** (2001-) は、Vesalius 以来 500年続いた「**個別の筋肉**」概念を**便宜的虚構**として批判し、**12のミオファシャル・メリディアン**として身体を再記述する。**Wilke 2016 の系統的レビュー**は SBL (足底筋膜→アキレス→ハム→脊柱起立筋→帽状腱膜) に**強い証拠** (14研究) を与える一方、SFL は**証拠なし**——理論を捨てずに改訂する科学プロセスの典型。**Buckminster Fuller の tensegrity** (Ingber が細胞、Levin が biotensegrity) は身体力学を**点の力学から場の力学へ**転換。**Stecco の Fascial Manipulation** はヒアルロン酸凝集を介した滑走障害を生化学的に説明。**個別筋を意識する練習より、筋膜ライン全体を感じる練習**——Matthay/Taubman の "continuous arm motion" の理論的裏付け。

### bread: 水を分子に閉じ込める [[詳細]](https://github.com/YamamotoNaoyuki/DailyLearning/blob/main/domains/bread/2026-04-27_湯種法とアジア式ミルクブレッドのデンプン糊化.md)
**湯種 (日本)** と**タンチョン (台湾)** は、共に小麦粉の一部を予め水で糊化させ本生地に混ぜる技法。本質は**デンプン糊化** (gelatinization) で、58-65℃ で結晶構造が崩壊し顆粒が10-20倍膨潤、自重数倍の水を**結合水**として閉じ込める。**焼成中の水分蒸発が30-50%削減**され、クラム含水率 70-75%、柔らかさ3-5日維持。「**Hokkaido milk bread**」名称は台湾発のマーケティング——**地理的ラベルが化学的技法を象徴化**する現代食品グローバリゼーションの典型。**フランスのルー (1450年代)、日本の湯種 (戦前)、中華タンチョン (1990年代)** が独立に同じ化学に到達——**料理の収斂進化**。

### cs: 「ロックを増やす」でなく「版を増やす」 [[詳細]](https://github.com/YamamotoNaoyuki/DailyLearning/blob/main/domains/cs/2026-04-27_MVCCとスナップショット分離.md)
**MVCC (Multi-Version Concurrency Control)** の核心は「**読み手は書き手をブロックしない、書き手は読み手をブロックしない**」。書き込みは新バージョン作成、古い版は読み手のために保持。**Snapshot Isolation** は各トランザクションが開始時 snapshot のみを参照——ANSI SQL の REPEATABLE READ より強く、SERIALIZABLE より弱い独立カテゴリ (Berenson 1995)。**Write Skew** が SI の落とし穴を作り、**Cahill 2008 の SSI** が動的 rw-conflict graph で完全 Serializable を達成 (PostgreSQL 9.1)。**Spanner の TrueTime** は GPS+原子時計で分散 SI を実現——**ハードウェアで時間の不確実性を有界化**する。MVCC の VACUUM 問題は**Uber の PostgreSQL 離脱の主因** (2016)、理論的優美さが運用で破綻する典型例。

### house-hunting: 通れるが建てられない [[詳細]](https://github.com/YamamotoNaoyuki/DailyLearning/blob/main/domains/house-hunting/2026-04-27_接道義務と再建築不可物件.md)
**建築基準法43条**:「敷地は道路に2m以上接しなければならない」——戦後の都市火災対策で1950年制定。**42条2項道路 (みなし道路)** は1950年現状追認の経過措置で、75年経った今も大量残存。**セットバック**で実質4m確保するが、後退部分は容積率計算外、**実質的な資産価値減**。**再建築不可物件**は接道義務未達——民法上の通行権 (囲繞地通行権) は確保されるが建築基準法的に建替え不可、**通れるが建てられない**奇妙な状態。市場価格は周辺の30-70%。**43条但し書き**の特定行政庁許可で救済可能だが**自治体運用基準が大きく異なる**——横浜市建築局は厳格な部類で、買う前の建築指導課事前相談が必須。

### deutsch: 法律語彙が日常会話に浸透する社会 [[詳細]](https://github.com/YamamotoNaoyuki/DailyLearning/blob/main/domains/deutsch/2026-04-27_Internet_und_Soziale_Medien.md)
オンライン日常で `**laggen**, googeln, posten, liken` といった英語動詞が**ドイツ語弱変化動詞パラダイム**に完全統合 (er laggt, hat gelaggt)。`Hab gelesen, dass die was Neues angekündigt haben — irgendwo bei Insta oder so` のように、**完了形 + dass 副文 + 漠然語尾**で口語省略パターンが体系化。`**datenschutzrechtlich heikel**` のような4音節合成形容詞が日常会話で出てくるのは**法律語彙の口語化が異常に進んだドイツ社会**の言語的指標——**法律語彙の口語化は社会の法治志向の計量装置**。`Du, du laggst grad` の `Du, ...` は文法的意味を持たない**注意喚起 Diskursmarker**——日本語の「ねぇ」、英語の "Hey" と同型の普遍的会話戦略。

### other: 30年で「他の太陽系」が観測科学になった [[詳細]](https://github.com/YamamotoNaoyuki/DailyLearning/blob/main/domains/other/2026-04-27_系外惑星検出法.md)
**Mayor-Queloz の 51 Peg b** (1995, 2019年ノーベル賞) 以来30年で確認 5,800個。**Transit method** (4,329個) が主流、**radial velocity** (9 cm/s 精度の HARPS/ESPRESSO)、**microlensing** (一回限りのイベント、Roman 2027 期待)、**direct imaging** (10億:1コントラスト、JWST が大気分光時代を開く)、**astrometry** (Gaia の3μsec角測位)。Kepler の最大発見は「**主系列星の50%以上が惑星を持つ、地球型 HZ 惑星 20%**」——**コペルニクス的脱中心化の継続**。Hot Jupiter / Super-Earth の発見は**「太陽系を典型と思った理由はサンプル1だった**」ことの宇宙物理学的教訓。**TRAPPIST-1 b は大気を持たない可能性** (Nature 2023)——赤色矮星周りハビタブルゾーンの想定が見直される負の結果。

## 横断的パターン

### 1. 「制約の受容が安全/機能を生む」設計哲学 (eBPF / 接道義務 / Pelz / Chaffin)
- **eBPF verifier**: bounded loop と型付きポインタで**チューリング不完全に**することで安全を生む
- **接道義務**: 「2m以上接道」という形状規制を強制することで**緊急時の安全**を担保
- **Pelz 17インチ**: 「失敗時にホールから17インチで止まる速度」と**失敗を計算に入れる**ことで in 確率最大化
- **Chaffin の PC**: 「自動運動連鎖が切れる」前提で**意識的なリカバリーポイント**を埋め込む
- **共通原理**: 「**完璧を目指さず、失敗時の許容を設計に組み込む**」——リスク調整最適化の同型構造

### 2. 「水平・連続性が垂直・個別性を打ち負かす」(Wagner / Anatomy Trains / 湯種)
- **Wagner Tristan**: 線的半音階声部進行が、垂直的和音同定を打ち消す——**和声の自律性が崩壊**
- **Anatomy Trains**: 個別筋肉という分節化を否定し、**筋膜ネットワークの連続体**として再記述
- **湯種法**: 個別の小麦粉粒という分節化を糊化で**水分マトリックス**として連続化
- **共通原理**: **個別/分節化を解いて連続体として再構築**することで全体的な機能 (音楽的緊張・身体運動・パン物性) が立ち上がる

### 3. 「時間を物質化する」装置 (MVCC / 系外惑星 transit / Wagner 大規模調性)
- **MVCC snapshot**: 「ある瞬間のデータベース全体」を物質的に保持し、時間を**離散版**として並列保管
- **Transit method**: 恒星光度の**時間軸変化**を捕捉して、**通過の瞬間**を観測可能にする
- **Wagner 大規模調性**: 4時間にわたって解決を遅延させ、**時間そのものを未解決感の媒体**に変える
- **共通原理**: **時間を「流れ」でなく「物質的に保持できる対象」に変換**する装置——Git のコミット、Wagner の解決遅延、Spanner の TrueTime も同型

### 4. 「分布として/平均として」記述する文化 (Pelz 17インチ / Kepler occurrence rate / `gleich`)
- **Pelz 17インチ**: 「目標」でなく**統計的平均**——分布として putt 結果を記述
- **Kepler occurrence rate**: 主系列星 50% が惑星を持つ——**確率分布として宇宙を記述**
- **ドイツ語の `gleich`**: 5秒～30分の幅を持つ**確率分布的時間表現**
- **共通原理**: **点の決定論を諦めて分布で世界を記述する**統計的思考の普及——観察と発話の両方に浸透

### 5. 「文法/構造が哲学を内包する」(Wagner-Schopenhauer / ドイツ語 Pertinenzdativ / DSGVO 語彙)
- **Wagner**: Schopenhauer の意志哲学を**和声構文に直接翻訳**
- **ドイツ語身体表現**: `Es zieht mir im Rücken` の Pertinenzdativ は**「身体は経験の場」というフッサール現象学を文法で先取り**
- **`datenschutzrechtlich`**: DSGVO の規範国家性が**4音節合成形容詞の日常使用**として言語化
- **共通原理**: **言語/文法/構造そのものが哲学を運ぶ媒体**——意味は語彙でなく構文に宿る

## 意外な接続点

### Wagner Tristan和音 ↔ MVCC 「複数の意味を同時に持つ」
**トリスタン和音は同時に複数の調圏で複数の機能を持ちうる**——Schoenberg の "**vagierender Akkord**"。これは MVCC の「**同じデータが複数の snapshot で異なる版を同時に持つ**」と構造的に同型。**多義性を維持したまま並行存在させる**設計——音楽和声と並行制御が同じ哲学的問題に到達する。

### Anatomy Trains ↔ Wagner の「水平が垂直を消す」
古典解剖学が「**個別の筋肉**」を一次単位とするように、古典和声は「**個別の和音**」を一次単位とする。Anatomy Trains の筋膜ネットワークと Wagner の線的半音階は、**個別性を解体して連続体を取り出す**点で完全に同型。Tom Myers と Wagner は、約140年離れた異分野で**同じ認識論的革命**を起こした。

### Pelz の "lumpy donut" ↔ 系外惑星検出のノイズ環境
**ホール周辺の凹凸 (lumpy donut)** が真の最適速度を歪めるように、**恒星のマイクロフレア・スターポット・恒星活動**が transit 信号にノイズとして加わる。**「シグナルそのものではなく環境を記述することが本質的な観測精度を決める**」——Pelz の物理学と exoplanetary detection は同じ**ノイズ環境の科学**。

### Chaffin の暗譜 PC ↔ MVCC の snapshot 復元点
Chaffin の **performance cues** は「**運動連鎖が崩れた時に意識的に降りられる駅**」——MVCC の snapshot が「**任意のトランザクションが参照できる過去の整合的状態**」と完全に同型。**システムが崩壊した時に復帰可能なアクセス点を網に織り込む**——音楽暗譜とデータベース並行制御が同じ「**冗長性 by design**」の哲学。

### eBPF の「ポリシーをコード化」 ↔ ドイツ語 DSGVO 語彙の日常化
**eBPF (sched_ext, Tetragon)** は「**カーネルポリシーを動的にロード可能なコード**」にする——伝統的にカーネル内で硬直化していたものを**プラグイン化**。ドイツ社会の `datenschutzrechtlich` の日常化は、**伝統的に専門家領域だった法律を市民レベルでプラグイン可能**にしている。**専門家の独占を解体して一般者がカスタマイズできるレイヤーに引き降ろす**——技術と法律の並行的な民主化。

### 湯種のデンプン糊化 ↔ 接道義務のセットバック
